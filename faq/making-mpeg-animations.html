making Animations

Question:

How can I make animations using FERRET?

Solution:

Making animations with ferret is a relatively simple, albeit a multi-step, process. The method involves saving ferret snapshots as GIF files, or .png files created in Pyferret, and sending them to third-party software to create an a nimaged gif. Another option is to create a gif animation. See the documentation in chapter 5 Note on using a set of gif images to make a movie .

For up-to-date suggestions search the Ferret Users List archive, looking for more recent threads, such as this one:
