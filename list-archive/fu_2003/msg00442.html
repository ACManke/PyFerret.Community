Pedro,
  I have used the MI/X server successfully with ferret, although under Windows 98.  You need to set your DISPLAY environment variable to the IP address (or network name) of your Windows machine.  The command varies with the flavor of unix but it's something like

setenv  DISPLAY IPaddress:0

There are instructions in the MI/X help about it.  Lynn

At 3:44 PM -0700 6/4/03, Kevin O'Brien wrote:
Hello Pedro -

From the error message, it looks like there is something amiss with your X server software.  I'd recommend checking the configuration and making sure that things look like they are set correctly.   I'm not familiar with MI/X, but do know that Ferret works under XP using Exceed.  You might try running a simple X application (like "xclock) from another machine and seeing if you can get that to work.

Let me know if you continue to have problems.
