Installing or Building PyFerret

PyFerret is best installed using either the pre-built tar files if available for your operating system, or by using the the Anaconda installer. All installation information may be found in the PyFerret GitHub repository in this README file. For building PyFerret please scroll down in that file to find the section about the miniconda Anaconda installer.
