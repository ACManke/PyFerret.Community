Hi -
To read three rows, you would need to set up a grid and
use it to read the data.  There are several examples that
show how to read ASCII data in the users guide; look in
the index under "ASCII data, reading,examples".  These
show how to handle data which has one and two dimensions.
In general, you don't need the FORMAT qualifier if the
data is separated by spaces as yours is.  Ferret can read that
as free formatted data.
