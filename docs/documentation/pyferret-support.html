PyFerret Support

PyFerret is publically available and is supported by the PyFerret Community.

  • The Github pages list important information about installing PyFerret.

  • Consult the on-line documentation Ferret User's Guide and its extensive index as well as the FAQ pages.

  • Consult Ferret demonstrations and tutorials.

  • Consult the Archive of Ferret Users List (link commented out in html) to see if this question has been raised and answered before.
