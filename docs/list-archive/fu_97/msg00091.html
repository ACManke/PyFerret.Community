Greg,

	I don't know if there is a better way, but here are two ideas:

1) Run ferret in a directory that you don't have write permission

2) Run ferret; do a cancel mode journal; then have the next command be:
	spawn "/bin/rm -f ferret.jnl*"
 this last one isn't exactly creating no journal, but it will have the same
 effect.
