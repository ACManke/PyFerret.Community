SONGYONGJIA,

Can you tell me what unix shell you use?  If you

  echo $shell

what is the response.  If you use csh or tsch (the c or t shell) you should
have the "source" command.   The Borne and Korn shells do not have "source".
 To experiment you might enter the c shell (with command "csh") and then try
the source command.  Please let us know how that works.

The problem with the core dump may go away when you are able to source
ferret_paths.  That should define two environment variables specific to IBM
systems:

   setenv XLFRTEOPTS namelist=old
   setenv LANG C

The second command is important if you are running a recent version of the IBM
operating system.  If you ask

  uname -a

what is the response?


	Regards,

	Jerry Davison
