check your LANG environment variable.  Before running Ferret, it has
to be set to C, e.g, you need to do "setenv LANG C".  right now it's 
probably set to "en_US", or something similar.  That problem affect some
pre-compiled program out there, Netscape being another one.
At least that's my guess to the solution for your problem.
