On Tue, Jan 15, 2019 at 11:20 AM raden bima <radenbimayoga@xxxxxxxxx> wrote:

I have some problem when i install executables (fer_executables.tar.gz), it shows several error such as "lib not found in archive" and also "there is a problem in manipulating files in....". I try move the tar data (fer_executables.tar.gz) to another folder, but it still shows the same problem. I also attached my terminal's screenshot that shows the problem.

The first error "tar: lib: not found in archive" seems to come from

tar tzf ${ferexec_dir}/fer_executables.tar.gz lib

So, you may want to manually check whether the tar archive is okay.  Type this on your command line

tar -tzf /usr/local/ferret/ferret-7.4- . . . /fer_executables.tar.gz
 
By the way, please copy and paste the text from your terminal, not take the screenshot.  The latter is an "image", from which you can't extract text (without applying a special processing).  That's why I wasn't able to give you the exact command you want to use above.

Hope this helps,

Ryo
