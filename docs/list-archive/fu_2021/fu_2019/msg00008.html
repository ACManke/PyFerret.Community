Hi Deni,

You can shift the time of the event mask using the L=@SHF:n transformation. This just moves the values forward or backward in time. In your case

let/title="12 months before the evant" event_12 = event[L=@shf:12]

In your specific example it will move the first event off the time axis so you may have to make an extended time axis with nothing but missing values if you need  that first event.

Cheers,
Russ
