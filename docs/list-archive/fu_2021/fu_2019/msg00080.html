On Tue, Mar 19, 2019 at 6:05 AM Kwesi A. Quagraine <starskykwesi@xxxxxxxxx> wrote:

let nanom2 = anom[gt=upper[d=4]]
let nx = `upper,return=isize`
 **ERROR: variable unknown or not in data set: UPPER

As this message says, the variable "upper" isn't found in the current dataset.  (I suspect you need `upper[d=4],return=isize`.)

I suggest that immediately after you see this error message, you type

yes? show data

This command lists the datasets Ferret has loaded.  One of the datasets will have"(default)" at the end of its first line.  For non-default datasets, you need to specify its dataset number as in  upper[d=4] .

Hope this helps.
