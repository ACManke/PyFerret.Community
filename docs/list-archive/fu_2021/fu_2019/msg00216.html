Hi,
I am unaware of the command which displays all the data points, but dispaying all data points may be not convinient especially for large data. To see all data: (1) you may do plot itself  Or (2) you may do  LIST/FILE=my_file.dat/CLOBBER variable[i=section,j=section] and later see the .dat file (3) Perhaps for one dimensional listing, ferret lists all points, for more dimension, it skips few indeces.
