Hi Emma,

You need to define P and Q using expressions that include the averaging so that they are 1-D Time variables.  Then the regresst script will define qhat correctly.

Yes? Plot/vs mld[i=@ave,j=@ave],sst[I=@ave,j=@ave]


Yes? Let p = sst[I=@ave,j=@ave]

Yes? Let q =  mld[i=@ave,j=@ave]

Yes? Go regresst

Yes? plot/vs/line/over p, qhat
