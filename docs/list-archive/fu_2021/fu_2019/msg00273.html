Hi,

Just in general, there are a couple of demo scripts for vector plot styles.  To find them, from the Unix command line:

   > Fgo vector

makes a list of all the installed scripts whose names include "vector".  Try,

   yes? go vector_demo

Also there's a "stick_vector" script which you may have found. There is not a built-in capability to draw stick vectors with arrow heads.
