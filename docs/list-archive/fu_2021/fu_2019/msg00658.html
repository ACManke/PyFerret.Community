Dear all,

Another option to remove a "degenerate" coordinate is to use an NCO operator:

ncwa -a <name of the axis to be deleted> input.nc output.nc

See

https://sourceforge.net/p/nco/discussion/9830/thread/5a6d7319/

This is basically the same as Kwesi's method of using the @MAX operator or the like.
