Hello again,

 

Just a quick query on how to modify the size of my title, created through PPL TITLE...

 

set view ul

palette white_centered

sha/pal=white_centered/LEV=(-200,200,25)/NOLAB dic_bias[k=1:4@ave]

PPL TITLE,1000, Model Bias in Predicted DIC Between 40.4m and 346.4m (micromolkg^-^1)

PPL SHADE

go land

 

I added in the 1000 just to see whether it makes a slight bit of difference (as it wasn’t working when I was specifying the inches) but unfortunately it didn’t. This was following the documentation on PPLUS that says:

 

TITLE,HLAB,label

Sets the main plot title to "label" without generating a plot. If "label" is omitted the main plot title is cleared. Optionally the size of the title can also be specified.

HLAB = the height of the title in inches. (default = .16 inches)

Any clarification welcomed.
