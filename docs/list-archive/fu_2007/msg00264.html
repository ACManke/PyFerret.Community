Hi,

Use ferret under csh probably can solve this problem.
You can add source command in /etc/csh.cshrc
Like (assume ferret_paths installed at /usr/local/ferret):

# Adding Ferret path
source /usr/local/ferret/ferret_paths

Then change shell to csh, ferret should work now.
Dan
