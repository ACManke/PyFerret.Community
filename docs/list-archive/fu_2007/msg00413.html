Hi Rolando,
Some of the simpler definitions of these kinds of statistics between two time series are made in the script variance.jnl. This script defines a set of variables from your input variables P and Q. To see what it does, 

yes? go/help variance
In addition, take a look at the script "fft2drun.jnl" which makes a running power spectrum. The script mentions limits on the length of the time variable; I'm sure these could be increased if you just make your own copy of the script and change the SET MEMORY command. 

Anyone else have scripts or external functions to do these kinds of things??
