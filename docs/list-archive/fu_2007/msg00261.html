Hi Soenke,
One way to do it is to reverse the y-coords (in inches) in "ppl shakey". The setting does not "stick" so you will I think need to do something like the following: 

shade/set/i=1:10/j=1:10 i+j
ppl shakey,,,,,,,,,7.4,1.4 ; ppl shade

Good luck,
Mick
PS the 7.4,1.4 are the top and bottom coords in inches of the y-axis in a standard 8"x6" plot. You could use other values to alter the size of the colorbar, or maybe use the ppl symbols.
