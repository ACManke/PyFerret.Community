Hi Jimmy
Ferret v6.0 was linked with OPeNDAP/netCDF libraries which unfortunately lost us the ability to read those extra- large files. The ones we used in Ferret 5.81 were able to do large-file access, but lacked other features we wanted to implement. We are working now to link with even more recent versions of these libraries and I hope to release a new Ferret soon with all of the features that we want. 
Ansley
