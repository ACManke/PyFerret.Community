Hi,
with ferret 6.07 I get a disturbing white line at 20 E
in my plots.

use "http://www.ifremer.fr/cgi-bin/nph-dods/data/satellite/medspiration/l4uhrsstfnd/eurdac/med/2007/322/20071118-EUR-L4UHFnd-MED-v01.nc";
can axis/modulo lon
set win/asp=.5 1
shade SST_FOUNDATION
go fland 5

Saving the data and making the same plot with old 5.41 (AIX),
the plot is correct.

ferret adds the modulo flag to the x-axis (lon). This is
the reason to cancel axis/modulo, but this does not help.

Changing the aspect ratio to .51, the error disappears. One could
live with this workaround, but something seems to be wrong
here.
