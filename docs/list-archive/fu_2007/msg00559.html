* * * Announcing Ferret Version 6.07 * *



Announcing the official release of Ferret version 6.07. This version is required for the latest release of the Live Access server, which was just announced today.

This release includes 32-bit and 64-bit Ferret executables for linux, and a statically linked linux executables for these as well, which should run on many more linux platforms. There is also a build for Solaris 5.8.

To install Ferret, download the file from the Downloads pages of the Ferret Web site

http://ferret.pmel.noaa.gov/Ferret/Downloads/ferret_downloads.html

For a list of bug fixes and new features added in V6.07 see:
http://ferret.pmel.noaa.gov/Ferret/Documentation/Release_Notes/v607.html

And as always the Documentation and Users List are your on-line resources!
