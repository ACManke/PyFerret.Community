Hi  Hanh,
Which version of Ferret are these machines running?  A Google search for the error message
  px_get: Assertion `offset >= 0' failed
shows that this is an error with large-file access. Some versions of Ferret previous to the most recent ones were built with OPeNDAP libraries that did not have large-file support.
