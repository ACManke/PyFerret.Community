Hi Wendy,
Is this a netCDF file? If so try the SET DATA/ORDER command. It looks as though the axis called FILES has thrown things off, since it has no units and the name does not give Ferret a clue about its direction. I think the following will work on your file, but test it out, 

  SET DATA/ORDER=TZXY

Ansley
