         * * * Announcing Ferret Version 6.02 * * *

             * * * for Linux and Solaris * * *

Announcing the official release of version 6.02 of Ferret. This release is primarily a bug-fix release, with a couple of new features. In particular there is now a metafile batch mode, which lets us write multiple metafiles within a Ferret batch session, and "native" strides defined for netCDF file access.

Web address: http://ferret.pmel.noaa.gov/Ferret/Downloads/

For a list of new features and bug fixes included in V6.02 see:
http://ferret.pmel.noaa.gov/Ferret/Documentation/Release_Notes/v602.html
