On 7/7/07, William S. Kessler <William.S.Kessler@xxxxxxxx> wrote:
In a word: No!

The contour drawing algorithm in Ferret is decades old code from
Plotplus, which was based on even earlier code that Roger Lukas put
together for his thesis, and before that from someone named Thompson
at the Canadian Bureau of Mines in Ottawa in 1967 (details a little
hazy). Really, really old code. As you have found, the controls on
contour labeling are indirect at best, and not very user-manipulable.

A very crude solution, extremely user-intensive and only useful for a
final publication plot, is to draw each contour level separately and
overlay them (contour/over/nolab/lev=/col=). Then you can use the
various PPL controls (CONSET) to work on each contour line
separately, and probably be able to get the result you want. It will
require a lot of experimentation!

Billy K
