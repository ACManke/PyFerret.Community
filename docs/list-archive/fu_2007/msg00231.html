Hi Siram,
The Ferret documentation calls this kind of data "curvilinear data", so you can read more about this in the Users Guide. Because your nc file contains the lon, lat coordinates of the data, you can use this to plot with the "curvilinear" coordinate graphics commands in Ferret
 
   yes? fill/lev=(-2,25,1) temp, lon, lat

Ansley
