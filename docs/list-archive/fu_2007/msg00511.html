Hi Jimmy,
Another thing you could look at, to decimate the data along some direction, is striding when you read  the data, if it's in a netCDF file. If you're using Ferret v6.02 or higher, there's simple syntax which changes the way the axis appears to be defined for the entire Ferret session, see
http://ferret.pmel.noaa.gov/Ferret/Documentation/Release_Notes/v602.html#native_strides

Or, with any version of Ferret, if you have a variable on an x axis, you can specify strides as follows (see "strides, netCDF discussion" in the Users Guide index)
yes? use my_data.nc
yes? let nx = `var,return=isize` 
yes? plot/dash var[i=1:`nx`:10]
