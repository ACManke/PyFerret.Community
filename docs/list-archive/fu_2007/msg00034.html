Hi Emilie,
Jaison has some great suggestions.  Another means of removing the axis
labels is with the ppl axlabp command; 

   AXLABP,LABX,LABY 

   Sets the numeric and character label position for X and Y axes.   
   -1=bottom/left of plot, 0=no label, +1=top/right of plot.    default=-1 

So, along with the other ppl commands you're using in your original plot
command, add a "ppl axlabp,-1,0"  to remove the y axis labels.

Ansley
