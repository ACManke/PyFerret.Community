Hi!

You should create a file .bashrc in your home directory (for
example, /home/xhbhyq11). Insert the line:

source /usr/local/ferret_paths

Ask your system administrator to install csh or tcsh. To run ferret you just
have to type "csh -c ferret" or "tcsh -c ferret" (according to the c shell
version installed).
