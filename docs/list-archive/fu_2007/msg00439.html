* * * Announcing Ferret Version 6.04 * *


Announcing the official release of Ferret version 6.04. This is mostly a bug-fix release, with a couple of small new features.

This release includes 32-bit and 64-bit Ferret executables for linux, and a statically linked linux executable, which should run on many more linux platforms. We are working on a build under Solaris 5.8 as well.

To install Ferret, download the file from the Downloads pages of the Ferret Web site

http://ferret.pmel.noaa.gov/Ferret/

For a list of bug fixes and new features added in V6.04 see:
http://ferret.pmel.noaa.gov/Ferret/Documentation/Release_Notes/v604.html

And as always the Documentation and Users List are your on-line resources!
