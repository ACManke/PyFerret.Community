Hi Everyone,
Ned points out that if the LIST command is used to list more than one variable, then LIST/PREC= does not affect the precision of the output,
yes? LIST/precision=6 var1, var2   ! has no effect
But, SET LIST/precision=  does change it.
yes? SET LIST/precision=6 
yes? LIST var1,var2   ! the precision is changed.
This is a bug, and we have a fix for it which will be in a future Ferret release.

Ansley

E. D. (Ned) Cokelet wrote:
Issue the command:

set list/precision=6
