Use the PPL SHAKEY command:
http://ferret.pmel.noaa.gov/Ferret/Documentation/Users_Guide/v605/ fer_html.htm 

fill/set ....
ppl shakey [shakey options, see above link]
ppl fill

To simple change the font size use the third option:

ppl shakey 1 1 .12       ! get 0.12-inch letters
Otherwise the remaining option allow complete control of the position, size, etc.
