Hi Mark,

Try using string variables instead of symbols:

        NOAA/PMEL TMAP
        FERRET v6.07
        Linux(g77) 2.4.21-32 - 08/29/07
        17-Sep-07 14:05

yes? let a = "a"
yes? let b = ""
yes? let c = "c"
yes? let d = "`a``b``c`"
 !-> DEFINE VARIABLE d = "ac"
yes? list d
             VARIABLE : "ac"
        "ac"

On Mon, 17 Sep 2007, Ansley Manke wrote:


Hi all,
I think Jaison is right that the best way to get what you want, Mark, is the string edit functions. I'd just add that you can do all of this without defining a blank symbol; if the symbol b is undefined, referring to it results in a blank. 

  yes? def sym a a

  yes? cancel sym b

  yes? def sym c c

  yes? def sym d ($a)($b)($c)

  yes? show sym d

  D = "a c"
It would be a nice little addition to Ferret, to add some easier-to-use FUNCTIONS to do the string editing that's in the PPLUS $EDIT command. 

Ansley
