Hi Ferreters,
                Here comes a tool for creating single colored palettes,
"palette_maker_solo". You can use the RGB colors available on your computer
to create fancy color palettes. Let us get some nice colors for our plots
....please have a look at attached GIF file.

   This GO tool is a follow-up to the "rgb_colors" posted earlier (see the
links below). palette_maker_solo takes 3 arguments, like

    GO palette_maker_solo [rgb_color_name], [palette_filename], [YES/NO]

All arguments should be in double quotes. 1st argument is the RGB color
name, case/space insensitive. The GO tool will assign a default palette
file name, based on the specified color name. If you want to override this,
use the 2nd argument. The 3rd argument is for changing the case of filename
to lower case, if "YES" is chosen (default is NO). Try the following 
example :

  yes? GO palette_maker_solo "LightSeaGreen"  !will create LightSeaGreen.spk
  yes? GO basemap "","","","LightSeaGreen"  

As with the rgb_colors, there is support for users who has problem with 
"showrgb" command. Please see the comments and examples to the beggining
of GO files. The attachment (palette_maker_solo.tar.gz) contains the 
following file :

   palette_maker_solo.jnl        --> for those whom "showrgb" command works
   palette_maker_solo_custom.jnl --> for those whom it does not work
   rgb_colors_custom.dat         -->            "       "
