Hi Antonio,
If I understand correctly what you want to do, it would be this, averaging 1:10 in time 

let sst_fav_tav =sst13[X=10w:4w@FAV:7,Y=35n:38n@FAV:7,L=1:10@AVE]
Or, if you want the x and y averaging at each time, you would just define the fill in x and y and then use the desired time steps to save or in whatever other calculations you want to do. 

  let sst13_fav = sst13[X=10W:4w@FAV:7,Y=35n:38n@FAV:7]
  save/file=myfile.nc/L=1:10 sst13_fav
In any case this shows how to apply the @FAV transformation to a region in x and y.
Most often, the REPEAT command is not needed. Ferret operates over entire grids or axes, or the subsets you specify with qualifiers /X= /Y= or SET REGION/X=/Y= or in square brackets [T=]. 
Ansley
