Hi -
You need to first define the desired output z axis, and on your first SAVE command, specify /KLIMITS=10 (or however many Z levels there are). See example 4 in this section - the example shows input data that's ascii and is showing how to convert to netCDF, so your case will be a bit easier, as you're starting from input netcdf. 

http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/converting-to-netcdf/SIMPLE-CONVERSIONS-USING-FERRET#_VPID_348
