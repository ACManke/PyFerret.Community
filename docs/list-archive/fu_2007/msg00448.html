* * * Announcing Ferret Version 6.05 * *


Announcing the official release of Ferret version 6.05. An alert user found a serious bug in the recent v6.04 release, which is fixed in Ferret v6.05.   If you had jumped right on v6.04, my apologies!  Please update to v6.05.

This release includes 32-bit and 64-bit Ferret executables for linux, and a statically linked linux executable, which should run on many more linux platforms. We are working on a build under Solaris 5.8 as well.

To install Ferret, download the executables and environment files from the Downloads pages of the Ferret Web site

    http://ferret.pmel.noaa.gov/Ferret/

For a list of bug fixes and new features added in V6.05 and V6.04 see:

   http://ferret.pmel.noaa.gov/Ferret/Documentation/Release_Notes/v605.html

And as always the Documentation and Users List are your on-line resources!

Ansley Manke
