Ansley Manke wrote:

Hello,
If you look at the file ferret_paths_bash_template from the Ferret distribution, you'll see a few lines at the end having to do with the palette and fonts files. You have the environment variable for the palettes, but not one for the fonts. The error message you are seeing indicates that you don't have the PLOTFONTS variable defined. 

 export PLOTFONTS="$FER_DIR/ppl/fonts"
This should point to the directory which contains the file fondat.dat as well as the files fnt01.chr, ...
If you add this to your .bashrc file, I think you will be in business. Or put all of these definitions into a ferret_paths script and execute that from your .bashrc. 

Ansley
