    ***** Announcing the official release of Ferret version 6.4 *****

Release notes are available at http://ferret.pmel.noaa.gov/Ferret/documentation/v6-4-release-notes

This Ferret release includes executables for Linux 32-bit and 64-bit, Altix build, ia64-sles10; and a statically-linked linux executable for other Unix systems.

This is primarily a bug-fix release, with the first listed fix important for users who are running Live Access Server.

To install Ferret, follow the instructions on the Ferret Downloads pages:
http://ferret.pmel.noaa.gov/Ferret/downloads
