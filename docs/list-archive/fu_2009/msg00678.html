Hi Sarah,
You need to do a regridding operation:

  LET/UNITS="`tempsurf,RETURN=units`"  tsurf = tempsurf[gxy=sst[d=1]]
This command defines a new variable tsurf, which is tempsurf regridded to the x and y grid of sst[d=1], and keeping the units of tempsurf. The regridding operation will be linear interpolation by default, or you can choose another regridding transformation. The time dimension will not be changed. 

Ansley
