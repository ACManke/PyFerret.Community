Hi Rahul,
Ferret has a capability called multi-file datasets, which uses a "descriptor file". This file lists the file names and locations, and information about their time axes. There are a couple of different tools, which have been contributed by users, which will take your list of filenames and write a descriptor file. Then you can point Ferret to that descriptor file and all the data appears as a single dataset.
Take a look at the topics under "descriptor file" in the Ferret Users Guide index (http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/index-1/index) and particularly the link to "tools for creating" under that. I think this will do just exactly what you need. 

Ansley
