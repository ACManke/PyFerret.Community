Hi all,

I am wondering why I cannot list a axis after having define it as a depth.

yes? use file.nc
yes? show grid temp
   GRID GKD1
name       axis              # pts   start                end
LON       LONGITUDE          136mi   177.87W(-177.87)     173.61E(173.61)
LAT       LATITUDE           115 i   90S                  90N
PRESNIVS  Z (mb)              39 i   0.043068             1009
TIME_COUNTER T                 1 r   1                    1
yes? show axis PRESNIVS
name       axis              # pts   start                end
PRESNIVS  Z (mb)              39 i   0.043068             1009
  Axis span (to cell edges) = 1013.798
yes?
==> Normal listing of the PRESNIVS axis

Now set vertical axis as a depth

yes? set axis/depth PRESNIVS
yes? show axis PRESNIVS
name       axis              # pts   start                end
yes?

==> No more listing
