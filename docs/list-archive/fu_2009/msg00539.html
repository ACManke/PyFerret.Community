Hi Isabel,
Yanhgxing is quite right. You can set up a single file which will allow you open your data as a single dataset. This is called a mc file (multi-CDF file)  also known as a "descriptor file".  There are tools to help you create this file. 

A more recent link for the Ferret documentation is
http://ferret.pmel.noaa.gov/Ferret/documentation

And the section to read for directions on hoe to create a descriptor file is at
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/converting-to-netcdf/CREATING-A-MULTI-FILE-NETCDF-DATA-SET

Ansley
