Ah. A very hidden feature. The font size for this label is controlled by the 4th argument to the PPL LABSET command: 

vector/set/.....
ppl veckey,xpos,ypos,,(f3.1," cm/s") ! an example. Note that there can be only 11 characters total in the label 
ppl labset,,,,vector_label_size
ppl vector

Billy K

Hi,
I have a very short question, I am plotting vectors and I have been able to change the vector key with veckey and vecset, however I don’t know to change the size of the text that appears in the vector key so it looks a bit bigger. 

An example:

vector/over/nolabels/set_up $2_ave,$3_ave
ppl veckey/nouser,1,-1,,(f5.0,”m/s”)
ppl vecset,0.5,5
ppl vector/overlay,1,1
