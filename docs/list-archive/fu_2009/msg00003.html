Title: column reversal
Hi Vincent,
I think you'll get what you need with the syntax,

USE/ORDER=

You can specify a reversed axis or even different ordering of axes.  For instance, try this:

   yes? use/order=X-Y etopo20
   yes? shade/pal=dark_land_sea/lev=15c rose


Look up "reverse axis" in the Users Guide index, which will point you to the documentation on this.
