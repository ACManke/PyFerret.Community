It sounds like you found out what you needed, but just so there's an answer on the Users List for others to refer to, I thought I'd write. You can use Pseudo-variables to define a variable containing coordinate values, and use that variable in a calculation: 

yes? let phi = y[gy=var]
where var is some variable in your dataset whose y axis is the latitudes you want to use.
