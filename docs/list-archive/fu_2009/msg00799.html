Hi Scott,
If you look up the information about SET WINDOW/ASPECT, there's an option to set the ratio of the axis lengths by adding :AXIS to the specification:

SET WINDOW/ASPECT=0.5:AXIS

Without :AXIS, the ratio is for the entire window, and the fact that the margins around the plot are a bit different probably accounts for what you are seeing.

Ansley

I am using Ferret to plot an 1800x900 (X by Y) model grid. 

I used   SET WINDOW/ASPECT=0.5

larger than 2x the Y axis
