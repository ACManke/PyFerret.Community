Hi Paulo,
The FLOWLINE command is set up so that "The size of the arrows indicates the speed of the flow."  This means that the /LENGTH qualifier on the command will control the size of the arrows (see VECTOR/LENGTH).

compare:
flow/length=10 u,v

flow/length=1 u,v
And so forth.
