There have been several discussions of this in the user group, with several people suggesting various automatic ways to thicken plot lines for ps/pdf output. There are scripts that work on the "line weight" (lw) parameter in the ps output of Fprint. I think if you search the archives for "thick lines" or "line weight" you would find these. Billy K
