Another is to reset the shadings:

  set view left
  shade/line/key/nolab/pal=blue_darkred temp[k=1]
       
  ppl shaset reset !!<==

  set view right
  shade/line/key/nolab/pal=blue_darkred temp[k=1]

which works for me (Ferret 6.10 and 6.02 for Linux).
