    ***** Announcing the official release of Ferret version 6.2 *****

Release notes are available at http://ferret.pmel.noaa.gov/Ferret/documentation/version-6-2-release-notes

This Ferret release includes executables for Linux 32-bit and 64-bit; and a statically-linked linux executable for other Unix systems. An Altix build, ia64-sles10 is also available.

To install Ferret, follow the instructions on the Ferret Downloads pages:
