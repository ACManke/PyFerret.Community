     ***********************************************************
      ANNOUNCING SERVER IN A BOX -- THE LIVE ACCESS SERVER V3.7
     ***********************************************************

The Ferret group has been working for several years to develop
a portable Web server for gridded data sets.  With the Live Access
Server (LAS) any data that you can analyze and visualize with Ferret,
you can also make available to others on the Web.

The LAS provides a Web front-end to Ferret and is highly configurable --
it uses Ferret scripts (which you can modify) to create graphics and files.

It is available for all systems on which Ferret is supported, and is easily
installed.

The LAS enables the Web user to

 * visualize data with on-the-fly graphics
 * request custom subsets of variables in a choice of file formats
 * access background reference material about the data (metadata)

The current version of the LAS also supports distributed data -- data from
multiple sites presented as a single data base. For an example, see

  http://www.ferret.noaa.gov/fbin/climate_server

and then use the Data Set interface to select the Pathfinder TOVS Benchmark
data set -- data served on a separate host.

Future versions of the LAS will provide a true collaborative framework in which
you will be able to compare (difference or overlay) variables from distributed
locations.

And if you're sitting at a PC with a VRML v.2 viewer installed,
visit a current prototype of the LAS that demonstrates fast 3D visualizations:

  http://ferret.wrc.noaa.gov/jbin/climate_server

Select "VRML globe" output.

For examples of the server at other sites visit:

  http://las.saa.noaa.gov/las-bin/climate_server
  http://www.lmd.jussieu.fr/sechiba/live_access.html
  http://www.lmd.jussieu.fr/mars/frameset/000000.html


The LAS is available free of charge from NOAA/PMEL/TMAP, the developers
of Ferret.  To get the latest version of the software please contact
us at ferret@pmel.noaa.gov.
