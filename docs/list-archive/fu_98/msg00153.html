Antonio,

	you are correct, it is a very easy question. You are confusing the
/xlimits modifier with the syntax for region selection. /xlimits and /ylimits
are used to modify the axis limits, not the data limits. The command you want
is:

fill/LEVELS=(-15,15,2) teste[x=40w:8e]
