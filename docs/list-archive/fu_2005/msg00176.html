Hello Ferreters!

I'm wondering whether it is possible or not with Ferret to plot a temperature field using CONTOUR and where the line color is changing according to the temperature value. A bit like a FILL with a palette, but without filling, or a bit like associating a color palette to the lines that CONTOUR draws.

All I've found in the documentation is:

! mix: three line colors
contour/level="(1,30,1,-1) PEN(5,10,1,2) PEN(11,20,1,3) PEN(21,30,1,4)" my_var

(from the custom_contour_demo.jnl example)

But it's then just 3 colors and the change is not progressive. I'm imagining something like:

CONTOUR/pal="default.spk"/level=(-20,20,1) my_temperature


If anyone knows how to do that, I'll be glad to know it also!
