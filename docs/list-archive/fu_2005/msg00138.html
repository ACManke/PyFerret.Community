You can use Ferret's curvilinear mapping procedures for this. Let's suppose you want to contour variable v1 which is on a rectangular i,j grid that is not necessarily a global lat,lon grid, but you also have lat and lon on that same rectangular grid. Then the following will do the job

contour v1,lon,lat

Your other data set with variable v2, already on a lon/lat grid, is contoured by

contour v2
