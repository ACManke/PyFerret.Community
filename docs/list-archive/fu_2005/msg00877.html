Hi Jian,

If you need legend for the time series plot here is what you can do:

use /nolabels qualifier when ploting and then use legline.jnl script 

for example:

plot/color=7/nolabels your_data 
go legline xsc  ysc  color fs "description of your data"

arguments for the legline are as following

xsc is x starting coordinate for placing the legend in user units
length is the length of the line in units of x axis
ysc is y axis coordinate for placing the legend line
color is color of the line
fs is font size

if your x axis value are from 1960 to 2000 and and y axis are from 
0 to 700 then issuing

go legline 1965 5 650  7 .16 "Precipitation"

will result in ploting the line of color 7 (black) in upper left corner with 
starting coordinate (1965,650) ending at (1970,650), beside the line with the 
fontsize .16 it would be written Precipitation

I hope this helps, but if you google for: "ferret legline" I am sure you will 
find more script and descriptions.
