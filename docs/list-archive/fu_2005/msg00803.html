Herve,

There are examples within the LAS distribution that show how you can store you data in a simple table in MySQL and use LAS code to extract the data and create visualizations.  Please look here:

http://ferret.pmel.noaa.gov/LASdoc/serve/cache/20.html

and also try the following after installing your default LAS (>6.41)

cd las/contrib
make insitu

This will install two tables of scattered points into a new MySQL database and set up LAS to provide access to them.  There should be enough examples there to get you started.
