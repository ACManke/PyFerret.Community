Hi David,
There is an FAQ which does something similar to what you're describing
How can I show the value of my variable at its deepest depth?

You would need to define a variable lowbox = zboxlo[gz=var]  and use lowbox where the FAQ uses the temperature variable.

( a hint on finding FAQ's, which I always use when I have an inkling that we have something related to a topic -- go to the FAQ's link from the Ferret web page, http://www.ferret.noaa.gov/Ferret/FAQ/ferret_FAQ.html  and just use your browswer's search function to look for some likely strings.  I looked for "deep" and found this one.)
