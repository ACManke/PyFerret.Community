Hi Goran,
There was a really dumb bug in v5.80 only, where the qualifier /CONTINUOUS was not recognized if it appeared in uppercase letters. This has been fixed in v5.81, but you should also be able to run your script if you put /continuous in lowercase letters

Ansley
