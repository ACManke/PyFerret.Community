At 10:56 AM 9/6/2005, Tony Jolibois wrote:
Hi,
Do you run ferret in the good directory ? This error say that ferret does not find your data file. You must have the file in the directory you run ferret with the command "use ocean_month_1991_01.nc".
In an other way, ferret search the data files in the directory defined in the environment variable $FER_DATA.
My $FER_DATA looks like that :
FER_DATA=. /usr/local/ferret/fer_dsets/data /usr/local/ferret/go /usr/local/ferret/examples /usr/local/ferret/contrib
If your data file is in one of these directory, it should work.

Hope this help,
Tony
