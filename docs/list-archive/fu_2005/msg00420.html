Hi Tony,
Ferret does not do this now.  As you've seen, it saves only the attributes it uses internally.  We are currently working on a project to add capabilities to Ferret that will  give you control over the attributes that are written, as well as editing or adding attributes. Look for this in the next release of Ferret. For the moment your best bet is probably to do your analysis, write the file, and then use NCO operators to edit the file and restore the missing attributes.  The documentation for NCO is at   http://nco.sourceforge.net/nco.html

Ansley
