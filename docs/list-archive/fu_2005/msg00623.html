Ansley and ferreters,

| (Anyone know how 
| to perhaps turn off beeps made by the terminal or map the beep to 
| something so it wouldn't make a sound?)

On X Windows,

	$ xset b off

will silence all terminals.

If you want a visible bell (the flashing of the screen
instead of a beep sound), some terminals have options
to replace audible bells with visible ones.  For example,
xterm has a "-vb" command line option for it.  If you
want the visible bell _and_ you are a paranoia :-),
you may want to edit your terminfo/termcap database:

	http://www.faqs.org/docs/Linux-mini/Visual-Bell.html

Hope this helps,
Ryo
