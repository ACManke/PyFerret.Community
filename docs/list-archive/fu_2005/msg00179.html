Hello All,
	While looking at the manuals for ideas about the color contour
line problem I noticed something I had forgotten which may be of use
to someone.

Among the @ options in the definition of character strings for labels
is the @Z option. A digit following the Z (0:9 plus A-G) allows one to
customize italics with slopes of 0-45 degrees in 16 steps. Try the
following for fun - maybe someone can find a way to use it to effect.

plot/vs/nolab {0,1},{0,1}
label/nouser 3 4 0 0 0.4 @sr@p7DIY Italics
label/nouser 3 4 0 0 0.4 @sr@p2@zgDIY Italics
label/nouser 4 2 0 0 0.4 @sr@p7Weird Lightsource
label/nouser 4 2 0 180 0.4 @sr@p4@z9ecruosthgiL drieW
