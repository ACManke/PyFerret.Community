Hi Mark,
The two datasets seem to disagree as to what the direction of the dimensions are: the first has J=1:189, K=1:31 and the second has I=1:189, J=1:131.  Ferret won't change the direction of axes when you do a regridding operation. I think you should be able to use the SET DATA/ORDER= to change the way one of the datasets is read, and then use the  regridding using @ASN as you suggest.

Ansley
