Hi Amy,
Do you see a plot window? If you do a simple line plot, without reading any external data, what happens

> ferret

yes? set mode meta
yes? go ptest yes? quit

When you issue the "set mode meta" command, a plot window should pop up on your screen, and you'll also get a metafile. If the plot window does not come up, then the x-server software is not starting up correctly.

Ansley Manke
