Hi Vincent,
We improved this in version 5.8 of Ferret.  I suggest you upgrade if you can.  (An
example of how it looks now is shown in the left frame below).  Also in version 5.6
we changed the functioning of the 5th argument of shakey so that it sets the actual
number of decimal places (klab_dig < 0) in the key; previously it had set a maximum
number of decimal places. This is shown on the right in my example.

   yes? let aa  = i/10+j/10

   yes? set view left
   yes? shade/i=1:10/j=1:10/title="default key" aa

   yes? set view right
   yes? shade/set/i=1:10/j=1:10/title="PPL SHAKEY 1,1, , ,-1" aa
   yes? ppl shakey 1,1, , ,-1
   yes? ppl shade

   yes? frame/file=key.gif
