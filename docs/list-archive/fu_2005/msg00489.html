It may be worth noting that this doesn't work with older versions of
Ferret (5.5?): the postscript that Ferret produces has been cleaned up (a
lot!) in recent versions.

> Recall that to make a postscript metafile, you must:
> 
> yes? set mode meta  ! set only once; cancel with "can mode meta"
> yes? plot ....
> yes? ppl clsplt     ! close the metafile

cancel mode metafile also closes the metafile - so if you put a
set/cancel mode metafile around each plot there's no need for the ppl
clsplt command.
