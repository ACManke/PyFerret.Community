Jean-Marie,

This seems inordinately slow for Ferret. Especially for a file that is only 1.5 Mb! When this happens it usually means that the time axis is marked as 'irregular' which means that Ferret has to read in the entire time axis before it can begin. We have found many cases where the axis was actually 'regular' in spite of what the NetCDF attribute says. If this is the case then you can use ncatted to change the time axis attribute and Ferret should open this dataset much more quickly.


-- Jon
