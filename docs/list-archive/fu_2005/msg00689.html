         * * * Announcing Ferret Version 5.81 * * *

   * * * for Solaris and Linux * * *


Annoucing the official version 5.81 of Ferret. This is mostly a bug-fix 
release, and is being made now to coincide with the release of LAS 6.5.1
It is required for LAS 6.5.1 to run properly. Look for a Ferret release 
with more new features later in the summer of 2005.

With this release we introduce a statically linked linux executable for 
Ferret, which should run on many more of the linux platforms.

To install Ferret, download the file from the Ferret Web site or anonymous ftp site:

	Web address:   http://www.ferret.noaa.gov/
	anonymous ftp: ftp.ferret.noaa.gov


For a list of bug fixes and new features added in V5.81 see:
http://www.ferret.noaa.gov/Ferret/Documentation/Release_Notes/v581.html
