Here is a string with some messages from the archive which will answer part of this question. 

  http://ferret.pmel.noaa.gov/Ferret/Mail_Archives/fu_2003/msg00600.html

The grave accent, I believe, cannot be used because the command parser always attempts to evaluate the contents of the grave accent first, so it looks for a closing grave accent.  You would need to use a symbol font to draw that mark.

The reason for the inconsistencies Mick points out in his answer is that command parsing is done differently for Ferret comands as in  LET/title= and for PPLUS commands (LABEL is an alias for the PPLUS command %LABEL).

so, you can make the label you want with two single quotes

  label/nouser ($xpos_ac),($ypos_ac),0,0,0.10 @AS''
