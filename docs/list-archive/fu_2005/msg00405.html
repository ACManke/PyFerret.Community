In fact redefining line colors is useful in many situations, and you can
specify any RGB value. For example, I often redefine line 3 (green) to
be darker:

PPL PEN,3,0,60,0     ! original pen 3 is 0,100,0

Or brown:

PPL PEN,3,60,40,30

Or yellow (to overlay on a fill plot):

PPL PEN,3,100,100,0

It's easy to experiment with this.

The only problem (as discussed in this forum a year or so ago) is that
only 6 colors are available. A useful improvement in a future version of
Ferret would be the ability to have more definable lines.
