Hi David,
Ferret just isn't set up to have more than one mode in a session -- it opens a graphics "workstation" of a given type on startup and is then in that mode for the duration of the session. You might be able to save intermediate results, say in a NetCDF file, and then open up a new session to make the final plots in the graphics format that you want.

Ansley Manke
