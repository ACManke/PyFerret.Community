This is a problem I have run into as well. I work on one Linux box where the administrator is relectant to upgrade. For projects where I have to use that machine, I am unable to use the new regridding functions in Ferret, which I find to be a big limitation. It is really helpful having ready built binaries available when they work, but it would also be helpful for the Ferret developers to provide a link to the source code so that those of us with unsupported or older machines would be able to compile our own binaries as needed.
