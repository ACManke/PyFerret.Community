There have been several discussions of pen colors in Ferret over the years, which I think could be 
found in the mail archives under "pen color" or "line color". The bottom line is that it is possible 
to define any color you want, but are pretty much limited to 6 in any given plot (sometimes you can 
sneak in another few, with unpredictable results; see the archives).

Any pen color can be redefined in RGB values:

ppl color,5,44,0,50		! changes line 5 from cyan to dark purple
plot/line=5 variable_name

You should avoid changing color 1 which is used for plot axes and labels.
Sometimes you can redefine a color a few times within a single plot to get more than 6 colors, but 
after "too many" changes (see the archive discussion at: 
http://ferret.pmel.noaa.gov/Ferret/Mail_Archives/fu_2005/msg00558.html)
