Hello,

I've tried ferret 5.41 on IBM AIX 5.2. (P615/1,2Ghz/1024Mb RAM), but it cannot open large files (>2Gbytes). It seems that the open command used doesn't manage the file size in the struct associated with the file at the open.
Tests made in C (testing a open("fic",O_RDONLY) ) show the problem encoutered by ferret.
Note that changing open() by open64() or compiling in 64bit (with -q64) clears the problem.

Does any on has a working version of ferret on AIX compiled with -q64 ?
Is it possible to get a tarball of the lastest ferret source code ?
