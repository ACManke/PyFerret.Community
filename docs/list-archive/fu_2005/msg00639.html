Hi Jimmy,

I figured that thing out a few years ago: there is a particular order when you wish to combine shade (or fill or countour) with land.

First, you can add your own title within the shade command like:

shade/nolabel/title="Your title" /set_up/etc your_var

ppl shade

then

go land

all the ppl options should be set before the ppl shade command


it should work ;-)

Helene
