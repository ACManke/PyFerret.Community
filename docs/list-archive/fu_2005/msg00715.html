Hi, Duli:
Here is how do you do it:

use uwnd.mon.mean.nc
use vwnd.mon.mean.nc
set region/x=x1:x2/y=y1:y2/z=z1/l=l1
vector uwnd[d=1],vwnd[d=2]

That's all!
Good luck!
Muyin
