  1. Where should I store my NetCDF files in order for me to USE them?
The files can be anywhere, and you would point to them by including their full path names. If you include a directory, put the entire path and filename in quotes. For example the Ferret command

yes? set data "C:\tmp/newdata/mydata.nc"

You can also put the files in the default Ferret directory, /tmp.  If files are in /tmp then you can say,

yes? set data localdata.nc
