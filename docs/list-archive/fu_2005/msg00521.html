 Manke wrote:
Hi Yogesh,
A few suggestions:

o In your script, put the SET WINDOW/ASPECT command before the SHADE command. I think that putting it after the shade command is what is causing the plot to disappear when you're in gif mode.

o On the SHADE command, use SHADE/SET to put Ferret into setup mode. Then the PPL SHADE command will apply your settings to the plot.

Ansley

 Manke wrote:
Hi Yogesh
ferret -gif works for all plot commands. What operating system are you running Ferret on? This works fine, for example:

> ferret -gif
yes? shade/set/i=1:10/j=1:10 i+j
yes? ppl title "new title"
yes? ppl shade
yes? frame/file=mygif.gif
yes? exit

(Just a reminder to everyone, when you ask a question, especially about something that seems to be not working as advertised in the documentation, it really helps to know the Ferret version you are using, as Yogesh did, and also the operating system.)

Ansley
