Hi Sriram,

The NVODS Live Access Server hosts the WOA. Point your browser to http://ferret.pmel.noaa.gov/nvods?dset=By+Dataset+Name and scroll down to 'World Ocean Atlas'. There are various output products by which you can access the data, including downloading NetCDF files.

The NVODS WOA 2001 is a collection of NetCDF files, one per variable, created at CSIRO by Mark Collier. His contact information can be found by clicking the blue info icon next to the dataset name and then clicking on the 'documentation link'. I've also cc'd him this message.
