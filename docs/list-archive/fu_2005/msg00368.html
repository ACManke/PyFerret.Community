John,

 missing symbol is _libc_register_forkhandler and not
_register_forkhandler.

Maybe you would find it into the Sun C-compiler library, typically at 
/opt/SUNWspro/... in your directory tree ?
If it is the case, you will need to set environment variable
LD_LIBRARY_PATH to the library location.
