Hi Sriram,
You can define new X and Y axes with the 2 degree interval, and regrid the data to those new axes. Please look up "regridding" in the Users Guide. (Navigate to "regridding, syntax and examples" from the index, Chapter 4 Sec2.4) There is also a demonstration script that shows the ideas of regridding:

yes? go regridding_demo

Read the text that comes to your screen as the demo runs to learn how to use this important capability of Ferret.

Ansely
