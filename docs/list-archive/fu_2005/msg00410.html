Hi Jammal,
HDF formatted data can be read by Ferret, but only via DODS, so if the datasets are on a DODS server, or can be put on one, you can open it with a SET DATA (USE) command.  Some HDF files have a different ordering of axes, which needs to be fixed up in Ferret either on theUSE command with USE/ORDER=  or once you've opened it, by redefining axes.  Here's a quick example with USE/ORDER, where the initial data is upside down (for those of us used to looking at northwards = up!). Some HDF data contains longitude and latitude data which can be used to define a grid for the data.

You can also read binary data with Ferret, but the commands need to be tailored to the way the data file was written.
