Hi Benyang,
These files don't follow the COARDS or CF conventions which Ferret needs to
read HDF files via DODS. If you use a browser to look at the DODS .dds file, that is
point your browser to filename.hdf.dds:

http://g0dug03u.ecs.nasa.gov/dods-bin/nph-dods/datapool/AIRS_L2_PRODUCTS/AIRX2RET.002/2003.03.03/AIRS.2003.03.03.001.L2.RetStd.v3.0.8.0.G04137131312.hdf.dds

you will see that this is a structure of sequences. The DODS NetCDF client cannot
read sequences. One idea is to use the browser form to get ascii values, save them
to a local file, and then read those into Ferret. The link for that is filename.hdf.html

http://g0dug03u.ecs.nasa.gov/dods-bin/nph-dods/datapool/AIRS_L2_PRODUCTS/AIRX2RET.002/2003.03.03/AIRS.2003.03.03.001.L2.RetStd.v3.0.8.0.G04137131312.hdf.html

Ansley
