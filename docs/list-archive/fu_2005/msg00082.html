Takaya,

Are you opening the files one at a time? Since you are dealing with a time series, you may want to try
using a Ferret descriptor file to connect the files together so that Ferret sees them as one data set.
The details for descriptor files can be found in the Ferret documentation in Ch. 10 Sec. 4. I currently
use a descriptor file connecting 120 files so I know they can get at least that far.
