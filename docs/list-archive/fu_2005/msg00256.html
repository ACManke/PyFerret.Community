Hi Wendy,
	That is one you can patch together with other font elements.
Try
	label/nouser 4 6.3 0 0 0.2 @srEl Nin\^@ma45@sro
which draws the n in a regular font, backspaces with the \, goes to
superscript with the ^, then gets a tilde from the Math symbol font.
To complete the word you then revert to the original font @sr.

There may be other combos to get the same result.
