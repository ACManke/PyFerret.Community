Hi Ramesh,
Ferret can create graphics using this kind of grid, which we call curvilinear data in the documentation. You can make a shad plot of variables rain or evap with the command

shade rain[L=1], nav_lon, nav_lat

fill evap[L=20], nav_lon, nav_lat

There are also functions to regrid data from a curvilinear grid to a rectilinear grid: CURV_TO_RECT_MAP and CURV_TO_RECT. These functions allow you to do analysis or comparisons with data on a retilinear grid.

Ansley
