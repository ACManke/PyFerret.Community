first find out what grid you're on:

yes? show grid myvariable

where myvariable is your variable name in the netCDF file.

To put the data on a new grid, you'll have to define a grid, see
"define grid" in the manual.

http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm

You'll have to define axes, see:

"define axis"

And finally you'll have to regrid to the new grid using the @ASN option.


Ch4 Sec2.4. Regridding
