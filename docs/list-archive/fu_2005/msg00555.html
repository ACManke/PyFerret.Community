You would need to mask the data to include only the data in the ocean basin, then use the ideas in the FAQ on averaging/integrating over irregular regions.

http://www.ferret.noaa.gov/Ferret/FAQ/analysis/avg_integ_irreg.html

Recently Jonathon Callahan made some "ocean basin" masks available; see this message to the Users' group, which should be just what you need.

http://www.ferret.noaa.gov/Ferret/Mail_Archives/fu_2005/msg00401.html
