Hi Ronaldo,
There are some scripts that make histograms and bar charts - maybe you can use the ideas in these scripts to do what you want. Use the Unix command Fgo to find the names of the scripts

> Fgo histogram
> Fgo bar_chart


And in Ferret, go/help lists the start of the script so you can see what the arguments are and how to call them:

yes? go/help frequency_histogram2.jnl ...
