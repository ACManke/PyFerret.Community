Hi Jimmy,
Is your data 1-D in X? If so you can send XSEQUENCE(var) into the FFT functions and that's it. If you want to set yourself to write and compile external functions, the code for all the FFT functions is available and it'd be a quick change to make the function operate on another axis. See the Ferret external functions web page at

http://www.ferret.noaa.gov/Ferret/External_Functions/ferret_EF.html

where there are links to the source code for external functions. You'd want the link for "internally linked external functions" about 3/4 of the way down the page.
