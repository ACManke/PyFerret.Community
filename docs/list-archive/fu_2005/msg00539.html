Hi Steve,
It all depends on how the data is arranged in the ASCII file.  The way you've set up the grid and the FILE command, Ferret expects the data to be listed with one value per record, with X varying fastest.  So, the commands you have set up expect this in the ascii file:

r(x1,y1)
r(x2,y1)
r(x3,y1)
...
r(xn,y1)
r(x1,y2)
r(x2,y2)
...
r(xn,y2)
r(x1,y3)
r(x2,y3)
...

If your data is arranged differently, then you need different commands.  See "ascii data, accessing" in the Users Guide.
