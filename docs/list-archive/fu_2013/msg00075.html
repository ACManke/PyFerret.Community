Hi,

I had to concatenate global daily precipitation from 1998-present. With the help of Ansley I was successful. I have attached an excel file of what I did for 1998-2011 more than 5100 files ( I have also similarly worked with more than 5400 files). Please see the attached excel file and see if it can be of help to you. What I did is open the first file and save it with a new name  then cancel all open data and open the next one and concatenate it to the new first and so on until all are concatenated to the same file. You can write a similar script in what is convenient for you. In the attached excel file, the first sheet is where I prepared the script and the second sheet is the script that I saved as .jnl to run in Ferret.
