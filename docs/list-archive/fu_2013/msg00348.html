Your question is not clear to me, there is one more provision in ferret for sorting something like this you can try

! whatever variable

let order1 = SORTI(lat)
let var = SAMPLEI(temperature, order1)
then you can list it

I have created a .nc data file with taxis as independent variable,In this data file one parameter is day1[l=1:2531],lat11 and lon11 in -7 to 40 and 60 to 100,But I want to sort out(extract) other parameters for lat11 and lon11 in 20 to 30 and 80 to 95 respectively.how is it possible??

Use can try this :
yes? use <your file>.nc
yes? list/file=<output file>.nc/format=cdf/x=80:95/y=20:30  <variable name>

Check this link
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/commands-reference/LIST
