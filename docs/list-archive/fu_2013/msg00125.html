This should work. Are you putting the TXLINT command between PLOT/SET and the final PPL PLOT command?
This is not a "persistent" command (like PPL AXLINT, for example). You must give it each time. 

BK

PLOT_or_SHADE/SET ...
PPL TXLINT,0,10
PPL PLOT_or_SHADE
