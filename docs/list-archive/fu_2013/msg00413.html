Hi
If you want to apply a condition to a variable that's not a scalar then the concept you want is a mask.  Try it on a couple of variables to get a feel for it.
let lat_le30_mask = if lat LE 30 then 1
list lat_le30_mask

let month30_mask = month1* lat_le30_mask
list month30_mask
It looks like you might want to do something else though - define the IF condition using the scalar variables. 

let latmax = max(lat1,lat2)
...
if `latmax le 30` then
