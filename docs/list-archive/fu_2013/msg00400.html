This is a known error, I do not know the reason.

The workaround is, to prepare a script, that generates the plot, say myscript.jnl.
If ready and everything is fine, quit ferret. Add the "frame" command as the last line of the script.

Now start ferret again:

ferret -gif -script myscript.jnl

This generates the desired gif image.

Hope this helps,
Martin
