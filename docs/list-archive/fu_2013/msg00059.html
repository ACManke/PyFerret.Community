FISSEHA G. BERHANE wrote:

> I was wondering how I can show station locations on a map using symbols.

plot /overlay /symbol=20 /color=blue {4.5,7.3},{50.3,52.6}


will plot 2 symbols, one at 4.5,50.3 and one at 7.3,52.6 .  
You can use the label command to also add station names.
