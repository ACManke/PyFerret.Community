Hi Raden Bima Yoga Barata -

The gif-images are for quick-look plots. You can make vector graphics from ferret using the Fprint utility from the command line:

yes? set mode meta
yes? [commands to make your plot]
yes? ppl clsplt
yes? can mode meta

This will produce a file called metafile.plt.

>From the command line (outside Ferret):
Fprint [options] -o filename.ps metafile.plt

This will produce filename.ps, which is a vector graphic that can be enlarged indefinitely. You can now use ps2pdf or whatever other utility you want to make other formats.

See the documentation for Fprint for the options, also see the user list for many contributions to thicken lines or otherwise improve the postscript output.
