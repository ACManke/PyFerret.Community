Hi Marco,
I also do not get always the behaviour of labelling:

yes? label/nouser/ov 6.5,-0.88,0,0,0.22 "C'1" ->  **ERROR Unmatched '
yes? label/nouser/ov 6.5,-0.88,0,0,0.22 "C''1" -> C'1 ! this is what I want. Wonder why double 's were needed.
yes? label/nouser/ov 6.5,-0.88,0,0,0.22 C''1 -> this one has the same effect as the previous one. Strangely, quotations were not needed.
Also,
yes? label/nouser/ov 6.5,-0.88,0,0,0.22 C^1 -> C1 (1 is uppercase, good to know), but it works fine if I put two ^s, as well.
Also, Ferret does not like % between "".

Getting back to your problem, I think you should try avoiding "", it worked for me.
yes? label/nouser 1 2 0 0 1 @P2 test

I am using FERRET v6.67.

Peter
