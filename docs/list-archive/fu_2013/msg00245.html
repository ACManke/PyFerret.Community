Dear fellows, I'm trying to isolate the velocity field of a water parcel through a density interval. The problem is, when I try to plot this field, it returns the following error:

yes? fill uvel

**ERROR: command syntax: Unknown transformation:5500
          Use SHOW TRANSFORMS for a list of legal transformations

What an I doing wrong??

I'm using the Ferret 6.8 version.



!Rotina para plotar o campo de correntes no AN referente a APAN

!Temperature data
use thetao_O1.200101-201012.nc        !d=1
let temp = thetao - 273

!Salinity data
use so_O1.200101-201012.nc        !d=2

!Zonal velocity data data
use uo_O1.200101-201012.nc        !d=3

!Meridional velocity data
use vo_O1.200101-201012.nc        !d=4

!set mem/size=74

set reg/x=-100:20/y=0:80

let dens=rho_un(so[d=2],temp[d=1],0)

let sigma = dens[g=uo[d=3]]

let u=uo[l=1:120,d=3]
let v=vo[l=1:120,d=4]

let water1 = if (sigma ge 1027.6 and sigma le 1027.8) then u[d=3]
let water2 = if (sigma ge 1027.6 and sigma le 1027.8) then v[d=3]

let uvel = water1[x=-100:20,y=0:80,z=0:5500@int,l=1]
let vvel = water2[x=-100:20,y=0:80,z=0:5500@int,l=1]
