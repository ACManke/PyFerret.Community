Hi,
This is a hard-coded limit within Ferret, so it's not possible for you to increase that number. Usually there are ways to let go of variable definitions that are no longer needed. It sounds as if you are stepping through a set of files. Once you have used data from the January file, for example, if you don't need that data any more, you can "cancel data" to close that file and recover storage indices for the variables in that file. Use "cancel variable" to remove all or some of the variable definitions you have made and restore storage for user-defined variables.
If you don't see a way to handle this immediately, you are welcome to send your commands, or a simplified version of your script and people here on the list will probably be able to help you find ways to get it to work. 

Ansley
