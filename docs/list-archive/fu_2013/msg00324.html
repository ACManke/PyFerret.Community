Hi ! One more easy way you could try

let ii=0
let model={spawn: 'ls *.nc'}
repeat/range=1:no of your file ( let ii=`ii+1`;\
let dummy=model[i=`ii`];\
use dummy;\
sh d)

you are getting error message because you are not canceling last used data

use canc data1 before the end of loop

Hope it will work
