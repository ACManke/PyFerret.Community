Dear Nitin,

Take a look in the following documentation page:

9.4 HARD COPY AND METAFILE TRANSLATION
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/commands-reference/computing-environment/HARD-COPY-AND-METAFILE-TRANSLATION

That is, use the "set mode metafile" command to output your graphics to PLT format and later (outside Ferret session) use Fprint script to produce your postscript images.

If you are using PyFerret, I think it's better to follow the instructions given here:

Graphics in PyFerret
http://ferret.pmel.noaa.gov/Ferret/documentation/pyferret/graphics/
