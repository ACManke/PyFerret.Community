Hi all,

I am glad to announce you that you can now use
some ferret code within the IPython Notebook
a web-based interactive computational environment where
you can combine code execution, text, mathematics, plots and rich media
into a single document.
Your plots are automatically embedded into the notebook document.
You can control size and antialiasing of plots and produce pdf as well directly 
from execution of the cells.
And of course, objects can be passed from ferret to python and vice versa.

If interested, please read http://ipython.org/notebook.html
and extensions that are available : https://github.com/ipython/ipython/wiki/Extensions-Index 
You will find from there the ferret extension ready to be installed
and some notebooks to help you to start.

Beside this small ferret extension I have coded, there is of course
the great work on the pyferret module coded by Karl Smith and
the work of the NOAA/PMEL team on ferret.
Thank you for them.
