On Fri, Jan 4, 2013 at 10:25 AM, Ansley Manke <ansley.b.manke@xxxxxxxx> wrote:
Hi Luke,
You can set the aspect ratio of a plot.  The command is SET WINDOW:ASPECT, and in the documentation have a look at the :AXIS option for that command. It's probably what you want to use.  Your plot will look fine with an aspect ratio of 2, or you could set it using the limits,

yes? let asp = (51.6180 - (-52.4828))/(-33.7500 - (-84.3750))
yes? set win/asp=`asp`:axes

ansley
