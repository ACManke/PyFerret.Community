Hi Ektara,
The NetCDF version 4 library which Ferret uses for reading netcdf files and OPeNDAP URL's, can read HDF-5 formatted files. There are HDF-5 files that have features that aren't included in the netCDF data model and those cannot be read. So the short answer is that if your file is compatible with netCDF-4, then Ferret can open it and read it without your having to know anything more about it. Try, 

yes? use filename
or

yes? use "url"


For more information, have a look here,

http://www.unidata.ucar.edu/software/netcdf/docs/faq.html
at the topics that refer to HDF for an overview of the issues surrounding HDF files. The NetCDF library can read many but not all HEF-5 files. 

Here are a couple more documentation links,
http://www.unidata.ucar.edu/software/netcdf/docs/interoperability_with_hdf5.html
http://www.unidata.ucar.edu/software/netcdf/docs/parts_of_netcdf4.html\

-Ansley
