    * * * Announcing the official release of PyFerret Version 1 and Ferret version 6.85 * * *

We are pleased to announce the official release of PyFerret Version 1, and "classic" Ferret V6.85

If you've been running a Pyferret beta version, please check the release notes. Your feedback has resulted in several changes including the settings for graphics windows; and lots of improvements overall. 

If you have not yet tried PyFerret, please do! To see what it's all about, have a look at "What is PyFerret?" and "Why Use PyFerret?". There's lots there for the Ferret user whether you know Python or not.  PyFerret includes all of the latest improvements that are in classic Ferret. And your existing Ferret scripts will run fine. The full PyFerret documentation is here, http://ferret.pmel.noaa.gov/Ferret/documentation/pyferret, where you will find the Downloads page.

We also release a classic, non-python Ferret. The release notes are here:
http://ferret.pmel.noaa.gov/Ferret/documentation/release-notes/version-6-85-release-notes


Ferret v6.85 Highlights include:
  • New commands to examine the definition and dependencies of variables. Is the variable defined? What variables and datasets does it depend upon?

  • Enhancements when defining Ensemble datasets with DEFINE DATA

  • A small change, using PLOT/COLOR to plot multiple variables together as a set of colored lines

  • Ferret v6.85 is linked with NetCDF version 4.3


Ansley Manke, Karl Smith, Steve Hankin
