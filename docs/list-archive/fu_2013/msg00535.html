Hi Nitin,
NetCDF-4, which is linked with Ferret, includes reading and writing of HDF-5 formatted files. Have you tried just opening the file with Ferret? If the file is HDF-5, it will open it as if it's a netCDF file.  If the file is HDF-4 formatted, then I believe current releases of Ferret will not open it.  Akshay's suggestion might be the way to go in that case.
