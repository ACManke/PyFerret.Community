Hi Ashkay,
For the issue of the Z axes of these profiles, you could define a Z axis and regrid the variable from each file to it.  But we have talked with you before about the larger issue; the profiles are at different locations and times. The x-y locations do not represent a grid, so you can't just append them all into a file and have a gridded dataset.
