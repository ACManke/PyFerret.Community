Hi Matthew,
I haven't tried your commands, but what you've done look fine, except that you'll need a 

     /SKIP=6

on the FILE command to skip over the header lines.
To assign a date to the data coming in from each of these files, you need the concept called "timestamping". Essentially you define a time axis, and then add 0*time to the variable at that time step, to give it a time. This is described here, 

http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/commands-reference/SAVE#timestamp

-Ansley
