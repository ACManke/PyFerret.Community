Sudheer Joseph wrote:

> > Or is there any way to regrid a monthly climatology data to a daily
> > climatology data? Simply i need to repeat the values for all the days of
> > the month.

This can be done using the @ave regridding transformation:


define axis /t="01-jan-2000 12:00":"31-dec-2000 12:00":1 /unit=days my_t_d 
define axis /t="15-jan-2000":"15-dec-2000":1 /unit=months my_t_m

let myvar=sin(t[gt=my_t_m])
let myvard = myvar[gt=my_t_d@ave]

plot myvar, myvard
