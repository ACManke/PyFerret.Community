Hello,
For an /OVERLAY plot, the axis limits have already been set by the original plot command, and therefore /VLIM has no effect.  If the data needs to be reversed, then one could use a function such as XREVERSE or ZREVERSE. 

I agree with Patrick that defining the vertical axis with a /depth qualifier is the best solution, so that the data is treated correctly throughout.
