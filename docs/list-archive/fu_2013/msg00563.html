Hi Marco,

I think you should set a depth qualifier on your vertical axis (z-axis).
From documentation http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/commands-reference/SET 
it is said that :

/DEPTH
Indicates that an axis is to be treated as a depth axis (graphics made with positive down). 
yes? SET AXIS/DEPTH z_ax
