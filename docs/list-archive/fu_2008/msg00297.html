Peter Kim a écrit :

Hi Pierre


Please check S_FILENAME in your descriptor file.
If you run ferret in a directory not including data files, you need to write a whole directory. 
:
:
S_FILENAME         = 'whole path of the directory/something.nc'
:
Otherwise, the descriptor file try to read data files in the ferret running directory.
