PLOT/SYM=nn variable

See the two font tables at:
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/ appendix-b-pplus-guide/COMMAND-DESCRIPTION#_VPID_752 

The link for "Symbol fonts" is broken, but the list is actually at:
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/appendix- b-pplus-guide/COMMAND-DESCRIPTION#_VPID_752
(Note to Ansley: remove " #_TN_Ref_viewaxes_b" from the link in the first page listed above).
Then click "Plot marks" for a listing of the symbols available for use in the PLOT command. 

Note that you can get both symbols and a line with:

PLOT/LINE[=line_num]/SYM=nn variable,
where line_num is a number from 1 to 18 (the default is 1). The same color/line weight will be used for the symbol as for the line. 

To color a symbol plot alone, specify color:

PLOT/SYM=nn/COLOR=col_num variable

where col_num is a number from 1 to 6 (maybe 1 to 18?).
