Dear all,

I'm trying to access the Level 3 QuickSCAT gridded vectors data using OpenDap on ferret.
However, the grid is not organized in degrees of latitude and longitude. How can I handle this on Ferret?
Best,
Domingos.

yes? list ASC_AVG_WIND_VEL_U[x=38w,y=-5]
 **ERROR: illegal limits: ASC_AVG_WIND_VEL_U is not in the range Y=-5
          Axis extremes are Y=0.5:720.5
yes? show grid ASC_AVG_WIND_VEL_u
    GRID GPO3
 name       axis              # pts   start                end
 FAKEDIM5  X                 1440 r   1                    1440
 FAKEDIM4  Y                  720 r   1                    720
 normal    Z
