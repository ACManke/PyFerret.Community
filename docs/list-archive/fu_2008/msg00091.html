Hi,

Thank Ansley's workaround and explanation, as well as Andrew's history lesson. I later found yet another trick as the following:

let tstart = 0`temp,return=tstart,p=1`

which avoids null string meanwhile doesn't affect those larger than 0001 (e.g., 01985 == 1985).

Thanks,
D.

track.  (It would be nice to get the issue fixed, though.)
