Hi Andy,
Are you looking at the FAQ about correlations?
http://ferret.pmel.noaa.gov/FERRET_17sep07/FAQ/analysis/correlations.html

The regress* scripts are designed to do correlations of 1-dimensional variables. If the two regions you want to correlate have the same number of points, x+y+t, then you can unravel the points into a single list and call one of the regression scripts (specify regions with either index I= ... or coordinate X= ... )

yes? let p = XSEQUENCE(var[x=301:321,y=21:33,L=1:12])
yes? let q = XSEQUENCE(var[x=251:271,y=1:13,L=1:12])
yes? go regressx
If you want to use the variance script, you can as long as the two variables you are comparing are "conformable" so they can be combined in the expressions that the script defines.
yes? let p = var[x=301:321,y=21:33,L=1:12]
yes? let q = var[x=251:271,y=1:13,L=1:12]
yes? go variance
For a discussion about conformable variables, do a search for the word "conformable" in the Users Guide at http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide, and choose the result that points you to Chapter 3.
