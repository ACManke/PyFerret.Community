There are also a couple of scripts included with the Ferret distribution, legend.jnl and legline.jnl.  To see what these do and how to call them, type:

yes? go/help legend

yes? go/help legline

There is also a demo script for legend.jnl, called test_legend.jnl

yes? go test_legend
