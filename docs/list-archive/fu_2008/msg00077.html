Dear Dr. Sudheer,
You could try to define an alias that sources the ferret_paths and then calls ferret, e.g. with 
alias ferret_start "source /usr/local/ferret/ferret_paths ; ferret "
if your shell is csh or with
alias ferret_start="source /usr/local/ferret/ferret_paths ; ferret "
if your shell is bash

To start ferret you simple type the name of the alias e.g.
ferret_start --help

Best regards, Ingo
