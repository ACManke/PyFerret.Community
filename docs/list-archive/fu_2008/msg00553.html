Hey Jayakumar,

use climate data operators. the page is at:
http://www.mpimet.mpg.de/fileadmin/software/cdo/

after proper installation the command should be something like this:

cdo -r -f nc copy inputfile.grib outputfile.nc

(If the projection is in Lambert then you wont be able to use the CDO either. but theres also a trick for that.)

Hope this helps,
Peter Szabo
