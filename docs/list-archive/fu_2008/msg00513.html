Hi Ferret-users,

I am trying to get the axis max/min values using 'return' command.
Here is an example (I am using Ferret V5.8):

set data NAGa0.01_3d_016_MNA.FILE

let blon = `u[d=1],return=xstart`
 !-> DEFINE VARIABLE blon = 47.52002E
let elon = `u[d=1],return=xend`
 !-> DEFINE VARIABLE elon = 59.68005E
let blat = `u[d=1],return=yend`
 !-> DEFINE VARIABLE blat = 30.73087N <===== NOTE HERE
 **ERROR: command syntax: 30.73087N
          .73087N
DEFINE VARIABLE blat = 30.73087N

yes?

yes? say `u[d=1],return=yend`
