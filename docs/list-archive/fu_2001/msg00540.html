Hello Weiq,
You can do this using the SAMPLEXY function.   If you have X and
Y locations for the coastline or other path, you can sample your data at
those points using SAMPLEXY and plot the result.  There is an example
of this in the Users Guide, see SAMPLEXY in the index. It is in
Ch3 Sec2.3.34.
 http://ferret.pmel.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm
