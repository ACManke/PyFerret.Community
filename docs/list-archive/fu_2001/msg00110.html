Hi Zuojun,
	A way to save portions of lines that go outside the plat window
is to use the "ppl window,off" command.  Normally the ppl window is ON
meaning that plot elements other than labels are chopped at the axes.
So for a demo
	        shade/i=3:8/j=3:8 i+j
                let u=10*(i+j-j)/i ; let v=10*(i-i+j)/j
		ppl vecset,2,1  ! to make the vectors longer
		vector/o/i=1:10/j=1:10 u,v ! some truncated, some missing
		ppl window,off
		vector/o/i=1:10/j=1:10 u,v ! now all are visible
