Francisco,

You get the GKS not in proper state when the DISPLAY environment variable is
not set properly.  For example, if I telnet to another machine I can run
ferret and list output.  But I get the "GKS not in proper state" error
unless I do the following:

setenv DISPLAY computer.I.sit.at:0.0


-- Jonathan
