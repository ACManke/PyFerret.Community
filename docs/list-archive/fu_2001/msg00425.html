Hello Hyun-Suk,
There are a number of scripts included with Ferret which set up
map projection parameters.   One is mp_lambert_cyl.jnl, which
sets up variables for the "Lambert Cylindrical Equal Area"
projection.   These scripts are part of the Ferret installation starting
with Version 5, and you could start with  mp_lambert_cyl.jnl and
modify it if the projection you need is different.   Try

     yes? go mp_demo

for a sample of some of the map projection scripts.
