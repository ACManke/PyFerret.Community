Hi Ahrens,

When you say "I can only go to a maximum of SET MEMORY/SIZE=30" I assume
that some error occurs at larger size.  Can you tell us what that error
behavior is?

Note that the "/SIZE=30" is measured in megaWords --120 Mbytes.  This is  in
addition to the basic code size of Ferret and memory used by system and
other applications.  Are you sure that you have adequate virtual memory
(paging file) allocated to your system?
