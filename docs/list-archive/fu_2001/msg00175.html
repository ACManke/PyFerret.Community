Hi Glenn,
	In a situation where you have a known number of contours
with known values (not necessarily the same as the underlying
data) it may be easiest to generate the labels using "label/nouser".

	The following demo puts labels like you have in example.gif
on an artificial data set

fill/i=1:10/j=1:10/set i+j
ppl shakey,,,0.001,100 ; ppl fill ! turn off the color bar labels
repeat/i=1:9 label/nouser
                 8.7,`i*($ppl$ylen)/10-0.05`,0,0,0.1,@sr`20*i+200`
label/nouser 8.7,`($ppl$ylen)-0.05`,0,0,0.1,@sr^0K ! a units label

In this way one could associate text strings with the color values too.
