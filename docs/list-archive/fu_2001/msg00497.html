Hi Jolene,
For the PC installation, "Fenv" does not appear to be enabled.   Your
PC's default browser setting is what isn't being found.  When you
you type "help" at the Ferret command line, your browser should open, and
you'd be pointed to the Ferret web page, http://ferret.wrc.noaa.gov/Ferret/.

Does Ferret work otherwise?  Try typing the Ferret command:

yes? go tutorial

You'll need to read the text in the command window and also view plots
that should pop up on your screen.   Welcome to Ferret, and the users
group!

Ansley Manke
