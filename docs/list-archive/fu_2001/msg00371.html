Hi Gaorz and Mick,

The "ASCII Simplex" (@AS) font also works well for special characters such
as %, >, <, @, \, etc:

label/nouser 1 1 0 0 .2 "@AScontour >90% signif"

However, this is only a good option if you have no more font changes after
the @AS, since after that, all @ signs are interpreted literally.  So the
following does not work:

label/nouser 1 1 0 0 .2 "@AScontour >90% signif for @SRv^2"
