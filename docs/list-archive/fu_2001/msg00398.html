Hello Ferret Users:

We have created a new download page for Ferret BETA releases. See
    http://ferret.wrc.noaa.gov/Ferret/Downloads/ferret_downloads.html
The BETA versions are typically very stable -- almost production
quality. This goes for V5.33beta, which you will find available for
several systems.

Documentation of the new features to be found in V5.33 may be scanned in
the release notes at

http://ferret.wrc.noaa.gov/Ferret/Documentation/Release_Notes/v53.html

We encourage you to download and use BETA versions of Ferret and send
feedback to us at ferret@pmel.noaa.gov.

Ferret official V5.3x has been delayed in order to incorporate the
latest release of the DODS networking software.

    - the developers
