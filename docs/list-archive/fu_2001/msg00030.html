Hi Jose;

There a program that automatically generates fortran code to read a
specified netCDF file.  It's called "gennet" and is available from 
http://www.unidata.ucar.edu/packages/netcdf/contrib.html.  Its fairly
primitive, and will make some minor mistakes sometimes, but saves you from
having to learn netCDF calls if you don't have time to do so.

Other lists of netCDF utilities are available at
http://www.gfdl.gov/~jps/GFDL_VG_NetCDF_Utils.html and
http://www.unidata.ucar.edu/packages/netcdf/software.html.  Perhaps you
can find a more modern or efficient code generator.
