Hi Billy,

You need to have a lot of Perl modules installed on your
system to run LAS. Check that Storable is installed.
You also need (at least for LAS-4.0) :
URI::URL
XML::Parser
HTML::Parser
MD5
Date::Calc
Time::JulianDay
Date::Manip
LWP::UserAgent
CGI

To get these modules, type perldoc CPAN and follow the instructions.
