Hi Viviane,
The Ferret EOF functions are not now available for the windows
version.   They are written as external functions and not included
in the group of statically-linked functions which are available under
Windows Ferret.  (I see you were part of a previous mail thread
on the EF's for windows, so you know what the issues are.)

I hadn't planned to link the EOF functions into the main Ferret
executable yet.  I'll let you know what the plans are for when
and if  we'll do it.

Ansley Manke
