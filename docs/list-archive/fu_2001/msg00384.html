Personally, I would be happy just with the capability to list the 
high/low values and their locations on the plot to my ferret session. 
Then it would be trivial to make these labels manually, in precisely 
the size/format/font/position desired. For instance, I cannot imagine 
an automatic procedure that would place such labels so as not to 
overrun contour labels, land boundaries, dense regions of contours, 
etc (e.g. the analogous NCAR contour package capability which only
gets the right position by chance), whereas the human eye can easily 
tweak the position for best readability.
