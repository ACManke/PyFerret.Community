Hi Yiquan,
Here's an example, which uses the L coordinate in the name of the files:

     yes? USE coads_climatology
     yes? SET REGION/X=180:200/Y=0
     yes? REPEAT/L=1:12 (LIST/FORMAT=(F9.2)/FILE=time`L`.dat sst)
