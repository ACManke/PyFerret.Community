I ended up doing it the simplest way, ie, a repeat loop over 'l' with drawing
a label at the correct place. Worked just fine:

repeat/l=1:11 (set reg/l=`l` ; label `t[g=toaflx3]-($XAXIS_MIN)` `toaflx3` 0 0 0.1 @P2@PM22)
