Hi Kun,

The /SIZE= qualifier allows you to control symbol size in Ferret V5.22.

Compare
   yes? plot/vs/i=1:5/sym i,i^2
with
   yes? plot/vs/i=1:5/sym/size=.2 i,i^2

    - steve
