Hi Pam,

Please see the discussion of "pen" colors in
http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/v50/chapter6.html#_1_167.

An example of controlling symbol color is:

     ! default symbol color
     yes? plot/vs/xlim=1:10/ylim=1:10/symbol=2 {6,3,5},{5,3,7}

     ! red symbols
     yes? plot/vs/xlim=1:10/ylim=1:10/symbol=2/SET_UP
     {6,3,5},{5,3,7}
     yes? ppl pen 1,2    ! RED
     yes? ppl plot

    - steve
