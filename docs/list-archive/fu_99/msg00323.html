Graham,

Try this

let plot_area_width = ($ppl$width)

or just use the symbol in the ppl shakey command.

ppl shakey ...,`($ppl$width)`,...


The documentation for this is at:

http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/v50/chapter7.html#_1_190


-- Jonathan Callahan
