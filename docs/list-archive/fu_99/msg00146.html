Hi Jozef,

The ASCII Simplex and ASCII Complex fonts allow you to use all of the
ASCII characters.

Examples:

  yes? GO ptest
  yes? label 0, 0,-1,0,.2 @ACHere is %
  yes? label 0,50,-1,0,.2 @ASHere is %
