I defined two new palettes from the default:

low.spk:
    0.0   80.0    0.0  100.0
  100.0   80.0    0.0  100.0

high.spk:
  0.0   60.0    0.0    0.0
100.0   60.0    0.0    0.0

Then I do:

yes? shade/levels=(-1.5,1.5,.1) temp
yes? shade/over/nolab/pal=low  if temp lt (-1.5) then temp
yes? shade/over/nolab/pal=high if temp gt ( 1.5) then temp
