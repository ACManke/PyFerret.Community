> Is there any possibility to generate the key step by step? 

You can make a key of any size or position you want through
the ppl shakey command. You can also control the color scheme
exactly through the ppl shaset command.

So a (tedious) way of getting what you want is to do several 
shade/overlay commands, each showing a fraction of the range 
of values you want. Each shade will need a custom color scheme
and a custom-made key (arranged to fit together so the total 
looks like a single key).
