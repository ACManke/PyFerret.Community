The most likely problem is a coordinate error in your time axis in the file
shum.94.nc

The way to check this is to examine the grid grille_h, especially the time
axis.
Just do the following:

use grilles_gcm.nc
use shum.94.nc
define grid/like=shum[d=2]/x=grille_v[d=1]/y=grille_u[d=1] grille_h
show grid/l grille_h

And make sure that everything is as you expect.

Mark Verschell
