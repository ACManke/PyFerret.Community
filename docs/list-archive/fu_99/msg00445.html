Steve Hankin found a little bug in ferret.el which makes vector and variable not
recognized as keywords. This will be repaired in a future release of ferret.el,
but if you want to fix this yourself then simply replace the line:

             "v\\(ari?a?b?l?e?\\|ect?o?r?\\|iewp?o?r?t?\\)\\|"

with

             "v\\(ari?a?b?l?e?|ect?o?r?\\|iewp?o?r?t?\\)\\|"

(or place \\ before |ect)

And then recompile the file.
