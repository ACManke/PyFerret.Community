Hi Andreas,

Sorry for the delay answering.

The commands you show aren't valid as-is. PLOT/SET_UP must be followed by an
actual plot before an overlay can be done. Specifically, the commands should
be

     plot/line=13/nolab/set_up/i=1:10  1/i
     ppl plot
     plot/line=15/nolab/set_up/ov/i=1:10  2/i
     ppl plot/over

Curious why this **did** work in V4.9 ...
