Hi Haijun,

In directory $FER_DIR/doc you'll find documentation.
e.g.

     grep -i TXLABP $FER_DIR/doc/pplus_manual.txt

You can also give the Ferret command

    yes? PPL echo on

then do the standard Ferret plot and quit Ferret. The file
fort.41 will contain all of the axis set-up commands sent
from Ferret to PLOT+ -- providing examples of how to use
the PLOT+ commands.
