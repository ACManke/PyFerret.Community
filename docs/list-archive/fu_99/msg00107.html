I have a user who is running into problems with what appears to be
a limit on the number of axes/grids in use. He has a large collection

file for some variables.

Have I missed something? If not, then I guess I'm asking if this could be
added in V5 (out real soon now, Steve...?) or considered as a candidate
for later updates. Something like cancel grid? I guess with source code
to be available the limit will be able to be raised via recompilation but
