Hi Graham,

When you define a viewport you can specify the "/TEXT_PROMINENCE" size. The
default size (if you do not specify it, yourself)  is the geometric mean of
the x and y fractional sizes:
    /text = ((xhi-xlo)*(yhi-ylo))^0.5    ! default calculation

Basic documentation on DEFINE VIEWPORT is at
http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/v50/partII.html#_1_250

I hope this helps - steve
