*** Ferret V5.0 (BETA) Available on Windows NT/9x ***

The Windows version of Ferret version 5.0 (beta 1.1) is now available
for download and installation. Installation instructions and release
notes are available at:

http://ferret.wrc.noaa.gov/Ferret/Downloads/ferret_installation_win32.html

General information on Ferret is available at:

http://ferret.wrc.noaa.gov/Ferret/

Be sure and read the software and hardware requirements before
downloading Ferret. In particular, note that you must have a X server
running on your PC.
