  **** Ferret V5.0 (Beta) Available for Solaris 2.5 ****

Functionally identical to the previous Ferret version 5 binaries that we
have released, this version resolves shared library incompatibilities
between the Solaris 2.6 and Solaris 2.5 operating systems.

Our thanks to Glenn Carver for his assistance with this port.

Note: this is the version to use ONLY if you are running Solaris version
2.5.  The Ferret Web site has a separate binary for Solaris version 2.6.
