It would be tricky, like using viewports that "touch", but could you use
"ppl shakey" to do each key and have them look continuous?

a la:

shade/set/lev=(0)(10)(20)(30)(42)(58)(75)(95)(120)(150)(185)(225)(275)(335) var
ppl shakey [bunch of args, including x1 y1 x2 y2]
ppl shade
