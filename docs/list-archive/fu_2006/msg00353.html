Hi Vivek,
Here are links to the COADS data information
  http://www.cdc.noaa.gov/cdc/data.coads1.html
  http://icoads.noaa.gov/coads1a.html

The coads_climatology data set that is released with Ferret is the mean of the coads monthly data set from January 1946 to December 1999.

The Levitus climatology data is the Levitus annual climatology documented here:
http://ferret.wrc.noaa.gov/LAS_docFiles/levitus82_ancl.html


To find datasets and view their metadata, some good resources are listed on the Ferret OPenDAP page. http://ferret.pmel.noaa.gov/Ferret/DODS/ferret_dods.html

Then any data set available on a DODS server may be used in Ferret by specifying its DODS url:

  yes? set data "http://ferret.pmel.noaa.gov/cgi-bin/nph-nc/data/coads_climatology.nc"
