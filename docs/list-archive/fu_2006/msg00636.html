Hi Zhen,
           If your ferret_paths file is proper, there should not
be any problem with the new .spk file added to your 
~ferret_v5.81/ppl/ directory. Please check the following lines
in your ferret_paths file : toward the begininng, the defenition
of FER_DIR, it should be

   setenv FER_DIR /home/zhen/ferret_v581

Then towards the end of ferret_paths, FER_PALETTE is defined & it
should be like

  setenv FER_PALETTE          ". $FER_DIR/ppl"

Sometimes in your ferret_paths file, executable will be pointing to 
/home/zhen/ferret_v581/bin and palettes will be to some other old
versions of Ferret. Check it out.

Regarding the "color-friendly-palettes" : All of them should be 
present in the ferret5.81/ppl directory by default. If not get 
fresh tar.Z files from pmel/noaa site and install it again.

Hope this helps,

Regards,

Jaison
