On Sun, 3 Sep 2006, Stephen R. Guimond wrote:

Hello,
Don't use "t" as one of your variable names. Ferret confuses this with the time axis. Others variable names to avoid: "i" or "x", "j" or "y", "k" or "z", "l" or "t". From the documentation on abstract expressions:

"An expression which contains no dependencies on any disk-resident data is referred to as "abstract". For example, SIN(x), where x is a pseudo-variable."
