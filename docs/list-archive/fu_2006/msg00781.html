Hi Steve,
The Ferret function writev5d was originally implemented as an external function in 2001 with minor upgrades in 2003, and was statically linked in the Ferret executable starting with Version 6.0 of Ferret. It incorporates Vis5D version 4.3
It takes a variable that's defined in Ferret, and makes calls to the Vis5D routines to write it to a file in the Vis5D format.

If you (or any of our users who might be interested) would like to take a look at the function and the code, it is contained in just a few files; the Fortran external function and the Vis5D library which is in c. If there are changes to be made, a new external function could be compiled.

Ansley
