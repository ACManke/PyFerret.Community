Hi Nadia,
To take and average, you can define a new grid and average to it in one step. You do not need to write any loops.  Please run the demonstration script

yes? go regridding_demo

If you read the comments that are displayed as  the script runs, you will see how averaging to a new grid is done, and how to control the region over which the average is computed and the coordinates of the new grid.

To write a binary file, please look up "binary data, output" in the Users Guide at
http://www.ferret.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm

The command "LIST/FORMAT=STREAM" will output data to a C-style unstructured binary floating point file, or "LIST/FORMAT=UNFORMATTED" will write a fortran-style unformatted binary file.
