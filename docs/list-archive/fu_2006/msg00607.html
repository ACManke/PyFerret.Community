Hi Nadia,
          The /BAD qualifier will not go with the FILE command.
But with the "SET VAR" command. So proceed as follows :

  yes? ..............
  yes? file/grid=xygrid/var="temp" july_2000.dat
  yes? set var/title="Meaningfull title"/units="^oC"/bad=-999 temp
  yes? save/file=july-2000.nc/APPEND temp

 Always try to give a meaningfull title and units before
writing a variable to NetCDF file. 

Hope this helps,

Jaison
