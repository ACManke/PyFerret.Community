Hi Ferret users,

I find strange dots between dashed-line segments when I use
a "plot/dash=(down1,up1,down2,up2) . . ." command.
They seem to appear only when thickness=1.  They also appear
on the legends.  Has anybody seen this?  I'm attaching a sample
figure and the script which produces it.

I'm using Ferret 5.81 on Debian GNU/Linux (testing).

Regards,
Ryo

====================================================
!--- script to reproduce the attached plot
plot/i=1:10/color=black/dash=(0.2,0.3,0.2,0.3)/thickness=1 i * 0 + 1
plot/i=1:10/color=black/dash=(0.2,0.3,0.2,0.3)/thickness=1/ov i * 0 + 1.05
