On Fri, 3 Nov 2006, Ansley Manke wrote:

Hi Glenn,
Ferret gets its netCDF libraries through the OPeNDAP libraries, which at this time don't support all of the capabilities of the Unidata netCDF Version 3.6.0 (which is able to support the file format with 64-bit offsets for very large files).

The OPeNDAP folks are working on this now, and I hope it won't be too much longer before we'll be able to offer this in Ferret as well.

Ansley
