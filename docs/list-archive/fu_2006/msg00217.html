Hi Jean,

You can try the zcat function, e.g.

use temp1.nc
use temp2.nc
use temp3.nc
let temp_all=zcat(temp[d=3],zcat(temp[d=1],temp[d=2]))
list /file=temp_all.nc temp_all

It might be that in addition you need to re-define the new z-axis (I haven't tried it)...

Hope this works,
Ingo
