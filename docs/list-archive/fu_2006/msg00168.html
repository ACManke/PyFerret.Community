A real wind field, such as in the COADS data, is three-dimensional. It cannot be represented by a two-dimensional stream function. There is vertical flow. You might try constructing a theoretical, two-dimensional velocity field and see if it satisfies the test you are attempting. If it fails, it should only do so to the precision caused by truncation error due to the lack of significant digits inherent in the computer word size.
