Hi Zhen,
The variance script defines a variable (not a function) called CORREL from the input variables that you define. To see the whole script, type

yes? go/help variance

You'll see that at the end, there is a variable called CORREL defined and another called COVAR. So, once you have run the script, you can just say

yes? list correl
yes? list covar

I'll see what we can do to add more information to the documentation to make this clearer.
