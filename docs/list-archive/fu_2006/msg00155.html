Hi Sang-ki,
This error message indicates that something more is going on besides the limit on coordinate storage. Something unexpected is happening where Ferret keeps track of all the various axes that have been defined by files or commands like DEFINE AXIS. I'll write you outside the Users List to see if we can track down what's happening. We always want to hear about an error that says something like "tm intern err".

Ansley
