Hello Ferreters,
I found the answer by digging through the archives. For an answer to this see:

http://ferret.pmel.noaa.gov/Ferret/Mail_Archives/fu_2002/msg00431.html


Basically, one can use FORTRAN format statements for the X and Y axis by typing:

yes? ppl xfor (FORTRAN FORMAT)
yes? ppl yfor (FORTRAN FORMAT)


This does the job for a time axis, too.
