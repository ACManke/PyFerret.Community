Hi,
    Here is an example for the fortran format :

      set data monthly_navy_winds
      set reg/x=30:120/y=-30:30/l=1
      vector/length=10/nolab/set uwnd, vwnd
         ppl veckey 103,34,,(f4.1, " m/s")
      ppl vector,1,1


 Regards,

 Jaison

On Thu, 2 Mar 2006, William S. Kessler wrote:
VECTOR/SET/NOLAB u,v
PPL VECKEY,xpos,ypos,,(fortran format)
PPL VECTOR

There has been some discussion here of the fortran format, which can include text labels. Try searching the mail archives for veckey.

A possible bug (or maybe it is a feature) is that the XSKIP/YSKIP qualifiers on VECTOR are ignored in the above syntax. Instead, vector skipping is done directly in the final command, for example:

PPL VECTOR,3,2 ! every third vector in x, every other in y
