To the developers

I have been using Ferret v5.53 on Windows 2000 for some time now with no problems.  The latest version available for Windows XP is v5.51 and this seems to have a bug relating to modulo axes.

For example if I type (v5.53 on a Win 2000 PC):

define axis/modulo/x=-90:90:0.1 xmaxis
Show axis xmaxis

 name       axis              # pts   start                end
 XMAXIS    X                 1801mr   -90                  90
   Axis span (to cell edges) = 180.1 (modulo length = axis span)

comes back.

Now if I do the in V5.51 on a Win XP PC:

define axis/modulo/x=-90:90:0.1 xmaxis
Show axis xmaxis

 name       axis              # pts   start               end
 XMAXIS                      1801mr   -90                 90
   Axis span (to cell edges) = 180.1 (modulo length = 0)

Note the difference - modulo length = "0" instead of "axis span".

This causes problems.  I've tried to work around by defining the axis span with limited success.

Are there plans to make a later version of Ferret available for Windows XP in the near future?

Thanks

Mark
