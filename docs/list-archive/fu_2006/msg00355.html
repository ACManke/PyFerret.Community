Hello Ansley,

Problem solved.

Yours second option worked, white line was giving trouble. I checked with black line and it was fine in postscript file.

I gave a search in ferret emails and found an answere by Billy K. dated 3rd May 2005.

The detailed solution is :

If white line is not appearing in the postscript file then redifine the color:


ppl color, 6,100,100,100

plot/color=12/nolab/ov  var


This will redefine color 6 (magenta) to be white and white line will appear in the ouput postscript file.

2) or, test and see if the white line is what is causing the trouble.  I
was able to run a similar script, and the white line shows up fine, but
as a test, you should draw it in another color and see if it shows up in
the postscript.  For example you might try

  plot/color=black/thick/dash/nolab/ov ...

If this is the case, you could hunt for "white line" in the Users List
archives and see if there are any suggestions.
