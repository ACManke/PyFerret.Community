Hi Lev,
One way to do this is to change the default font for the plot. The old way to do this is with the PPL command DFLTFNT, for example

ppl dfltfnt,@tr
shade/i=1:8/j=1:6 i+j

The axis value labels will also take on the default font and, as you know, the characters drawn with pen strokes are not as sharp as other apps.
To reset you can use another DFLTFNT (ppl dfltfnt,@sr).
