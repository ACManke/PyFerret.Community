Hi Zhen,
The script fland.jnl is the one to use to ovarlay land masses on a plot of your variable. So, after plotting your own variable (with qualifiers /X= and /Y= or /HLIM and /VLIM to specify the region), call fland.jnl. The first argument is the resolution of the land dataset to use; the second variable is the name of a color palette to use to change the land masses from grey to some other single color or to show the topography, with a palette such as dark_land_sea

fill/nolab/hlim=120:295/vlim=45S:50/key/line/sigdig=1/levels=(-1.0,1.0,0.1) CC
go fland 20 black

The basemap script is good to use when you want your data to overlay the map. Again the next argument is a color palette, such as black.

go basemap x=120:295 y=45s:50 20 black
fill/nolab/over/key/line/sigdig=1/levels=(-1.0,1.0,0.1) CC

For all these scripts, to see the documentation from the script, type

yes? go/help fland.jnl
yes? go/help basemap.jnl

and you'll see information about the arguments and how to call the script.

Ansley
