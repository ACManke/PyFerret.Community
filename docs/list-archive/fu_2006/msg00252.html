Hi -
The output of the SHOW DATA command shows that CLIM_SST is a variable, not an axis. Since it looks as if this file can be read by Ferret, you should be able to look at its axes with

yes? SHOW GRID CLIM_SST ! to get a list of the axes of this variable

yes? SHOW AXIS axisname ! using an axis name from the above listing

or use the RETURN keyword, inside grave accents, to get the name into the command

yes? SHOW AXIS `CLI_SST,return=xaxis`
