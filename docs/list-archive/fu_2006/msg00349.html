Hi Yogesh,
I can't run your script, because you use your own local data files. I don't see anything wrong with it, but if you can make us an example with datasets that are included with Ferret, we could test it. I have a couple of ideas for you to try:

1) Run ferret with the startup option -batch. You will need to comment out the metafile and Fprint commands in your script, then
> ferret -batch out.ps
yes? go script.jnl

2) or, test and see if the white line is what is causing the trouble. I was able to run a similar script, and the white line shows up fine, but as a test, you should draw it in another color and see if it shows up in the postscript. For example you might try

plot/color=black/thick/dash/nolab/ov ...

If this is the case, you could hunt for "white line" in the Users List archives and see if there are any suggestions.
