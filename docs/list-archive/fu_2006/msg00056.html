About your question, there should be an environment variable called
FER_PALETTE which points to the directory where these files are.  Type
the Unix command

 > Fenv

and you should see a list of all of the environment variables that
Ferret uses, including FER_PALETTE. These variables should have been set
up when Ferret was installed on your system.  The installation guide for
the operating system you have will give you more information; you can
find this by following the appropriate link on the Downloads web page at

http://www.ferret.noaa.gov/Ferret/Downloads/ferret_downloads.html

If you can't solve it, please write back and tell us what operating
system you have, and we'll see what we can do to help.

Ansley
