Hello James,
 
You can use define view command. Using this command you can control your figure and plot sizes, for example
 

DEFINE VIEW/XLIM=.22,.42/YLIM=.70,.90/AXES ONE

SET VIEW ONE

shade a

DEFINE VIEW/XLIM=.52,.73/YLIM=.70,.90/AXES TWO

SET VIEW TWO

plot b

.................

Good luck,
