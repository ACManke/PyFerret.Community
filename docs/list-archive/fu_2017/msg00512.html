Hi Oliver,
you may use mplayer/mencoder. It is part of some linux-distributions, but building by yourself works too.

The following command lines worked for me. May others know better ...

mencoder mf://*.png -mf fps=7 -ovc lavc -of lavf -lavcopts vcodec=flv:vbitrate=3300 -o movie.flv
mencoder mf://*.png -mf w=2800:h=2500:fps=5:type=png -ovc lavc -lavcopts vbitrate=5000 -lavcopts vcodec=msmpeg4v2:mbd=2:trell -oac copy -o video.avi

Best,
Martin
