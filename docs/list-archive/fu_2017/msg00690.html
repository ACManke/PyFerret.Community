Hi Saurabh,

It looks like there some shaving of edges and something odd in the calculation of the graticules.

I got a nice result by extending the X-Region by 0.5 degrees in both directions limiting y to 63S:63N and setting the graticules directly.


SET DATA coads_climatology
SET REGION/X=19.5:380.5/Y=-63:63/l=1  !!!!!!!!!!!!!!!!!!!!!!!!!!!!!
GO mp_mcbryde_fpp
SET GRID sst; GO mp_aspect
SHADE/NOLAB/NOAX/NOKEY sst, x_page, y_page
GO mp_graticule 20 380 30 -63 63 `126/12` 1   !!!!!!!!!!!!!!!!!!!!!!
GO mp_fland 120 gray
LABEL/NOUSER `($PPL$XLEN)*0.5`,-0.4,0,0,.20 McBryde Flat Polar


Russ
