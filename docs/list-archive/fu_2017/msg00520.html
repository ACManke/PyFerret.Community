Hi ferreters,


I have just wanted to close the matter. I fixed it. I will try to explain it, but I do not know why this happens. So my problem was I had different shade colours depending on the code (1.1 and 1.2). At the end, it was nothing the code worked. But, my mistake was to click to maximize the window (picture 1). After doing that, for some reason the colours of the shade plot (upper left and low left) were shifted.



Picture 1

set viewport ul; ppl shaset reset;\
shade/levels=(-inf)(0.2,0.5,0.01)(inf)  /nolab/set_up my_variable; ppl title "Title";ppl shade ;\                       (1.1)

Picture 2

set viewport ul; ppl shaset reset;\
shade/levels=(-inf)(0.2,0.5,0.01)(inf)  my_variable; \                                                                                                     (1.2)
