           * * * Announcing the official release of PyFerret and Ferret version 7.1   * * *

Please scan the release notes for new features, options and fixes: http://ferret.pmel.noaa.gov/Ferret/documentation/release-notes/version-7-1-release-notes. 

Note that there are updates to the datasets used to draw land, river, and national boundary outlines, so you will want to download and re-install the datasets as described in the Installation Guide.

The source code and releases are available as usual from the Downloads pages on our main web site, and also from from the NOAA-PMEL PyFerret and Ferret GitHub repositories.  See the main Ferret or PyFerret downloads pages for details.

This release does not include Mac OS builds at this time; we will write again when we have updates for our Mac users.

Happy Ferreting!

    Ansley Manke, Karl Smith, Steve Hankin
