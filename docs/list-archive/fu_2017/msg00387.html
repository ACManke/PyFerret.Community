Hi,

It's definitely a good idea to clean up multi-panel plots by removing repeated labels. There are separate PPL commands for plots that have formatted calendar time axes.  TXLABP would be the one to use. However in another string we talked about redefining your time axes to be simple T=1:n:1 axes; if the axis is not a calendar time axis then you should use PPL XLAB " ".

To control the labels for tic marks on the vertical axes, use PPL AXLABP to remove tic-mark labels but not the tic marks. Often that is enough, as it lets you see on each panel where the tic marks line up. To remove the tics entirely you would use the PPL TICS.

Ansley
