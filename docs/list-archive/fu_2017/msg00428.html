Hi
PPL PLOT always comes after a PLOT/SET_UP command. So you would do it this way, 

plot/set var1
go legend ...
go legend ...
go legend ...
ppl plot

plot/nolab/over var2
plot/nolab/over var3
plot/nolab/over var4
