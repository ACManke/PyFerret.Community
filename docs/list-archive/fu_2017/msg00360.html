Dear Estanis,

There could be many ways to fix this problem.
One way to do it is by changing the aspect of viewport, make viewport a rectangle (default is square).
If your shade key is vertical then use "set window/aspect=0.8" (aspect value should be less than 1)
If horizontal then "set window/aspect=1.2" (aspect value should be more than 1)

This should fix the problem.

Vivek
