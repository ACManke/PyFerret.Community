Hi Arjeet,
You can use ppl shakey along with your last view port. In each plot command you have to give 'nokey' option (like fill/nokey or shade/nokey). Then after the 9th plot command you can give ppl shakey option. By using ppl shakey you can adjust the orientation,label size, position of the color bar etc.  Kindly see the link:

http://ferret.pmel.noaa.gov/Ferret/faq/modifying-the-shade-key
