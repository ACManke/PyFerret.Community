Dear Yadidya

You just use your data in this script and change variable according to your data. it will give the shakey at the bottom with large size.

use data.nc
go multi_view 2,2,.41,0.065,.06,0.37,0.13,0.08,v
ppl shaset reset
set view v11
fill/nolab/nokey/set var
ppl fill
ppl shaset reset
set view v12
fill/nolab/nokey var
ppl fill
ppl shaset reset
set view v21
fill/nolab/nokey var
ppl fill
ppl shaset reset
set view v22
fill/nolab/nokey var
ppl fill
ppl shakey 1, 0, 0.17, , 10, 9, -4.5, 5.5, .4, .65
ppl fill

It is possible to explain the mistake easily once I know where exactly you are making mistake.
Anyways, still I'll try to answer the question.

If you want single shakey in view-port of multiple plots,
First, list the details of shakey - 

ppl list shakey

Ferret will give result similar to as of below.

         DO KEY  ORIENT  LAB SIZE  LAB INC  LAB DIG  LAB LEN
              0             1              0.00              0            3              9

         DEFAULT KEY POSITIONING
         X LO     X HI     Y LO     Y HI
         9.40     9.69     1.40     7.40

Then edit the above values as of your choice as explained in the link below.
http://ferret.pmel.noaa.gov/Ferret/faq/modifying-the-shade-key


Vivek
