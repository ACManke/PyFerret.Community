On Wed, Jun 7, 2017 at 6:06 PM, Martin Schmidt <martin.schmidt@xxxxxxxxxxxxxxxxx> wrote:
Hi Gopal,

ppl shakey modifies the parameters of the key. To display the key after modifying it with ppl shakey, you need to issue a "shade" or "fill" command without the "nokey" option.

I am not sure, if the final "ppl shade" does this without issuing a "shade/set " command in advance. I guess, Your final part of the script should look like this:

set view v53; ppl axlabp  -1, 0 ;ppl axlsze,0.12,0.12
fill/nolab/x=80:95/y=15:30/set/t=30-may-2009 olr
ppl xlab Longitude
!ppl SHAKEY  1, 1, 0, 0, 3, 8, 9.4, 10.2, 1.4, 7.4
ppl shakey 1, 1, 0.08, , 3, 6, 5.2, 5.8, 3.3, 5.9
ppl fill
go text_legend_put2 25 25 45 " (30th May)"
go land thick " " black
...

Does this help?

Martin
