Hi,

You may need to be wary that there was a memory leak that was only fixed in V7.1 when opening and closing many files.

If the files comprise a time series it may be possible to either create a descriptor file to join them into on virtual dataset (5000 files max) or  define an
aggregation (V7.0 and later).


http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/converting-to-netcdf/CREATING-A-MULTI-FILE-NETCDF-DATA-SET
