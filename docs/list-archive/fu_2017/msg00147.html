On Wed, Mar 1, 2017 at 2:24 AM, Ansley C. Manke <ansley.b.manke@xxxxxxxx> wrote:
The by-levels palette gives consistent colors for the same data values, for instance as you zoom in on a region; so 23 deg C will always be orange, for instance. The ocean_temp palette is for temperatures in degrees C.  A useful piece of documentation for palettes is here: http://ferret.pmel.noaa.gov/Ferret/faq/ferret-color-palettes



On 2/28/2017 6:50 AM, William Kessler wrote:
That ocean_temp.spk file uses "by-value" levels, not the 0-100 RGB scale of an ordinary palette. Note the first value in each line which is a temperature level. See the documentation for the difference between "RGB_Mapping By_value" and plain RGB.
