Dear sourabh,

You may try as the following to create a mask


let mask = if x[gx=myvar] gt 0 and  x[gx=myvar] lt 60 and y[gy=myvar] gt 33 and y[gy=myvar] lt 65 then 0 else 1

! Now create a new variable 

let new_var=myvar*mask

Hope this will help you,
