Hi,

Welcome.  I would suggest that you run some of the tutorial scripts.

yes? go tutorial

This runs through several of the plot commands, and also demonstrates opening netCDF files, looking at their grids, and some of the commands you can use for gridded variables. 

All of the demos and tutorials are listed here: http://ferret.pmel.noaa.gov/Ferret/documentation/ferret-tutorials.  Please have a look to see what is available.

For working with gridded data, try,

yes? go coads_demo

yes? go levitus_demo

The on-line Ferret manual is extensive, and reading the Introduction chapter will help you get started with the command syntax.

Ansley
