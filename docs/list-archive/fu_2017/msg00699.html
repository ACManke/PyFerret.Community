           * * * Announcing the official release of PyFerret and Ferret version 7.3   * * *

Please see the release notes:
http://ferret.pmel.noaa.gov/Ferret/documentation/release-notes/version-7-3-release-notes

The main improvement this time is that we have put behind us the limitation on storage for axis-coordinate values, so we are now free of the message

  Limit on Storage for Coordinates">**TMAP ERR: limit on storage for coordinates has been reached

Please look at the release notes for some small changes to the downloads and installation procedures, and the rest of the short list of improvements and fixes.

This release is for Linux RedHat 6 and 7, and comparable Linux operating systems. It does not include Mac OS builds at this time, but work is ongoing for Mac updates.  We will write again when we have news for our Mac users.

Happy Ferreting!

    Ansley Manke, Karl Smith
