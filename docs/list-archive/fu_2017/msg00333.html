Hi,

Use DEFINE AXIS, for instance:

   DEFINE AXIS/Z/units=meters/depth zaxis = {1,5,10,15,30,45,60,70,100,150,200}

and regrid to that axis using @ASN, or use RESHAPE to put your data onto that axis.
