Looks like you have a NaN somewhere. Or some other "BAD" value that is not defined as BAD.

Try:

stat/k=1/l=1 TEMP,TLONG,TLAT   ! shows the hi/lo values,mean,rms of each variable

sho grid TEMP   ! shows the range of the axes

sho dat/full    ! more info about the BAD values in the netcdf file
