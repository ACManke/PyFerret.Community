Hello Francisco,
An answer from Joe Sirott of our group:

This is an old bug -- frame/format=gif will seg fault if a user is using
certain types of X visuals (non-indexed). The workarounds are to either use
ferret -gif or to use a PseudoColor X visual.

Ansley Manke
