Hello again,

After looking for an testing solutions, I think I have been able to run 
Ferret properly in my SuSe Linux 7.2 server. SuSe Linux 7.2 does not ship 
with the library libtermcap.so.2. So to avoid error in Ferret launching about 
loading such library, I first created a libtermcap link pointing to 
libncurses by means of the command 'ln -s /lib/libncurses.so.5.2 
/lib/libtermcap.so.2' and after this Ferret 5.40 runs OK.
