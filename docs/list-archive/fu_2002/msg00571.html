Hi Gary,

I would use awk to solve the problem:

list/file=f2var/format=(F8.2 F8.2)/nohead hi,lo
sp cat f2var | awk '{printf("%f\n%f\n",$1,$2)}' > f1var
file/var=both f1var
list both
