Hi,

(1)
I have had success with ncgen using:
ncgen -o my_new_file.nc my_cdl_file.cdl
without the -b or -n options

(2)
Use Data file_name is not a valid Ferret command.  ferret tries to use the NetCDF file
called Data.
USE is an alias for SET DATA/FORMAT=cdf
