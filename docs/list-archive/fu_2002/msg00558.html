Use the newer PLOT command qualifiers: THICKNESS=n/COLOR=name.

PLOT/THICKNESS=[1,2,3]/COLOR=[red,green,blue,lightblue,purple,white] variable

By the way lightblue = cyan, purple = magenta (a truly ugly color).

See the manual under PLOT.
