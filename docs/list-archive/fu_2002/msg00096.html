Announcing Ferret V5.40 on Solaris, Linux, MS Windows, and "Compaq"

  • To: Ferret Users <ferret_users@xxxxxxxxxxxxxxxxxxx>
  • Subject: Announcing Ferret V5.40 on Solaris, Linux, MS Windows, and "Compaq"
  • From: Steve Hankin <hankin@xxxxxxxxxxxxx>
  • Date: Wed, 20 Feb 2002 16:11:20 -0800
  • Sender: owner-ferret_users@xxxxxxxxxxxxxxxxxxx
         * * * Ferret version 5.40 is now available for
download * * *

Ferret V5.40 is available for download on the following
systems:
      o Solaris
      o Linux
      o MS Windows (requires an X server)
      o Compaq
Follow the download instructions at
http://www.ferret.noaa.gov/Ferret/.
The SGI version will follow shortly. Please contact us at
ferret@pmel.noaa.gov if you are interested in ports of
Ferret V5.40 to other platforms.

The new capabilities available in V5.40 are described in the
Release Notes available at
http://ferret.pmel.noaa.gov/Ferret/Documentation/. The
updated V5.40 Ferret Users Guide is available there, too.

Note that Ferret V5.40 can directly access data sets from
the National Virtual Ocean Data System (NVODS). Learn more
about Ferret, DODS, and NVODS at
http://ferret.wrc.noaa.gov/Ferret/DODS/ferret_dods.html

        Happy Ferreting -- the Ferret developers
