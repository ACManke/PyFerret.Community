Hi Anton,

My tool mkdes can be used to generate both a Linux (f90) style or a Unix
(f77) 
style of MC descriptor file.

To generate a Linux style descriptor file, do:
mkdes --f90 netCdfFileList

Without the option --f90, mkdes generates a f77 style of descriptor by
default.

It is all in the documentation which can be read by
mkdes --help

mkdes is used routinely in my group. It reduces the tedious work of
manually 
making the descriptor file.

Hope it helps.

Benyang
