Hi Takaya,
You need the FILE/FORMAT=DELIMITED command.  This
reads fields from a file exactly as you need.  It can automatically
detect the different field types, or you can specify whether
each variable is text, or dates dates, and so on.  This feature is in
Ferret version 5.4 and later.  See "files, ASCII delimited" in the
index of the Ferret Users Guide:
http://ferret.pmel.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm
