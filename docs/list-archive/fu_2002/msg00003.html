Hi David,
Thanks for the report.  You might just update to v5.33; the patterns
and filled contour plots work fine under this newer version running
under windows.

To get all thin lines on a contour plot, you can use the LEVELS
command.  See levels(contour) in the users guide index, and look for
the examples showing how to set the line type.  For example

   USE coads_climatology
   SET REGION/L=1
   CONTOUR/LEVELS=(-5,35,5)LINE(-5,35,5)  sst
