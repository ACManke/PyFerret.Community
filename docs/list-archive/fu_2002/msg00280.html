Hi Vincent
You can use a PPLUS command to customize the axes, and
give a Fortran format specification. Use PLOT/SET_UP 
and then give the PPL command, and finally PPL PLOT to
actually draw the plot:

plot/i=1:100/set_up  500*sin(i) + 1000.
ppl yfor (e8.2)
ppl plot
