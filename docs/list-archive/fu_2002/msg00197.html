Hi,

Can someone please explain why I get "invented" constant data when I use the @AVE transform on data that does not have a continuous time axis?  Here's an example:

USE "http://dods.shore.mbari.org/cgi-bin/nph-nc/data/OASISdata/netcdf/hourlyM2.nc";
SET REGION/T=1-JAN-1998:30-APR-1998
PLOT AIR_TEMPERATURE_HR					! Plots fine
DEFINE AXIS/T="06-MAY-1997 20:00":"12-APR-2001 23:00":1/unit=day TIME_DAY
PLOT AIR_TEMPERATURE_HR[GT=TIME_DAY@AVE] 		! Problem

The last plot shows a constant value of 14.31 from 3-FEB-1998 to 16-APR-1998.  The original AIR_TEMPERATURE_HR variable has data until 3-FEB-1998-02:00 and then begins again on 29-JUN-1998 00:00.  Does this discontinuity cause the problem?
