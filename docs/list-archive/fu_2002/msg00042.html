Hi Jeremy,
	This is a similar solution to that of Andrew. Suppose you have
a favorite position for this label - say just above the upper left of
your plot. Then you could for example

!------------------------------------------------------------------
plot/set/i=1:10 1/i
sp echo 'lab/nou '($ppl$xlen) `1.05*($ppl$ylen)`' 1 0 0.1 '$cwd >\!
                                                            cwd.jnl
ppl plot ; go cwd

! as long as axis lengths remain same size you can reuse "go mycwd"
go ptest ; go mycwd
! but if you alter them then "plot/set ... ; sp echo ..." must be redone
!------------------------------------------------------------------

I don't have access to a linux machine to see if this method would
work there.
