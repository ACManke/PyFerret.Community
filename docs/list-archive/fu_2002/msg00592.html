Hi All,
It sounds like the question has probably been answered, but I'd like
to point to the documentation on this. The formatting of these files is
indeed finicky.  There's an FAQ on the use of descriptor files under
linux, which includes the script "linuxize_descriptors"
http://ferret.pmel.noaa.gov/Ferret/FAQ/system/linux_mc_descriptors.html

And, if you check the Users Guide index under "descriptor files, creating"
and "descriptor files, example"  there are some general notes about the
formatting of the records.  The Users Guide is at
http://ferret.pmel.noaa.gov/Ferret/FAQ/system/linux_mc_descriptors.html

Ansley Manke
