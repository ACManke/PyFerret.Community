Hi Jeremy,
This looks like a bug.  Thanks for the report - we'll look into
it.  One workaround is to overlay a shade plot using a level
that will never appear in your data.   If the range of rt doesn't
include 1000, then:
go graticule
fill rt
shade/over/nolabels/nokey/levels=(1000) rt
Ansley Manke
