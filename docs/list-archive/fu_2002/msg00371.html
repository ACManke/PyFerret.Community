Bingo! That worked:

[...]
set view mid
go margins 0.1 0.8 0.6 0.9
plot/set/title=""/nolabel/vl=-0.4:4.0:1.0/line=8 ignore0(null),aave[l=1:$1]
ppl axset 0 1 1 1
ppl yfor "(i1)"
ppl tics 0 0.15 0.1 0.15 0 0
ppl axlsze 0.2 0.18
ppl txlsze 0.12 ; ppl txlint 0 25
go remove_logo ; go unlabel 4 ; go unlabel 5; go unlabel 6
label/nouser 9.9 1.4 1 0 0.2 @SRGlobal Average Temperature
label/nouser 9.9 1.0 1 0 0.2 @SR(relative to 1961-1990 mean)
ppl plot
ppl %yaxis,-0.4,4,1.0,`($PPL$XMAX1)`,,,,1,,(i1)
[...]
