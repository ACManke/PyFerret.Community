On Mon, 25 Mar 2002, gaoxw wrote:
> Does anyone know how to label special maths symbols like "tau"(wind
> stress) on the plot?

Hi Rongzhen,

Check out the PPLUS symbol fonts
http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/current/Appendix_B_PPLUS_Guide.htm#_VPINDEXENTRY_1644

Here's a quick example:
   plot/vs/hl=-1:1/vl=-1:1 0,0 
   label .5 -.5 0 0 .5 "here @SGt@SR_x=1"
