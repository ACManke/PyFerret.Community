Your problem is likely that the file name is too long as specified in the .des 
file. I don't recall the limit, but I think that it is something like 60 
characters. The use and set data statements do not have that constraint. I 
don't remember for sure, but I think that this limit was raised in 5.33.

One fix would be to abbreviate part of the path, or else put the files in your 
FER_DATA path and only give the file names.
