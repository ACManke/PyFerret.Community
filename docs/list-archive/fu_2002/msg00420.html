Hi Patrick,
Here's an alternate way to draw a line on a plot.  Use the "constant arrays" syntax to define the start and end points.
yes? use monthly_navy_winds
yes? plot/vlim=-1:1/hlim=10000:120000 UWND[i=@ave,j=@ave]
yes? plot/over/vs/line {10000,120000}, {-1,1}
