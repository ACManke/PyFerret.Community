Hi!
	I am using Ferret 5.4 on RedHat 7.2 (KDE Environment) running on a
COMPAQ ML350 server. Seems to be working fine with me. I am yet to face
any problem with it. The basic commands all work fine. I have also tried
out some of the new features like animating on-the-fly, which worked
alright too.

	Regarding Bruce's problem with the -gif option, it was present in
earlier ferret versions also, isn't it? I am not sure if it has anything
to do with RH 7.2. I have faced the same problem with previous versions of
Red Hat also. Explicitly giving the -gif was always necessary to get me a
proper gif file.
