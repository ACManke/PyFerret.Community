I have sometimes encountered trouble creating sequences of gifs (for movies) on OSF
(Dec/Compaq/HP) machines with standard ferret. However, everything works fine if I use
"ferret -gif" instead. The only disadvantage is you won't see any output to an X
window, but you will create clean gifs.

-Al Hermann
