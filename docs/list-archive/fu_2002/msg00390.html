Brent A McDaniel wrote:


Hi Martinho,

What Glenn said is correct, they omitted the shared object files for
whatever reason.  If you install openmotif version 2.1.30-8, it will fix
the problem.

You can find the openmotif rpm's at rpmfind.net (a very handy tool).

Try
http://www.rpmfind.net/linux/RPM/rawhide/1.0/i386/RedHat/RPMS/openmotif21-2.1.30-1.i386.html
(which seems to be working properly).

Cheers,

Brent
