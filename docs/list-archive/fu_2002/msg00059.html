FYI:
I have run into this issue with other applications (not LAS yet!) and
was able to get around it by increasing the Timeout value (in seconds)
in the httpd.conf file.  

#
# Timeout: The number of seconds before receives and sends time out.
#
Timeout 1000

This fixed the "broken pipe" errors I had encountered using an Apache
web server on both Solaris and SGI systems.

Hope this helps,
Cyndy
