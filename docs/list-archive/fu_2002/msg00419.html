Hi Mulyono,
Just to close the loop  - With version 5.40 and before, the symbols yaxis_min and yaxis_max are only set if you used the PLOT/VLIM=   qualifier.  Starting with the next Ferret release, they will always be available after making a plot. With Version 5.40 and
before, one way to use my script is to use the /VLIM qualifier on the PLOT command:

plot/x=180/L=4:50/VLIM=-20:20 monthly[y=0], monthly[y=10], monthly[y=20], monthly[y=30]

And then the symbols ($yaxis_min) and ($yaxis_max) will be available.
