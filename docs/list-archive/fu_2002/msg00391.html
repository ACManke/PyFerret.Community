Hi John -

The main problem here is that the shared system libraries between RedHat 7.1 and 7.3 are not
compatible, and Ferret was built on RH 7.1.   Shared libraries are needed in order to allow Ferret
to access external function capabilities using dynamic linking loader calls (i.e. dlopen).  In the
past, this has only been a problem with backwards compatibility, not forwards compatibility.

We are working on a couple of solutions to this problem and will send a note around when one is
found.

Kevin
