Ferret plotters

To change the thickness of ALL lines on a postscipt plot edit the .ps
file and scroll down to the line which starts with /lw.

It should read

/lw {3000 div setlinewidth} def

If you change the 3000 to a smaller number, say 1000, then all lines on
the plot will be thicker, including borders, labels etc.

This is very useful when the plot appears to be faint and thin.
