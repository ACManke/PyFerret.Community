Hello Xin,
Yes, there is a way to do this.  Look up SAMPLEXY in the Ferret
Users' Guide.  It's in chapter 3, sec2.3.34:
http://ferret.pmel.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm

There is an example in that section which samples along a linear
track, and plots the vertical section.  The technique can be used just
as easily with any track coordinates.

Ansley Manke
