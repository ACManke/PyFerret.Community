Namba, I, too, do not like all those extra files.  In Ferret, one can place a
file in one's top level directory called .ferret.  That file is executed each
time Ferret runs.  Within it, I put the following commands:

spawn Fpurge ferret.jnl
spawn Fpurge metafile.plt

These eliminate multiple versions of those files.
