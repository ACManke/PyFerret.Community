Hi Namba,

Please see the "COLUMNS" command (an alias for "SET DATA/FORMAT=delimited")
in the Ferret documentation.

For the file you have shown, you'll probably want to use a command like

  yes? columns/delimiter="
"/skip=1/variable="dumy,dumy1,lon,lat,dumy2,depth,tem" test.txt

    - steve

P.S. offline from ferret_users (to minimize needless traffic) please let us
know what operating system you were running on.  On Solaris the commands
that you show give the error message  "**TMAP ERR: incomprehensible list
input"
