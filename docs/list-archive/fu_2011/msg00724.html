Hi -
The error message is coming from the netCDF/Opendap library which Ferret uses to access OPeNDAP URL's. Ferret simply passes the entire URL string to the library, which tries to open the dataset, and if it encounters an error, passes the error back to Ferret. It's saying that the username/password credentials aren't recognized. The question needs to be addressed to the data provider at Aviso.
