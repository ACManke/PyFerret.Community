Indeed using LABNUM*, a solution to get symbols LABZ and LABT
defined what ever the shape of the variable (XYZ, XYT, XYZT) will be

shade/set_up var
def sym LABZ=($LAB($LABNUM_Z)%_%)
def sym LABT=($LAB($LABNUM_T)%_%)

Doing this, I always get symbols LABZ and LABT defined.
Note that as the definiton of an empty symbol is not allowed,
I set by default LABZ and LABT symbols to character _
This character is not visible when you use label afterward (subscript)
