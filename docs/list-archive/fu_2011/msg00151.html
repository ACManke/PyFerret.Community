Hi Szymon,
The scat2gridgauss_xy function has arguments which determine how far the algorithm looks in the data to place it into the output grid. It would not normally be used for a regridding operation but there's nothing wrong with trying it of course. 

On 3/8/2011 1:13 PM, Szymon Roziewski wrote:

Dear Ferreters,

got neat result except to the boundary where I got a lack of data. Because
