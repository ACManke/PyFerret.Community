> plot /vs /line mx/my

Sorry - that should have been

plot /vs /line mx,my

If you want to plot bars over a line or a line plot over bars, use the
/overlay qualifier for the second plot.  For example

polygon /key ysequence(mx)+barx, ysequence(my)*bary, ysequence(my)
plot /overlay /vs /line mx,my
