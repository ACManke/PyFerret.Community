Hi -
What is the error that you're seeing?  You may be able to see what's happening by using Ferret to list and give information about the variables you've defined.   Commands like

show grid zt
show grid t_nd

and various plots showing slices of these variables, may tell you if there's an error in the definitions.

(Note by the way, that the argument to SET MEMORY is given in mWords, so the value you've given for that is probably larger than the memory on your machine. )

-Ansley

cancel data/all
cancel viewport
!SET MODE META
