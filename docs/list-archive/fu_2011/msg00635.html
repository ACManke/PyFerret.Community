* * * Announcing the official release of Ferret version 6.7.1 * * *

This Ferret release includes executables for Linux 32-bit and 64-bit linked with NetCDF-4.1.2 libraries.  It includes linux executables compatible with Linux rhel5 and rhel6 for both 32-bit and 64-bit systems.  An improved compile-and-build procedure is available in the source-code download for those who run on other operating systems and who want to build Ferret themselves.

The release notes may be found on the Ferret Documentation pages.

To install Ferret, follow the instructions on the Downloads pages of the Ferret web site
http://ferret.pmel.noaa.gov/Ferret/downloads

Ferret v6.7.1 is required for the upcoming release of LAS v7.3.  Be sure to install the ferret environment as well as the executables, as there is a new color-palette file used by LAS.
