Use the /LEV qualifier to SHADE or FILL.

SHADE/LEV=(many possible options) your_variable
Also note that once you have specified this, subsequent SHADE or FILL commands will reuse the same levels by giving the /LEV qualifier without options: 

SHADE/LEV your_variable
