Aleksander,

You need to use the ZW keyword in your embedded expression if you want leading zeros.

So instead of

set dat "iceh.`day`.nc"

use

set dat "iceh.`day,zw=3`.nc"

See

http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/variables-xpressions/EMBEDDED-XPRESSIONS#_VPID_184

For more information.

Cheers,
Russ
