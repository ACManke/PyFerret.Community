Hi
When you run the STAT command, a set of symbols are generated which contain the results:

yes? stat sst
yes? show symbol stat*

(these symbols were introduced in Ferret v6.08)

or you can always compute mean, max, min and so forth individually, using transformations, @AVE, @VAR, @MIN, and so forth. Run the command SHOW TRANS for a handy list of all the transformations.
