    * * * Announcing the official release of Ferret version 6.7.2 * * *

This is a bug-fix release, aimed particularly to those who run LAS and use the RHEL6-compatible Ferret executables. When LAS runs Ferret, it starts Ferret in -gif mode, and in some instances when trying to make plots we are seeing an error:

   ferret: gif.c:1005: GIFcellArray: Assertion `num_pt > 0' failed.

This is fixed, and the Ferret Downloads pages have the tar files for all currently-supported linux operating systems as well as the source code may be found under the Ferret Downloads page, http://ferret.pmel.noaa.gov/Ferret/downloads

 Release notes will be available shortly.

Note that the release tar files have been rearranged.  The Finstall script that is included in the fer_environment.tar file is set up to use the new arrangement, but if you used an old version of Finstall, it would not work.

Ferret v6.7.1 is required for the upcoming release of LAS v7.3.  Be sure to install the ferret environment as well as the executables, as there is a new color-palette file used by LAS.


     Ansley Manke, Steve Hankin, Karl Smith
