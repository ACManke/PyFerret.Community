Hi,
So it looks as if this data is defined in coordinates of meters? Ferret does not have built-in tools to handle this kind of coordinate system, but simply for the problem of inverting an x axis, you could read the data into Ferret, as you have done, and then save it to a netCDF file.  Then,

   use/order=-xy myfile.nc

and the order of the x coordinates will be reversed.

Ansley

Hi Jimmy,
Is the data in a netCDF file?  If so then you can open the file with the /ORDER qualifier and request that an axis be reversed.

yes? use/ORDER=-xyzt  myfile.nc

listing the correct set of axes are for your dataset.

I don't think I understand what you are seeing with the tic marks. Can you attach an image, and also show the commands that you tried?

Ansley
