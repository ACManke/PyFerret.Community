Hi -
I hadn't seen this before, so we'll have to look into it, but here's a workaround.  Define a custom color very close to white, and use that.
yes? set mode meta
yes? use coads_climatology
yes? fill/L=1 sst
yes? ppl color,5,99,99,99
yes? vector/over/pen=5/L=1  uwnd,vwnd
And the resulting plot can be viewed with no problems.

Ansley
