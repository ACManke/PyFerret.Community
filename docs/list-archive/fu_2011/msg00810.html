SET MODE LINECOLORS:n

Then you can create more colors:

ppl color,1,r,g,b
ppl color,2,r,g,b
...
(probably don't want to change colors 0 and 1 for the plot borders and labels) 

See the documentation: must set the mode, then open a new window.
