Sam, you should be able to use the missing facilities of ferret to solve your problem, but if you have to resort to Plan B, DO NOT write the data out in ascii.  There is an easier way.  Instead, start with the netcdf file outside of ferret, run ncdump on it to make an easily netcdf compatible ascii file, edit that with your editor to replace missing values with NaN, and then use ncgen to regenerate it as a netcdf file.  You could even write a script to do that.  The new file will be a nice netcdf file, but with missing replaced by NaN or whatever you want.  It's a lot less work.
