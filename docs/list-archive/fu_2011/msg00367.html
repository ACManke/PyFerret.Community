Hi -
What version of Ferret are you running?  I think you said earlier that you were using an older Ferret version under windows.  The commands you show work fine running the latest Ferret release for linux, which uses the latest netCDF libraries for reading opendap data.

One option for running up-to-date Ferret versions on windows machine is briefly described on our downloads pages,

PLEASE NOTE! An alternative for running Ferret under Windows: We have had good successes running Ferret under the (free) Virtual Box virtualization environment. Install Virtual Box and then intstall one of the available free and well supported Unix platforms such as Ubuntu. This gives you an environment where you can install and run a Ferret executable for Linux. It also means you can run the latest Ferret releases rather than the old versions available for Windows.

Is this an option for you??

Ansley
