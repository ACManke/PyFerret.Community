Hi all,

I have encountered a problem with dynamic regridding
when I use gxy=var with different datasets

I have reduced the problem to few lines.
The result is that I get a wrong regridding and
the problem diasappears when I plot regridded vars in a different order.

All materials can be found from
http://dods.ipsl.jussieu.fr/brocksce/ferret_bugs/err667_gxy/

Download netcdf files, then play
yes? go err667_gxy.jnl

####################
use "err667_gxy_1.nc"
use "err667_gxy_2.nc"
use "err667_gxy_3.nc"

set v upper
shade ASST[D=3,GXY=TSOL[D=1]] ; go land

set v lower
shade ASST[D=3,GXY=TSOL[D=2]] ; go land

####################

Tested with ferret 6.67, 6.7 on linux 32 and 64
