Dessy,

| I would like to display name of Island and the sea

I think you have to manually specify the locations
to print the names at.  To learn how to do that,
I recommend starting from the Ferret User's Guide:

http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/

What you need is the LABEL command, with which you can print
letters (characters) anywhere in your plot.
