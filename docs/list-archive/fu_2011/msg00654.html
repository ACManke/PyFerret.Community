Hi Richard,

Use the "lev qualifier" to specify the levels explicitly. E.g.

	shade/lev=(0,100,10) my_var						! plot my_var with 10 levels spaced 0-10, 10-20 etc

	shade/lev=(0,10,1)(10,100,10)						! plot my_var with levels spaced 0-1,1-2 until 10, then 10-20, 20-30 until 100 thereafter

Much, much more in the user guide. See section 6.8:
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/customizing-plots/CONTOURING/

Also, information on good palette choice can be found here: 
http://ferret.pmel.noaa.gov/FERRET_17sep07/FAQ/graphics/color_friendly_palettes.html

...Not to have a go, but most simple questions like this can be addressed by going to user guide or the email list archive: http://ferret.pmel.noaa.gov/static/Ferret_search/ferret_mail_archives.html

Cheers,

Paul
