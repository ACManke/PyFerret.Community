Hi Jimmy,
If there is a date associated with the data, it will be displayed on the figure. If the grid of the data does not have a time axis, you could use the idea called "timestamp". That is listed in the Ferret Users Guide index and points you to the documentation.
If you are new to Ferret, a good place for you to start as you learn Ferret is to run the tutorial script 

yes? go tutorial
It starts with basic plots, but part way through shows examples where datasets are opened up and regions are set up. Read the comments on the screen as you go. Other Tutorials and demo scripts are linked to the main Ferret Documentation web page, 
http://ferret.pmel.noaa.gov/Ferret/documentation
http://ferret.pmel.noaa.gov/Ferret/documentation/ferret-tutorials
