I see that my mail tool messed up the link to the chapter that I included in my previous message. Here's the correct link:

You can read ascii data into Ferret and plot it without converting to netCDF, or read into Ferret and then write the data to NetCDF which makes it easier to use from then on. There's a chapter in the Ferret Users Guide, "Converting to NetCDF"    http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/converting-to-netcdf 

Also look up  "ASCII Data" in the Users Guide index.

> You can give the following commands, so that you may plot four graph in a page:
>
>
>
> yes? set viewport ul
> yes? (now give your command to plot & the plot will create at upper left position)
>
> yes? set viewport ur
> yes? (now give your command to plot & the plot will create at upper right position)
>
> yes? set viewport ll
> yes? (now give your command to plot & the plot will create at lower left position)
>
>
> yes? set viewport lr
> yes? (now give your command to plot & the plot will create at lower right left position)
>
