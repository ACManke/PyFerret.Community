On 2/8/2011 3:29 AM, Franziska Schwarzkopf wrote:

Dear Ferreters,

I came across a confusing issue with the "shakey" command:
When creating a vertical key with infinity at the ends and displaying
every n-th level-label (klab_Inc=n) the first level labeled is the
smallest non-infinity number (see attached: key_vertical.gif). Changing
to horizontal key starts labelling at the (n-1)-th non-infinity number
(see attached: key_horizontal.gif).
Is there a way to tell ferret to do it in the same way for both,
vertical and horizontal keys?

Thanks and Regards,
Franziska




use coads_climatology

shade/lev=(-inf)(-2,32,2)(inf) sst[l=1]
ppl shakey 1,1,,2,,,9.5,9.8,1.4,7.4
ppl shade

creates key_vertical.gif

shade/lev=(-inf)(-2,32,2)(inf) sst[l=1]
ppl shakey 1,0,,2,,,1.2,9.2,7.7,7.9
ppl shade

creates key_horizontal.gif
