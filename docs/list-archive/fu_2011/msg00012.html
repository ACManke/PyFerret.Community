Hi,
The interaction of PPL pen 0,7 and graticule lines is a bug, so thank you for the report.  A workaround would be to use the trick where we overlay two viewports:

! Draw the graticules in one viewport. We could issue commands to remove the
! labels on the plot axes as well. 

yes? set view full
yes? use coads_climatology
yes? plot/nolab/vgrat=(line,color=blue) sst[x=60, y=15]

! Define a second identical viewport  and draw the plot with all heavy lines.
yes? define view/x=0:1/y=0:1 full2
yes? set view full2
yes? cancel mode logo
yes? ppl pen 0,7
yes? plot/thick sst[x=60, y=15]
