Hi -
To change the location of the key, you can use the PPL VECKEY command. Here's a note about that: 

http://www.pmel.noaa.gov/maillists/tmap/ferret_users/fu_2003/msg00833.html
The last argument for veckey is the formatting of the numeric label - for positioning it you'll just need the first two arguments.
Here's from a script I use - it puts the vector key up at the top of the plot 

   yes? vector/set uwnd, vwnd
   yes? ppl veckey/unouser 2, `($ppl$ylen)+0.2`
   yes? ppl vector

-Ansely
