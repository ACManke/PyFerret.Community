Patrick,

I believe that ferret is not yet ready to handle group structure in hdf5.
 
You can use a 4.2 or higher version of the unidata NetCDF-Java tool, for
example, toolsUI-4.2.jar, which you can download from 
http://www.unidata.ucar.edu/software/netcdf-java/

to see if there is any group structure in your data files.

--- Peng
