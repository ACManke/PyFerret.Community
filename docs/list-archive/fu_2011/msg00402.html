Dear ferret user,

I would like to say my sincere apology in this mailing list, especially to William.
I just realized that I am not careful in set the step of degree.
Now I can start to use my data using suggestion command from William.

yes? define axis/x=114.3389435E:115.1776276E:0.00453/unit=degree longitude
yes? define axis/y=8.0546379S:8.7746868S:0.00453/unit=degree latitude
yes? define grid/x=longitude/y=latitude bathymetry
yes? file/format=(186f12.3)/var=ps/grid=bathymetry/columns=186 bathy_input.btm
yes? set data bathy_input.btm
yes? show data
