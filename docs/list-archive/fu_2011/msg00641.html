Hi,
The LIST command doesn't give you full control over the date string that labels each time step, but you could use the TAX_DATESTRING function to create a string variable with the dates in the format you want.

   list/format=comma/norowhead  tax_datestring(t[gt=va], t[gt=va], "seconds"), VA

This will give a listing that looks like this,

    "10-AUG-2011 00:00:00", -0.12345
    "10-AUG-2011 00:30:00", -0.23456

and then you could edit the output to remove the quotation marks.  Note that you can use SET LIST/PRECISION= to control the precision of the values in the variable.

Ansley
