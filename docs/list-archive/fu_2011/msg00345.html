2011/5/5 Hein Zelle <hein.zelle@xxxxxxxxxxxxx>
Szymon, what exactly is your problem with the contours in the gif file
you sent?  They look normal to me.

The 0 line is always made thick by default, and negative lines will be
dashed by default.   Check the /levels specifier to see how you can
change that: look for the keywords THICK, DASH and LINE.

If you want to produce pretty plots, look into

set mode metafile
<make plot>
cancel mode metafile

and process the plot to postscript using Fprint.  The final quality
will be MUCH higher.  You can convert from postscript to png or gif
using a number of tools, let me know if you need help there.
