> Remove axes with:
> 
> PPL AXSET,top,bottom,left,right
> 
> use 1 to draw an axis, 0 to omit. (This command is persistent; reset
> to default by PPL AXSET,1,1,1,1).
> 
> To draw outside the plot box (with or without axes):
> 
> PPL WINDOW,OFF
> 
> This is also persistent. Reset with PPL WINDOW,ON
>
