Ryo,
Paul saw this error using an older version of Ferret, version 6.05, which linked with the OpENDAP library for NetCDF file access. Ferret Versions 6.6 and higher link with Unidata's NetCDF 4+ libraries.  So if a recent Ferret executable or the ncdump program from NetCDF4.1 show this error, then it would be a good idea to report that to the library developers.

Ansley
