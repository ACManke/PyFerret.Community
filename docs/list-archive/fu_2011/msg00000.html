Hi Juliette,
If you're running the tutorial scripts, you'll need to edit them to put in FRAME commands, as others have suggested.  To find where those scripts are, execute the Fgo command from the command line,

 > Fgo tutorial

That'll point you to where that script is. Make a copy of your own in the directory where you're running Ferret, and edit that.
