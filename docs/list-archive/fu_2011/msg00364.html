Dear Xiufeng Yang,

> I was trying to download online opendap data, and save them to my computer.
> When I was doing this, Ferret gave the following message:
> 
>  **ERROR: insufficient memory: 75815740 words were requested.

As Ansley said, your computer didn't have enough memory available or
you weren't allowed to allocate it by the operating system.

Keep in mind that a megaword in ferret terms is 4 megabytes.  
100 Mword = 400 megabyte.  If your system doesn't have that free and
available, the memory command will fail.  On my desktop with 6gb 
ram I can go up to 696 mwords (it uses about 2700 megabytes).

Our server with 8gb + 16gb swap lets me allocate up to 2000 mwords,
but if you actually start USING it, the machine will start swapping
badly and becomes unresponsive.

Play with the number to find out what still works on your machine.
Close some other programs if you can't get enough memory freed up.
