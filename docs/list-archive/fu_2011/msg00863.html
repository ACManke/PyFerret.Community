Sorry about that. Occasionally I do something when trying to cut and paste into email which instead sends the email! I was going to say, If viewport definitions such as, 

   def vi/axes/x=.73,1/y=0,0.31 v8

which extend to the right hand x=1 limit of the window, are changed to

   def vi/axes/x=.73,0.999/y=0,0.31 v8
Then Jagadish did not see parts of his windows cut off. This could be a precision problem. Extending viewports that are defined with /axes all the way to the edges should be fine, so this is a workaround. 

Ansley
