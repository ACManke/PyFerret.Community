Hi Xueli,


> Thanks for the all suggestions over contour labels.
> 
> I have another question: how can zero line be set thin. I don't like
> the thick lines on the plots.
> It looks ugly...

use etopo20
set region /x=-15:20 /y=30:65
shade /levels=(-10000,10000,500) /pal=dark_land_sea rose
contour /overlay /levels="(-10000,10000,500)LINE(0)" rose
go land
