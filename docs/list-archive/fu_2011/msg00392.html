Maybe you can try with Matlab t_tide package (http://www.eos.ubc.ca/~rich/#T_Tide) first, 
or use IOS foreman tidal package (http://www.pac.dfo-mpo.gc.ca/science/oceans/tidal-marees/index-eng.htm).
Both of them can be used to extract tidal signal of elevation & currents.
Also can try to write your own external function for ferret to de-tide by IOS package (which written in fortran).
