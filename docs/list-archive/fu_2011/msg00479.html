Try this

  define symbol hl = 76.8:77.8:0.5

  PPL AXLINT,2,2
  PPL AXNMTC,1,2
  PPL AXLSZE, 0.14 0.14
  ppl AXSET,0,1,1,0
  ppl tics 0.08 0.15 0.08 0.15
  plot/hlim=($hl)  your_variable
  ppl xfor,(f3.2)
  ppl plot

You can play with all the commands above to get what you need

Cheers
