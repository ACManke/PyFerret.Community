Hi,
For basics on controlling the plot axes, see Chapter 6 in the documentation, and particularly Section 3 of that chapter for axes.

You can set the format of the labels on the plot axes with

PPL XFOR
PPL YFOR

but there isn't any direct way to substitute a whole other set of labels.  You could define a new axis that would give the desired labels and regrid your data to that axis using the @ASN transformation, or you could remove the labels entirely using PPL AXXLAB and then use the LABEL command to put each new label in its correct location.
