Hi Lawal,
A quick guess of the problem is that you don't have the environment variable DISPLAY set correctly (or maybe it isn't set at all and needs to be). If you are working directly on the Linux machine, under csh/tcsh use the command: 
setenv DISPLAY localhost:0.0
or under bash:
EXPORT DISPLAY=localhost:0.0
If you are connecting to the Linux machine (the "remote" box) from another machine (the "local" box), then use the name of the local box instead of "localhost". You might need to also need to configure the local box to allow connections from the remote box. 

Karl Smith
