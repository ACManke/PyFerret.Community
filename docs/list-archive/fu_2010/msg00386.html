Hi -
You can read in the data using a simple FILE command (which is an alias for SET DATA/EZ and which reads ascii data) 

   FILE/VAR="lon, lat, var"  test.txt
Then you can use one of the family of scat2grid functions to interpolate the data onto a grid. For instance, see SCAT2GRID_XY in the Users Guide, where you'll find examples of how to define the grid and call the function.
