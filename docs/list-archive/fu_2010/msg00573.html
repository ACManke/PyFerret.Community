Seontae,

  14-NOV-2002 / 635:  1.528
  15-DEC-2002 / 636:  1.499
  14-JAN-2003 / 637:  0.937
  14-FEB-2003 / 638:  0.538
  16-MAR-2003 / 639:  0.388
The following uses the above time series, linearly (?) interpolates it to your grid and then computes the average for the time span Dec 15 ... Feb 15. So it will use half of the box around Dec 15, half of the box around Feb 14, and all of the box around Jan 14 which approximately amounts to 

(0.5*1.499 + 1.0*0.937 + 0.5*0.538) / (0.5 + 1.0 + 0.5) ~ 0.978

list nino3[t=15-dec-2002:15-feb-2003@ave]
              VARIABLE : NINO3
              FILENAME : nino4_nino3.dat
              TIME     : 15-DEC-2002 00:00 to 15-FEB-2003 00:00 (averaged)
           0.9760

Here, the full boxes around Dec 15 and Feb 14 are used:

(1.0*1.499 + 1.0*0.937 + 1.0*0.538) / (1.0 + 1.0 + 1.0) ~ 0.991

list nino3[l=636:638@ave]
              VARIABLE : NINO3
              FILENAME : nino4_nino3.dat
              TIME     : 30-NOV-2002 04:43 to 01-MAR-2003 12:11 (averaged)
           0.9911
