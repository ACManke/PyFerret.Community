Yes it sounds weird
 
One way to get isolated data values to appear in a smooth FILL plot is to do FILL over SHADE:

shade my_variable             ! every gridbox that has data
fill/over/lev my_variable     ! smooth gradations while omitting single gridpoints
ppl contour/over                  ! contour lines appropriate to FILL

Yes, this is the method I needed. Thank you!
