Hi Cathleen,
Good news! A colleague has built a new version of Ferret, and it can be installed using the same Install script that's used under Unix.
The executable that Markus Pahlow built runs on Leopard (OSX 10.5) and Snow Leopard (OSX 10.6). You can find this from our Downloads page, 
http://ferret.pmel.noaa.gov/static/Downloads/ferret_downloads.html

and in particular the MAC for Intel page,
http://ferret.pmel.noaa.gov/static/Downloads/mac_intel_downloads.html
There may be details in the install procecure that we'll need to tweak; apparently Mac uncompresses the tar files when you download them. I've changed the Install script to handle this. If it offers to un-tar them for you as well, it'll probably be easiest to say NO, and let the install procedure take care of that step.
I'll make a formal announcement as well so that all of our Mac users who have been awaiting this will see it. 

Ansley
