Hi -
The fact that ncdump gives the same message as Ferret indicates that the library is finding a problem with the file. Within Ferret, a NetCDF library call is made to open the file, and that same library is used by ncdump, therefore you see the same error message and the call exits.

Reading the error message it looks like a problem with the data-type of a variable.
