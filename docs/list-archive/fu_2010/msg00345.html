Dear Saji,

First, please check whether you have "default.spk" in your ferret installation directory (for example : cd /usr/local/ferret/ppl). If not, then create one.

The  "default.spk" file is as below:
    0.0   80.0    0.0  100.0
   10.0   30.0   20.0  100.0
   33.0    0.0   60.0   30.0
   66.0  100.0  100.0    0.0
   90.0  100.0    0.0    0.0
  100.0   60.0    0.0    0.0

Then try again.

or

you can try as bellow:

fill/l=1/palete=blue_orange.spk SST

[Note: Go to above mention directory to see the blue_orenge.spk  and many other sample palete file ]
