Hi Marco,
Yes, you can get more than 7 colors for line plots. See
  SET MODE LINECOLORS

But you might more readily do this with polygons.  The script  polymark.jnl was designed for exactly the kind of thing you want to do; coloring dots by the value of a third variable.  For an example, try
yes? go/help polymark
Ansley
