Paul, Billy, and Marco,
Thank you for the suggestions. Looks like it's working. The final commands are: 

use 030410_init_wsoi.nc
use vegtype_5min.nc
show grid vegtype[d=2]
show grid wsoi_lay[d=1]
define grid/X=GJR1/Y=GJR1/Z=GOL1/L=GOL1 veg_grid
let wsoi =  wsoi_lay[d=1,G=veg_grid]
set variable/title="Soil Moisture" wsoi
set memory/size=200
save/l=1/file=wsoi.nc wsoi
save/l=2/file=wsoi.nc/append wsoi
etc...

Best,

Jonathan
