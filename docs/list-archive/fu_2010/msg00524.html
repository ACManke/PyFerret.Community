Dear Users,
                           I am using below set of commands to  get a graph  with multiple view ports,  after top left corner plot with axis labels on left yaxis  and top x axis I want to switch off the left y axis labels on the top right view port but it do not happen as usual. After little bit exploration I found that if I repeat ppl axlabp command ones again it works !! is it a bug with ferret ? or some coding issue with multi_view? which need to be updated with ferret update? . I am using ferret 6.4

with best regards,
Sudheer

go multi_view 2,2,.34,.12,.07,.34,.12,.05,v,.06,.04
use levitus_climatology
set reg/x=40:110/y=0:30
set mode metafile
!!!!!!!!!!!
set vie v11
ppl axlabp 1,-1
fill/nolab temp[k=1]
con/ov/nolab temp[k=1]
go fland
set vie v11_tl
($poll)/pal=white/color=1 xpol,ypol
label 5.1,4.2,0,0,.25 @acTMP=1
!!!!!!!!!!!
set vie v12
ppl axlabp 1,0
fill/nolab temp[k=2]
con/ov/nolab temp[k=2]
ppl title first iteration
go fland
pause
!! second iteration which works??
ppl axlabp 1,0
fill/nolab temp[k=2]
con/ov/nolab temp[k=2]
ppl title second iteration
go fland
set vie v12_tl
($poll)/pal=white/color=1 xpol,ypol
label 5.1,4.2,0,0,.25 @acTMP=2
