* * * Announcing Ferret Version 6.6.2 * *


Announcing the official release of Ferret version 6.6.2. This version will be required for the upcoming release of v7.7.1 of the Live Access server.

Release notes are available at http://ferret.pmel.noaa.gov/Ferret/documentation/v-6-6-2-release-notes

This Ferret release includes executables for Linux 32-bit and 64-bit linked with NetCDF-4 libraries.

To install Ferret, follow the instructions on the Downloads pages of the Ferret web site
http://ferret.pmel.noaa.gov/Ferret/downloads


New with this release is a release of the source code, see the download page, above; and new wording of our License and Agreement.


     Ansley Manke, Steve Hankin, Kevin O'Brien
