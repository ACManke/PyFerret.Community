Dear kankan,

if the grids for the two files are same, then you can try as following:

use temp.nc  ! first file with temp as the variable

set memory/size= 200 (increase or decrease it as required)

list/file=combine.nc/format=cdf  temp

use salt.nc  ! second file with salt as the variable

list/file=combine.nc/format=cdf/append  salt


this will give you a file called "combine.nc", which will include both temp and salt.

Hope this will help.
