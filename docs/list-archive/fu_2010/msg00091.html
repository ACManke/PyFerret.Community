You want to do a regridding operation.  Define the new 6-hour time axis, and then define a new variable, which will be your hourly variable regridded onto the new time axis.  There is a demo script that will help you to learn this concept.  Read along in the command window as the script runs for an explanation of the commands that are running and drawing the figures.
