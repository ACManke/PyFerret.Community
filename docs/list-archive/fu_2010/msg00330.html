Hi Izidine,
The latest release of Ferret included some new axes in the climatological_axes.cdf file, including a monthly climatological axis on the 360-day calendar.

So you can follow the examples for regridding with @MOD, (see climatology, creating in the Users Guide Index) and use this axis.
yes? use climatological_axes
 *** NOTE: regarding ./go/climatological_axes.cdf ...
 *** NOTE: Climatological axes SEASONAL_REG, MONTH_REG, MONTH_IRREG, MONTH_GREGORIAN, MONTH_NOLEAP, MONTH_360_DAY  defined

yes? cancel data climatological_axes
yes? use my_data
yes? PLOT my_var[GT=month_360_day@MOD]
