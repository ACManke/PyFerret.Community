Hi all,

I have detected a problem when you have a regular spaced axis
defined as float then saved with ferret. The axis becomes irregular spaced
because convertion to double brings numerical noice.
Note that this not detected by ferret, which still considers the saved axis as regular but by other software such as paraview 3.6 that I am currently testing (netCDF reader in particular).
