Hi,
Instead of
   yes? if `depmin` then let u1=u[z=`depmin`]

The syntax would be

   yes? LET u1 = if u1 EQ depmin then u1

Or to have Ferret interpolate your data and find where U crosses a value, see the @WEQ transformation.
