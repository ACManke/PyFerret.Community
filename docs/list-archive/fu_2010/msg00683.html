Hi -
Without seeing the plot,it's hard to know what's happening.  Are you using a FILL plot?  Sometimes the algorithm used to do the color contour fill leaves some white space at the edges. A workaround for that is to do a SHADE plot and then a FILL/OVERLAY,

   SHADE var
   FILL/OVER/LEV/OVER/NOLAB var

The second command uses /LEV to repeat the same color levels, and /NOLAB so that there isn't a label telling us there was an overlay.

Ansley
