Wish you all a very very Happy Christmas and Happy New Year.

I am using fill and vector commands to plot u,v wind data. Plot is fine but there is extra line
outside y-axis. How to remove this line (plot attached).

yes? use uwnd.mon.mean.nc
yes? use vwnd.mon.mean.nc
yes? fill/nolab/pal=blue_orange/x=40:100/y=-10:40 UWND[k=17,l=637, d=1];go land
yes? vector/nolab/ov/length=10/x=40:100/y=-10:40 UWND[k=17,l=637, d=1],VWND[k=17,l=637, d=2]

I am using Ferret v6.07 on Linux.
