    ***** Announcing the official release of Ferret version 6.5 *****

Release notes are available at http://ferret.pmel.noaa.gov/Ferret/home/documentation/v6-5-release-notes

This release is be required for the upcoming LAS v7.1 release. Either v6.5 or v6.6 will work with LAS.

This Ferret release includes executables for Linux 32-bit and 64-bit,  and a statically-linked linux executable for other Unix systems.

In addition to the Ferret V6.5 executables, there are also V6.6 executables for Linux 32-bit and 64-bit.  These are linked with the NetCDF-4.1 library which includes the ability to work with NetCDF-4/ HDF-5 files.  The release notes for V6.6 are at http://ferret.pmel.noaa.gov/Ferret/documentation/v6-6-release-notes.

Unidata is currently between official releases of the NetCDF library, but beginning with their release of NetCDF-4.1.1 and for all future releases of Ferret, we will be linking only with the NetCDF libraries. Executables for other operating systems will follow.


To install Ferret, follow the instructions on the Ferret Downloads pages:
http://ferret.pmel.noaa.gov/Ferret/downloads
