>  Karthik.


go/h fland 

gives :



! usage:       arg1        arg2        arg3          arg4      arg5      arg6
!  GO fland [resolution] [palette] [overlay_style] [detail] [x=lo:hi] [y=lo:hi]

The trick is then to use a color palette. For instance :

go fland "" blue

You can run the Unix command "Fpalette '*'" to obtain the list of available colors palettes.
