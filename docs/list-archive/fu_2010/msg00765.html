* * * Announcing the official release of Ferret version 6.6.5  * * *

This version is required for the upcoming release v7.7.2 of the Live Access server.

This release is being made primarily for the LAS release, but also includes a few bug fixes and new items. Release notes are at
http://ferret.pmel.noaa.gov/Ferret/documentation/version-6-6-5-release-notes

This Ferret release includes executables for Linux 32-bit and 64-bit linked with NetCDF-4.1 libraries.

To install Ferret, follow the instructions on the Downloads pages of the Ferret web site
http://ferret.pmel.noaa.gov/Ferret/downloads
