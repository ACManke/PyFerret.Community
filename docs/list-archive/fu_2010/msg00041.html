Hi Sandip,
I used to have this problem reading in NCAR CCSM output until someone pointed me to the version operator in the ncks function of NCO (http://nco.sourceforge.net/nco.html ). The syntax (from the command line) is something like: 

ncks --fl_fmt=classic $WRONG_VERSION.nc $OUTFILE

Hope this helps,

Paul
