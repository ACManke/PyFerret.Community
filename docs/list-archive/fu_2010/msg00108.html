Hi Hanh,
You can append to a file in directions other than the time axis, but you need to define the entire depth axis first. This is described in the documentation - look up "append, slab to netCDF file". The example in that section shows reading from ascii files and writing to a netCDF file, so you only need to look at what is done in the SAVE commands. 

Ansley
