Hi Bill,
The data we download to create the datasets containing outlines is from the "pscoast" tool in the GMT software, as outlined in the Ferret FAQ 
http://ferret.pmel.noaa.gov/FERRET_17sep07/FAQ/graphics/coastlines.html
That source does not include data for outlines of counties, and I'm not aware of any source for that data. The coastline-extractor mentioned in our FAQ page contains political boundaries, but I believe that consists only country and state boundaries as well.As you say, it must be available somewhere. The data consist of (lon,lat) pairs, separated by a missing-value where the pen should pick up, so the Ferret command to plot them is just 

  plot/vs/line lon_data, lat_data.

Ansley
