Hi Prescilla,
                         You can use NCO utility - ncrename for this purpose.
you should have nco installed in your system then below command will rename the variable properly
ncrename -v x,y  tx.nc
where x is old name y is new name and tx.nc is ncfile
if you use ubuntu
sudo apt-get install nco will install nco in your system else you need to get that from sourceforge.
hope this helps.
Sudheer
