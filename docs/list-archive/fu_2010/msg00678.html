Ferret users,

|                       you can define a new grid
| like
| 
| let ngrd=x[gx=sosstsst]+y[gy=sosstsst]+t[gt=sosstsst]
| 
| let sst_n=reshape(sosstsst,ngrd)

Another idea:  The @MAX operator removes that particular dimension,
so we could do

   let temp_n = temp[k=1:1@max] 

to remove the K dimension.  I'm attaching a concrete example.

Regards,
Ryo
----------------------------------------------------------------
yes? set data levitus_climatology
yes? show data
     currently SET data sets:
    1> /usr/local/ferret-6.61/data/levitus_climatology.cdf  (default)
 name     title                             I         J         K         L
 TEMP     TEMPERATURE                      1:360     1:180     1:20      ...
 SALT     SALINITY                         1:360     1:180     1:20      ...
yes? let temp_n = temp[k=1:1@max] 
yes? save/clobber/file=tmp.nc temp_n
yes? cancel data/all
yes? set data tmp.nc
yes? show data
     currently SET data sets:
    1> ./tmp.nc  (default)
 name     title                             I         J         K         L
 TEMP_N   TEMP[K=1:1@MAX]                  1:360     1:180     ...       ...
