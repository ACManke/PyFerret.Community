I sometimes use the following 'tic trick':

def axis/t="01-JAN-1860":"30-DEC-2300":5/edges/units="Days"/cal="360d"
simutaxis1
plot/vlim=0:1/set t[gt=simutaxis1]*-1E+34
ppl txnmtc -1
ppl tics,0.0,0.25
ppl plot
