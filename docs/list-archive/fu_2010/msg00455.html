Hi Jean,
The LIST/FORMAT=STREAM and LIST/FORMAT=UNFORMATTED list floating-point data.  However you can write netCDF files with the data type specified as integer*2.  Do this using SET VARIABLE/OUTTYPE=, documented in section 3.1.8.6 Output Variables to NetCDF files:

http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/variables-xpressions/VARIABLES#_VPID_87

-Ansley
