Very quick answer: if you put your Grads files onto an Opendap server, then you can usually read it into Ferret straightforwardly. Opendap/ DODS has the ability to read several file types that Ferret cannot read directly. For example Grib files (met data and model output) is transparent to Opendap, so you can then read it in Ferret and SAVE it to netcdf.
Setting up an Opendap server is not hard, but I don't have time to explain it here.
