The fucntion which reads Vis5D datasets is written as a Ferret external
function.  It calls some routines from the v5d distribution, and I imagine
it wouldn't be too hard to write a readv5d function.  The writev5d code
can be found in the External Functions web pages.  Find the links under
"More Examples, Externally linked" on
http://www.ferret.noaa.gov/Ferret/External_Functions/ferret_EF.html

Would you like to give it a try?
