Hi Lynn,
          Earlier I tried it only on my PC(Linux: Redhat-9, Ferret5.7).
Just now I tried on few other machines (the same script i posted earlier
; and all are on remote machines but display on my Linux PC).

    IRIX64 6.5 : Ferret 5.70 : working fine
    AIX 5.2    : Ferret 5.41 : OK

 On AIX, sometime it failed with "set window/size=.*"...but sometimes it
worked without any problem.

 I think it is worth giving a try with 

       1. Skipping the "set window/size=.*" line. Use the default
             window size.
       2. Give the "Backingstore" option on your machine (on the machine
             you get Ferret's display). (On Linux we can give this option
             in /etc/X11/XF86Config under  Section "Device" as a new line  
             	Option	    "Backingstore" )
       3. On your machine (on the machine you get Ferret's display) choose
             the window behavior "Focus Under Mouse" instead of "Click to
             Focus" ( on linux : start application ==> Desktop ==> window
             behaviour --> focus --> Policy)               

Hope This Helps

With Regards 

Jaison
