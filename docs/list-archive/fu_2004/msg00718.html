Hi Masoud,
Use FILL/KEY=cont  and SHAKEY together.

SHAKEY is the best way to control the number of digits in the colorbar
labels. The qualifier /key=cont gives you all the functionality of the
ccbar script, but is simpler to use.

What version of Ferret are you running?  If it is version 5.6 or higher, you
have complete control over the the number of digits using the fifth argument
of SHAKEY.   That argument, klab_dig, is the number of significant figures
in the labels if it is positive, or if you set it to  a negative number, then
ABS(klab_dig) is the number of digits after the decimal point.

For example,

yes? use levitus_climatology
yes? let var = salt[k=1]/100
yes? fill/lev=(0, 0.5, 0.061)/key=cont/set  var
yes? ppl shakey,1,1,0.1,1,-3
yes? ppl fill
