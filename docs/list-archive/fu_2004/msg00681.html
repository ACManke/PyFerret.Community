Hello All,
	That is a tricky problem!  However if you want to include
an underscore and a degree symbol in the same label and IT IS NOT
ESSENTIAL to use the ASCII font try
	@srblah_-blah#blah_-blah

The # does the degree symbol in some fonts.  If the underscored
minus is too short you could try
	@srblah_@el01@srblah#blah_@el01@srblah
