Dear Ferret Users,

 I am trying to represent surface velocity field using vector function
but I want to smooth the field. I tried to use the different option of
vector as 
/ASP or XSKIP or YSKIP /LENGTH ... but it is not sufficient to smooth
correctly the velocities.

I currently do it with this script :

!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
set viewport ul
cancel mode ascii
set region/x=0e:80e/y=50s:15s  
ppl dfltfnt @p7@tr
vector/y=50s:15s/k=1/x=0e:80e/set/asp=1./length=100 u_sum, v_sum
ppl xlab  "@p7@tr longitude"
ppl ylab  "@p7@tr latitude"
PPL vector
go land
label 40 -12.5  0 0 .12 "@tr@p7 a) Summer 1998 "
ppl vector/over,8,5
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

 Has somebody already plotted "smoothed" horizontal currents ?
