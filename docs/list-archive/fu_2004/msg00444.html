Hi Michael,
            You can use "SHAKEY" to change the label size of the color 
key.
   usage : SHAKEY do_key, orient, klab_siz, klab_inc, klab_dig, klab_len, 
kx_lo, kx_hi, ky_lo, ky_hi

Earlier I had send a modified version of polymark.jnl(polymark_new.jnl) as 
a reply to your querry titled "trajectory plot + colorbar"
(http://ferret.pmel.noaa.gov/Ferret/Mail_Archives/fu_2003/msg01213.html)
and it does have an option to include shakey arguments. Use that go file 
with required "klab_siz" in the 7th argumnet.

Example:
   LET xpts = 180 + 50*COS(2*3.14*I[I=1:50]/60)
   LET ypts = 15  + 20*SIN(2*3.14*I[I=1:50]/60)
   LET sst = 25 - ypts + RANDN(ypts)
   GO basemap x=120:250 Y=25s:65n 20
   GO polymark_new poly/over/nolab/key/levels=(-15,30,5)  xpts,ypts,sst,circle, 0.5,",,0.15,,,,,,7.4,1.4"

Hope This Helps
