Hi Chris,
            In user manual multiple files/descriptor files are described  
in two sections;

	Ch2  Sec2.1. Multi-file NetCDF data sets
	Ch10 Sec4.   CREATING A MULTI-FILE NETCDF DATA SET

	From these two sections you can get the basic details of
descriptor file used to access multiple netcdf data files. 

   Remember :
		name the descriptor file with .des extension
		open using  "set data **.des"    
 
 Have a look at the attached descriptor file for 
	daily data ( hence D_TIME_UNIT   = 86400.0 , in seconds ) 
	yearly netcdf files ( S_END - S_START = 365 for non-leap year )

 
 hope this helps.....
