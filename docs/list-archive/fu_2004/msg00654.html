Hi Paul,

The aggregation server at http://opendap.aviso.oceanobs.com/dodsC is out of date. Please see the message at http://my.unidata.ucar.edu/cgi-bin/getfile?file=/content/support/help/MailArchives/dods-tech/msg02100.html
Ferret v5.7 was built with post-september,2003 DODS-NetCDF client libraries. I imagine your dncdump was built with pre-september,2004 libraries. The up-to-date version of the aggregation server you are trying to access is at http://opendap.aviso.oceanobs.com/thredds/dodsC/en/h/msla_oer_en_err.
