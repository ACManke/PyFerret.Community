Hi Enils,
To find DODS dataset addresses, go to the Ferret Homepage, and click on the
OPeNDAP (a.k.a. DODS) link.  Scroll down a bit on that page, to the heading
"Finding DODS datasets".  A search  of the master data set list turns up the address
you need; the info and site-info links provide README files about the data.

  TheDODS address for the Smith-Sandwell 2-minute topography/bathymetry data is

http://www.ferret.noaa.gov/cgi-bin/dods/nph-dods/data/PMEL/smith_sandwell_topo_v8_2.nc

so in Ferret you can say,
  yes? use "http://www.ferret.noaa.gov/cgi-bin/dods/nph-dods/data/PMEL/smith_sandwell_topo_v8_2.nc"
