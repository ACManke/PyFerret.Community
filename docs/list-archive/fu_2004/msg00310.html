Hello -

The reason that you received the "command not found" error when you typed the "ferret" command is that there wasn't a link in $FER_DIR/bin from ferret_v560 to ferret. I have fixed this problem in the executable tar files. You can either download the Ferret executable tar file again, or run the following command in your $FER_DIR/bin directory:

ln -s ferret_v560 ./ferret

If you have any questions, let me know.

Sorry for the hassle -

Kevin
