Hi Vincent,
What value did you send to the function for the last argument, frac_timeser?
This should be greater than zero if there are places in the input field where
there is missing data at all times. The code finds all of the locations where there
is at least the fraction of data given by frac_timeser and uses all of those locations
in the eof computation.
Ansley Manke
