Hi Wendy,
The previous messages are correct, pointing you to ways to be
sure you aren't doing a larger calculation than you need.
But, I also want to point out that you may want to upgrade Ferret
to a more recent version. Starting with Ferret version 5.4 we removed
the limit on the plot buffer. Ferret now uses use dynamic memory
allocation and you will not run into this limitation.

Ansley Manke
