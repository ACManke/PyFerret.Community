A quick follow up:  Ferret graphics are based upon PLOT+, which uses the old Hershey fonts.  The font files can be replaced or edited ... perhaps Hershey-like font files already exist for Chinese.  It would take some custom programming to do this -- not trivial, but not a huge task either.

Through the Ferret command line interface plot labels in Chinese would be specified as

    yes? LABEL [... qualifiers...] @CCfthd.hguifui

where "@CC" selected the (new) Chinese character font
and "fthd.hguifui" was whatever the Chinese characters happened to map to in their English equivalents

     - steve

Ansley Manke wrote:

Hi Yoshen,
No, unfortunately Ferret does not have a Chinese font.  It incorporates
a set of fonts used by a version of the PPLUS graphics package, drawing
the fonts as graphics primitives.  Thus it is not possible at this time to add
new fonts.

You could use Ferret to draw your graphics and then edit the postscript
or gif output with another graphics package to add the characters you want.

Ansley Manke
