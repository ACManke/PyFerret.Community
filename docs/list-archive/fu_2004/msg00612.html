Hi Wendy Wang,
                It is very easy ( if you want to get the box defined in
user units ), use go file "box.jnl" distributed with ferret.

example :

   yes? use coads_climatology
   yes? fill/x=100:300/y=-50:50/l=1 sst
   yes? go box 200,220,-5,5,7


Hope This Helps
