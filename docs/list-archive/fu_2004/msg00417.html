Hi Yogesh,

In order to change the size of the tic numbers you can use:

PPL AXLSZE xsize,ysize
(PPL AXLSZE 0.16,0.16), for instance

If you also want to change the size of the titles:

PPL LABSET main-title size, x-label size, y-label size
(PPL LABSET 0.12,0.15,0.15), for instance

Toni
