Hi David,
Did you try defining your viewports with the
/TEXT qualifier?  It is designed to address this
very element of viewport plots

  yes? define viewport/TEXT=1.2/xlim=0:0.5/ylim=0.5:1 v1
  yes? set view v1
  yes? fill my_var

You may need to eliminate or move some automatic
labels, though on your example plot I see that you
have already removed most of the labels already.
