Hi Wendy,
To get the underscore character use the the ASCII Simplex or
ASCII Complex font (AC or AS)

yes? go ptest
yes? label/nouser 3.6,6.5,0,0,0.1, "underscore_here"
yes? label/nouser 3.6,5.5,0,0,0.1, "@ASunderscore_here"
yes? label/nouser 3.6,4.5,0,0,0.1, "@ACunderscore_here"

In the first label the underscore character has the effect of making
a subscript out of the next character.  (^ would make a superscript.)
The labels using @AS or @AC keep the underscore as is.
