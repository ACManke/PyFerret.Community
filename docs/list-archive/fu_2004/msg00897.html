In addition, the next version of Ferret, version 5.8, which will be released
within the next couple of weeks, has a new PLOT qualifer, PLOT/NOKEY
which removes only the key and not all the other labels as /NOLABEL does.
With this option and the legend.jnl script, you can have a plot with all the other
automatic labelling, and the keys positioned as you wish.

Ansley

Hein Zelle wrote:
Benyang Tang wrote:

  
When I plot 2 lines, I get 2 line keys (something like Matlab's
caption). How do I move them, and change the texts?
    

The easiest way is to use the "legend.jnl" script, provided with your
ferret installation if you have ferret 5.6 or higher. Use it like
this:

plot /nolabel /set_up variable_1, variable_2, variable_3
go legend 1 "my variable 1"
go legend 2 "my variable 2"
go legend 3 "my variable 3"
ppl plot

you can also specify the location of the key with ul, ur, ll, lr after
the title. Run the script test_legend.jnl for an example.

Hein Zelle
