Hi Julien,
Please take a look at the recent thread in the archives that starts with this message.
http://www.ferret.noaa.gov/Ferret/Mail_Archives/fu_2003/msg01672.html

The short answer is that for now the number of line types are limited, but
that you can change the colors. I had hoped it would be an easy addition to
the code to add more line "types" and thus let users set more colors to suit
their needs, but my first look at that showed it wasn't as simple a change to
implement as I had hoped. It is on our list of improvements to work on for a
future release.
