Dear Patrick,

Thank you for contributing your software to the Ferret community!  
Hopefully this will encourage the use of MC descriptors, which are a great
way to save time & disk space when working with multiple NetCDF files.

If you're interested, I've written a complementary tool in Perl, which
diagnoses the timestamps directly from the NetCDF files:

   http://www.gfdl.noaa.gov/~atw/ferret/make_des

Basic usage is

   make_des file*.nc > file.des

Run "make_des --help" for a detailed usage message.  This tool was
inspired by Benyang Tang's earlier work:

   http://ferret.pmel.noaa.gov/Ferret/Mail_Archives/fu_2002/msg00595.html

Andrew
