Hi all,
 
I have LAS 6.4, Ferret 5.7, DODS 3.4, and MySQL 4.0 running on an ES 3.0 system (due to go public soon).  They all appear to work fine, and to work well together.  My experience has been that in general, ES 3.0 is compatible with RH 7, RH 9, and Fedora binaries (but not RH 8 binaries).
 
The only difficulties that I ran into was that the RedHat distribution of MySQL 3.23 appeared to be incomplete, so I downloaded and installed version 4.0 from MySQL.com (which then required a newer DBD-mysql package). The perl-netcdf package just wouldn't install, so I downloaded an RPM instead.  Also, the ParseLEX package required a patch, which is now included in LAS 6.4.0.1
 
Finally, I am using IBM Java 2-141, rather than the Sun Java distribution -- I'm not sure if this makes a difference.
 
Cheers, Mike
