Hi Tony,
By default, Ferret plots along a linear scaling of longitude and latitude.  There is also a ferret jnl script for plotting in Plate Caree projection which is also x=longitude, y=latitude.  It is called mp_plate_caree.jnl  See Ch6 Sec10.4.  Map Projections in the Ferret User's Guide for more info.  http://www.ferret.noaa.gov/Ferret/Documentation/
Joe
