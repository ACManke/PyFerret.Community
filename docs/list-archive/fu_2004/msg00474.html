Hi Holger,
Ferret does almost all its computation in single precision. Only the coordinate
values of the grid are saved and used as double precision; data for variables is
read, used, and written as single precision. We're aware that this is a limitation,
but at this time there is not a way to force Ferret to do calculations in double
precision.

Ansley Manke
