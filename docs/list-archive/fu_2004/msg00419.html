Hi Toni,
	You want to represent in 2-D a scattered dataset - i.e. one
that you cannot read in as a grid. You have two options

1) since the contour, shade, or fill commands operate on gridded
   data you could grid the data - see OBJECTIVE ANALYSIS in the Ferret
   manual and the example there.  Once it is gridded you can plot up
   the data with one of the commands above then perhaps use
	plot/over/nolab/sym=3/vs xx,yy
   to show where the data actually are.

2) alternately you could use the "polymark" method where you plot a
   color-coded mark at each x,y location; you will need to choose a
   size and shape for the marks so that they will not blot each other
   out in the data-rich areas. This approach gives a good feel for
   how much data you actually have.

Good luck,
