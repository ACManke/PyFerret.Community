Hi Gary,
Another option with version 5.53 of Ferret is the new -script command-line
option for Ferret:

  > ferret  -script  scriptname.jnl

or if you prefer,

  > ferret -gif -script  scriptname.jnl

Here's some documentation on the -script startup option
http://ferret.pmel.noaa.gov/Ferret/Documentation/Release_Notes/v553.html#script
