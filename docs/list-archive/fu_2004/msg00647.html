hi Kim,
	a go file is provided with Ferret distribution for histogram 
plotting: frequency_histogram.jnl
The minimal usage is very simple.

use your_file
set the region
go frequency_histogram lo hi delta

where lo, hi and delta are lower, upper and increment of the 
independent axis. Also you can provide a weight to multiply with each 
value of the variable. The detailed description and examples are 
provided inside the go file.
