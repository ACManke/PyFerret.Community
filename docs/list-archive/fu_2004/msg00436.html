Hi Wendy,
Ferret's definition of a continuous key is that there are no lines between the
colors. Otherwise it is the same as the default key. Instead of using ccbar,
try the /key=cont command qualifier instead.

fill/l=3/k=2/key=cont/lev=(0,0.5,0.05) cl

Ansley Manke
