Hi Wendy,
These are all commands that need to be used after a  /SET qualifier on
your ploting command.

 contour/set var
 ppl conset 0.3
 ppl labset 0.3, 0.08, 0.3
 ppl contour

Because the contour labels are placed automatically, sometimes
when you change their size they will all disappear!  The last argument
to the ppl conset command will put more labels on.  It may take some
fiddling to get it right.

 contour/set var
 ppl labset 0.3, 0.08, 0.3
 ppl conset .2, , , , , , ,2
 ppl contour

Hi Wendy Wang,
Here are few simple commands. For details refer user
manual.

default numerical labels (before fill/plot command)
-----------------
ppl axlsze .1 .1 numeric axis label sizes: x and y axes
ppl txlsze .1 numeric axis label sizes: time axis

default text labels (before fill/plot command)
-----------
ppl labset 0.3, 0.08, 0.3 ! LABSET,HLAB1,HXLAB,HYLAB,HLABS
HLAB1 = main label default=.16 inches
HXLAB = x - label default=.12 inches
HYLAB = y - label default=.12 inches
HLABS = movable labels default=.12 inches

(also see LABS, RLABS, LLABS)

user defined label (after fill/plot command)
------------------
LABEL 50, 1.2, 0, 0, .2 @P2MY SIN PLOT ! LABEL xpos,ypos,center,angle,size text
! @ to specify color,font style
etc.


Hope this helps
