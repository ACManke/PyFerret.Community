Hi Mark,

A few points:
 

  • For small-area plots you may want to reduce the amount of white space used around the borders of the plot ... in extreme cases reducing this to zero and locating the axis tics, etc, *inside* of the plotting rectangle.  There is a script, "GO borders" that can do this for you.
  • Ferret uses PLOT+ fonts, which are not the nicest.  For publication graphics some folks like to output the graphics unlabelled and add the labeling with another application.
  • I'm not sure of the issues that you refer to with colors "blurring" together ... 300x300 pixels will impose the same limitations on colors regardless of the application.  You can experiment with different palettes to find what works for you.
    - steve
