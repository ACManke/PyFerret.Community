Hi Wendy,

I've used the "go regressx" script to do linear fits with my ferret data.
The go regressx (and regressy, regressz, regresst) scripts are included in
the provided Ferret scripts, and entering the command will provide you
with a description of the script and instructions to enter your
independent and dependent variable.  The script provides a slope,
y-intercept, and r-squared output.
There might be a clever way to plot the linear fit directly, but I just
overlay the linear function only my plot and use the label command to describe its
characteristics.
You can get a quick preview of the script by entering "Fgo -more regressx"
at your shell prompt.

Hope this helps,
Lia
