Hi Vincent,
The functions do have an upper limit on the overall size of the problem they
can solve. See this item from the mail archives:
http://ferret.pmel.noaa.gov/Ferret/Mail_Archives/fu_2003/msg00142.html

The test for this limit was added to the functions starting with Ferret version 5.53.
You might consider regridding your data to a coarser grid before calling the
EOF function.

Ansley Manke
