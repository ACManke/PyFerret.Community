the original message mentioned a wrong machine
type and we finally fix our problem.


So here is the complete flag to be use on 

a Silicon Graphics Origin 2100 

such as Rhodes (Idris/CNRS, France) 

------------------------------
INCLUDES        = -I. -I$(TMAP_LOCAL)/include
FINCLUDES       = -I../../fmt/cmn -I../common -I../ef_utility/ferret_cmn
-Iferret_cmn
                                                                              
                                          
CCSHFLAG        =
CC              = cc
RANLIB          =
CFLAGS          = -c -n32 $(INCLUDES)
FC              = g77
F77SHFLAG       = -shared
F77             = g77
FFLAGS          = -c -fno-second-underscore -fno-backslash -fdollar-ok
-fugly-init -fintrin-case-any -ffixed-line-length-132 -u $(F77SHFLAG)
$(FINCLUDES)
SYSLIBS         = -ldl
                                                                              
LD_DYN_FLAGS    = -G -shared -n32
CPP             = /usr/freeware/bin/cpp
CPP_FLAGS       = -P -traditional $(INCLUDES)
CFLAGS_DEBUG    = -g -Ddebug
CFLAGS_DEBUG    = -g -Ddebug
FFLAGS_DEBUG    = -g -C -Ddebug
------------------------------------
                                   
Note that set linking with ld -lc yields 
a trouble with ldopen() and ldsym().
