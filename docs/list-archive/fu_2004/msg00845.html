Hi Tony,

You haven't included the OPeNDAP URL, so I cannot test ... this is a guess.

It looks to me like in your dataset missing value points are represented as
zero's.  In the bottom figure (with white continents) the dataset documents
the missing value flag, either with the missing_value attribute or with the
_FillValue attribute.  In the top figure the missing value flag is
undocumented.

If my above guess is correct then to fix this problem you need only include
the command

    SET VARIABLE/BAD=0.0 <my_var_name>

prior to making the plot.  In the context of LAS such a command would normally
by included in the initialization script for the dataset.

I hope this helps.

    - steve
