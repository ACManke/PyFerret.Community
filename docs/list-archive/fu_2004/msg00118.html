Hi Jute,
Please see "fonts" in the Ferret Users Guide. For example, to make
a lowercase beta within a label, for instance, you could use the Complex
Greek Font.  For instance,

yes? plot/i=1:15 1./i
yes? LABEL/NOUSER 5,1,0,0,0.14 "beta is @CGb @AS regular text"

@CG  changes the font to the Complex Greek font, and @AS changes
back to the default Ascii Simplex font.
