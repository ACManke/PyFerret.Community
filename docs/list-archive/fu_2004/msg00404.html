Hi Vincent,

I've been using ferret to work with SeaWiFS data without a DODS server.

I use SeaDAS to output the navigation and data to binary files and read
them with ferret with file/stream qualifier.

Once you have var, lat, lon in ferret, use shade var,lon,lat to get your
image
displayed.

Paulo.
