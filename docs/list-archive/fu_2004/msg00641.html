more details ...
The global SeaWiFS chlorophyll data is available on line via the US JGOFS LAS.   If you have any difficulty accessing this data or navigating the server, please contact me. 

The general US JGOFS SMP LAS URL is: 
http://usjgofs.whoi.edu/las

The SeaWiFS data can be found in the Ocean Color category.

The best way to access the global SeaWiFS chlorophyll data specifically is to access the project description page from Jim Yoder and Maureen Kennelly's US JGOFS SMP work.  

http://usjgofs.whoi.edu/mzweb/smppi/yoder.html
Maureen updates this dataset annually, and this data entity currently includes smoothed 8-day composites for the time period 18 Sep 97 - 31 Dec 03.  The project description page includes links to additional information available from their server at URI.

I hope this information is helpful,
Cyndy Chandler
US JGOFS DMO

Dear Wendy,

Maureen Kennelly at URI has graciously made her NetCDF SeaWiFS dataset
available via anonymous FTP:

ftp po.gso.uri.edu
cd pub/uploads/maureen
get SW19972003_8d_fsm.nc

The data is documented at

http://usjgofs.whoi.edu/mzweb/smpdatadocs/swclim.html
http://usjgofs.whoi.edu/mzweb/syndata.html#SeaWiFS_Chlorophyll

This is the same dataset that is usually available via the WHOI LAS:

http://usjgofs.whoi.edu/las/servlets/dataset?dset=Ocean+Color/Global+SeaWiFS+chlorophyll+1997-2003

Although this LAS server is not working right now, Maureen is working with
WHOI to get it fixed.
