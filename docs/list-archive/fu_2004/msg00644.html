Hi Ansley,

This is what I did.

1) created the ps file from the metafile (myfig1.ps)

2) ps2epsi myfig1.ps myfig1.eps -->to convert ps to eps in UNIX.

3) ps2pdf myfig1.ps(eps) myfig1.pdf -->to convert to pdf file (again in
UNIX).
