Hi Vincent,

Ferret can access .hdf files only through DODS (also known
as OPeNDAP.  Please see our documentation on DODS and
Ferret at
http://ferret.pmel.noaa.gov/Ferret/DODS/ferret_dods.html

If the data can be put on a DODS server, then it can probably
be opened by Ferret, and then you can list it in other formats,
like NetCDF
