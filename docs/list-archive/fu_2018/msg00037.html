Title: Signature

Or, if you're running PyFerret, you can set the color and other characteristics of text with SET TEXT.  This label is an extra label, added by Ferret commands, rather than the automatically generated and placed labels, so it's a "moveable" label.


set text/font=arial             ! or some other system font

... [commands setting up for the taylor diagram] ...

set text/color=red moveable     ! to set the color of "moveable labels"
go taylor_label 2.4 0.6 "MODEL" 

set text/color=black moveable   ! restore the previous setting.
