Is it possible to compute variance for each year in a single scrpt in ferret?

Here Andrew shows a method to calculate a "running correlation":

https://www.pmel.noaa.gov/maillists/tmap/ferret_users/fu_2018/msg00306.html

The method includes (or implies) how to calculate the running variance.  (See my response to Andrew.)

The value you want is the running variance at the center of each year.  (I haven't thought about how to extend it to cases where the length of the year isn't constant.)
