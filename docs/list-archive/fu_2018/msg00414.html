In this case, probably easiest to add a another line color and make it white.  
See https://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/customizing-plots/COLOR#_VPID_243
but basically:
yes? set mode linecolor 7
yes? ppl color,7,99,99,99
(then use pen 7 for white)
