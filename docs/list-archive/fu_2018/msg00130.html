And yes, a Homebrew recipe would ideal for Mac OS X, as that is how I create this package.  Doing this would use your own Homebrew .dylib libraries instead of the ones packaged up in the .tar.gz file and accessed through the DYLD_FALLBACK_LIBRARY_PATH environment variable (defined in the pyferret script).
