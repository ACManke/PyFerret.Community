   * * * Announcing the official release of PyFerret and Ferret version 7.4.2 for Linux and OSX * * *

All of the details are in the v7.4.2 Release notes.

This is primarily a bug-fix release.  In particular, a bug introduced in Version 7.1.1 caused incorrect results for multi-dimensional transforms (such as an XY average), when the region start or end doesn't coincide with a grid cell edge.


As always we encourage Ferret users to upgrade to PyFerret.  PyFerret includes all of the latest improvements that are in classic Ferret. And your existing Ferret scripts will run fine. For Ferret users, here are a couple of handy bits of documentation:

PyFerret for the Ferret User
PyFerret command syntax: quick-start notes. This lists the enhanced options available only in PyFerret.

as well as PyFerret for the Python user


Happy Ferreting ,
