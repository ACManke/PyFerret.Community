Announcing the official release of PyFerret and Ferret version 7.4.3

This is  a minor release with fixes for two serious but seldom-encountered bugs fixed, both having to do with coordinate data in netCDF files. The details are in the release notes. 

For Mac users, please see the GitHub PyFerret release page for details, which are the same as for PyFerret v7.4.2
