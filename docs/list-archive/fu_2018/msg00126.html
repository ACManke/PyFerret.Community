   * * *Announcing the official release of PyFerret and Ferret version 7.4 for Linux and OSX * * *

All of the details are in the v7.4 Release notes.

If you have not yet tried PyFerret, please do! To see what it's all about, have a look at What is PyFerret? and Why Use PyFerret?. There's lots there for the Ferret user whether you know Python or not.  PyFerret includes all of the latest improvements that are in classic Ferret. And your existing Ferret scripts will run fine. For Ferret users, here are a couple of handy bits of documentation

PyFerret for the Ferret User
PyFerret command syntax: quick-start notes. This lists the enhanced options available only in PyFerret.

as well as PyFerret for the Python user

We also support the classic, non-python Ferret.
