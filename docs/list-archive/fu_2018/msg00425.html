On Wed, Nov 28, 2018 at 3:04 AM Karl Smith - NOAA Affiliate <karl.smith@xxxxxxxx> wrote:
Hi,

PyFerret saves PNG images instead of GIF, even if you specify GIF as the format.  PNG format does not support animation that was present in GIF format.
At this time I would recommend creating and saving each image of the animation, and then use another program (such as ImageMagick or iMovie) to put the images together to create the animation file.
Search the Ferret archives about this topic as I think there was already quite some discussion in the last year (maybe two) about this topic (saving animations).

Karl
