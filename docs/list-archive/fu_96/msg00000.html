When contouring using a pen other than black, and then using the "-l ps" option
to
gksm2ps, you will find that not only are the contour lines dashed, but the
contour labels
are also dashed. To avoid this use the PPLUS command conpre to set the labels
to pen 1 (Black)
e.g.

ppl conpre,@P1
