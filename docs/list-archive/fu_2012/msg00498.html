Dear Ferret Users,

 

I have a very low resolution data set and want to plot it in color, with smooth isolines.

I either manage to plot smooth isolines without filing (V1.gif) or unsmoothed isolines with filling

(V2.gif).

 

The commands, which I used are as follows

For V1.gif:

set data EXP.nc

let RF_spec=1.e5*aRF_H2O

contour/fill/levels=(-inf)(0.5)(1)(2)(4)(6)(8)(10)(20)(inf) RF_spec[k=1,l=1]

ppl conset ,,10,,,,,,,

ppl contour

quit

 

For V2.gif:

set data EXP.nc

let RF_spec=1.e5*aRF_H2O

contour/fill/levels=(-inf)(0.5)(1)(2)(4)(6)(8)(10)(20)(inf) RF_spec[k=1,l=1]

quit

 

Does anyone know how I can get  a filled version of V1.gif?

(“ppl smooth 4” Doesn’t help since it smoothes the data not the isolines.)

 

Many thanks for any hint!

I tried so many things I am just lost …
