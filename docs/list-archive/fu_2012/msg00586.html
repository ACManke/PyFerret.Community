Hi,
This error message is coming from the NetCDF library.  Ferret gets the error code back when it issues a netCDF call to  open the file.  Then it makes another call to a NetCDF routine to translate the error code, and the result of that is "Unknown file format".

Unidata has recently added to their documentation a list of their error codes:
http://www.unidata.ucar.edu/software/netcdf/docs/netcdf-c/Error-Codes.html

It means that there is something about the file that's making it impossible for the library to open it as a netCDF file.

Ansley
