Sudev,

If you name the viewports appropriately you can then use the repeat loop
to do the job, it will be something like: 

repeat/range=1:12/name=ll (\
 set view v`ll`;\
 contour/line/fill/lev=($t_levels)/k=1/L=`ll`/nokey/nolab temp;\
)

Regards,

Paulo
