Hi,
Ferret syntax always treats vector components as individual variables.  So to subtract one vector field from another one, you subtract the components.

let a=u[d=1,k=2,l=1:30@ave]
let b=v[d=1,k=2,l=1:30@ave]

let c=u[d=1,k=2,l=1:60@ave]
let d=v[d=1,k=2,l=1:60@ave]

vector/length=5 a-c, b-d
