Dear all,

The monthly climatology file under the monsoon definition category had a bug. The December month was missing on that file due to an error in a ferret script, which was used to combine all monthly files to a single NetCDF file. This is now rectified. So, if you have downloaded monthly climatology based on monsoon definition of seasons, please download it again.

All other files are correct.

Sorry for the inconvenience (if any.)
