Hi -
We need more information. Is the satellite data on a curvilinear grid, where the horizontal location information is in 2D variables?  If that's the case, then you can use one of the SAMPLEXY_CURV functions to get the data at a set of longitude,latitude locations (and depth) and then plot that as an XZ or YZ plot.  There is an example in the documentation under SAMPLEXY  for data on a rectilinear grid; if the data is on a curvilinear grid you would call SAMPLEXY_CURV instead of SAMPLEXY.

SAMPLEXY_CURV:
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/variables-xpressions/XPRESSIONS#_VPINDEXENTRY_369

SAMPLEXY:
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/variables-xpressions/XPRESSIONS#_VPINDEXENTRY_356
