Hi,
Volker and I exchanged a bit more information and looked into this some more.  The FILL command does not make use of the CONSET settings to change the contouring of the variable when making color-filled contours. For the default settings, the color FILL matches the CONTOUR lines, but I do not see a way to match the contours you get with PPL CONSET on a filled color plot.  You can get some smoothing in to the contours by regridding the data to a finer grid.

We are talking about adding options to the CONTOUR command to let you draw a contour plot and color the lines with a color palette.  That would not be the same thing of course but would let us draw these smoother contours and at the same time make use of color on the plot.

Ansley



On 9/24/2012 2:28 PM, Volker.Grewe@xxxxxx wrote:

Dear Ferret Users,

 

I have a very low resolution data set and want to plot it in color, with smooth isolines.

I either manage to plot smooth isolines without filing (V1.gif) or unsmoothed isolines with filling

(V2.gif).

 

The commands, which I used are as follows

For V1.gif:

set data EXP.nc

let RF_spec=1.e5*aRF_H2O

contour/fill/levels=(-inf)(0.5)(1)(2)(4)(6)(8)(10)(20)(inf) RF_spec[k=1,l=1]

ppl conset ,,10,,,,,,,

ppl contour

quit

 

For V2.gif:

set data EXP.nc

let RF_spec=1.e5*aRF_H2O

contour/fill/levels=(-inf)(0.5)(1)(2)(4)(6)(8)(10)(20)(inf) RF_spec[k=1,l=1]

quit

 

Does anyone know how I can get  a filled version of V1.gif?

(“ppl smooth 4” Doesn’t help since it smoothes the data not the isolines.)

 

Many thanks for any hint!

I tried so many things I am just lost …
