Hi Michel,

Use /nokey option to supress the key except for the plot where you want it. There is a PlotPlus command called "SHAKEY" which controls the size, position, etc.  http://ferret.pmel.noaa.gov/Ferret/faq/modifying-the-shade-key?searchterm=shakey

Also when you use same key to represent multiple plots make sure they all have the same levels.
