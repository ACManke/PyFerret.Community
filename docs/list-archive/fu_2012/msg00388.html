Hi Andy,
Yes. The command is:

SET VAR/OUTTYPE=

http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/commands-reference/SET#_outtype
We've had thoughts of a SAVE/OUTTYPE= qualifier which could be used to write multiple variables with a type, but haven't yet implemented that.
