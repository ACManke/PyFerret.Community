Hi Marc,

I usually save an intermediate result of the calculation to netcdf, "cancel data/all" and then re-read in the saved file. If you have problems saving the intermediate variable, take a look at Jaison Kurian's famous "memory tricks" post:

http://www.pmel.noaa.gov/maillists/tmap/ferret_users/fu_2007/msg00489.html

Essentially, this has you write out a variable in "slices" along a given axis (time,lev,lat or lon).

Hope this works.

Paul
