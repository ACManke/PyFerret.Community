On 04 Jan 2012, at 9:29 AM, golla nageswararao wrote:


Hi all,
I am constantly searching for two doubts in ferret for long time and I am going for other softwares for editing my plots. 1. How to get axis (not labels but numbers i.e, lat and lon) in customised font like now default is complex Roman but I want Times new Roman?
You can control the font for any label (axis, contour, title) individually using the @NN syntax before the text in question. (And you can change the default font anytime as well with PPL DFLTFNT). See "PPLUS commands" in the documentation, which gives specific examples for axis labels.
2. PPL veckey is giving vector key. But there is no integer format to mention. Eventhough I used F4.0 I am getting decimal which I want to avoid it. For that I am going back to paint to rub it. How to avoid this in ferret?
As far as I know there is no way to avoid the decimal point in automatic vector key labelling. (But a decimal point alone is very small).
