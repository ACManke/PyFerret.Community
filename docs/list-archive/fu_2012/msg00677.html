Hi Carlos,
Yes, there's a method for this, using the SAMPLEXY function.  You can define a set of x and y locations, and if it's a diagonal, then it's even easier, using the script vertical_section.jnl.  For an example of that,

yes? go/help vertical_section

And the documentation of SAMPLEXY has a bit more.  I hope this will get you started.
