Dear Ferret users,


what is the appropriate way to plot thick contours?  I want to plot
thick, blue contour lines for mean sea level pressure, for example.
First guess:

contour /thickness=2 /color=blue /overlay mslp

**ERROR: unknown command qualifier: thickness=2

Second guess:

contour /pen=10 /overlay mslp

 *** NOTE: /PEN is deprecated. Use /COLOR.

That one does what I want, but it tells me to use /color instead.  How
should I specify thickness, then?  The manual (customizing-plots/CONTOURING) 
suggests that /thickness should work. Is this is something that is
broken, or am I doing it wrong?
