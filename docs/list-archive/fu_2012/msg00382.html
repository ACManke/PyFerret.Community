Hello,
This is to announce the release of Ferret v6.82, and PyFerret 0.0.9.  This version of PyFerret incorporates the double-precision and 6-dimensional enhancements that were part of the Ferret v6.8 release.

There is one bug fix to Ferret v6.8.2: previous 6-D Ferret had trouble correctly initializing the grids for some NetCDF or OPeNDAP datasets.  The release includes several new Ferret scripts as well.

Please see the Release Notes for a full description.
http://ferret.pmel.noaa.gov/Ferret/documentation/release-notes/version-6-8-2-release-notes
