I'd agree with Billy, to start somewhere and play with it. Some of the palettes have colors labeled, which can be helpful as a starting place. Have a look at, for instance, thirty_by_levels. That's a palette that is meant to change noticeably from one level to the next, so it's good only for some particular purposes, but it's a nice place to find colors.
