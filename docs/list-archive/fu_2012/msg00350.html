* * * Announcing the official release of Ferret version 6.8 * * *

This release of Ferret has two important upgrades: All calculations are done in double precision, and we have implemented a 6-dimensioned grid.  The release notes are linked to the Documentation pages, http://ferret.pmel.noaa.gov/Ferret/documentation/release-notes//version-6-8-release-notes

Ferret is backwards compatible, your existing scripts and datasets will work correctly with this version.

This Ferret release includes executables for Linux 32-bit and 64-bit linked with NetCDF-4.3 libraries and for Mac OSX. 

To install Ferret, follow the instructions on the Downloads pages of the Ferret web site
http://ferret.pmel.noaa.gov/Ferret/downloads
