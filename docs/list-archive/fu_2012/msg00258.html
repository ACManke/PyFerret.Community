Hi -
You will need to create a true monthly time axis and regrid your data to it.  The descriptor file framework will create a correct axis based on the files you list in the descriptor, but it doesn't have a way to define the "edges" between grid cells.

There is some discussion of defining monthly axes in the Users Guide, see "DEFINE AXIS/EDGES".  Do you want a Julian calendar, or Ferret's default Proleptic Gregorian one?

Ansley
