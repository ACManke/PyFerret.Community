Hi Verena,

 In Ferret, after extracting the appropriate level, using k=@ave  removes the k-dimension.

le uu = uwnd[k=8]
let u = uu[k=@ave]
save/clobber/file = new_data.cdf   u

Then, new_data.cdf is 3D.

Best

Hi Verena,

You can save 200 hPa U in new netcdf file, like this:
Assuming that U at 200 hPa is equal to k=1

save/file=u200.nc u[k=1]

Then, the new file u200.nc is 3D.
