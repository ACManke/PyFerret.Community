Hi Carol,
Everyone, Carol and I talked off-line and I think we have tracked this down.  It's a bug in the changes we made for double-precision Ferret, so that if there are more than 1000 variables in the datasets, information that Ferret stores for the variables and their datasets and grids is being lost.

This will be corrected in the next Ferret release.

Ansley
