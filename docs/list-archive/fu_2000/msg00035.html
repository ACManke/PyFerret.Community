Some nice tools and suggestions there, Billy!

One suggestion, you might modify the start-up file to check if the fjnl
subdirectory exists, and if not create it. I tried doing it with spawn, and
ferret doesn't seem to appreciate the command. So you could create a single
line command in your bin directory (e.g., jfile) with the two line command:

#/bin/csh
if ( ! -e fjnl ) mkdir fjnl

Make it executible, then on the front of your startup script add:

sp jfile
