Hi Karen,

Define the X and Y points along which you would like to sample the field
using the LET command. Then use SAMPLEXY to obtain your answer

     LET xpts = ...
     LET ypts = ...
     CONTOUR/Z=0:500 SAMPLEXY(my_3d_field, xpts, ypts)

    - steve
