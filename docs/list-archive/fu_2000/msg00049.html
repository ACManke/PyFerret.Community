Alberto,

Here are some suggestions for creating black & white graphics with
Ferret.

1) Use CONTOUR instead of FILL or SHADE.


2) Try FILL/PALETTE=grayscale your_variable.

Documentation on creating your own palettes is at:

http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/v50/partII.html#_2_10


3) Try using the patterns available in Ferret 5.0.

An example is available in the 'pattern_demo' script which is part of
the v5.0 release.

Documentation on using patterns is at:

http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/v50/partII.html#V5.0_PATTERN


Try some combination of all three.  Very nice B & W graphics can be
created with a tasteful combination of patterns, grayscale palettes and
contours.

I hope this helps.


-- Jonathan Callahan
