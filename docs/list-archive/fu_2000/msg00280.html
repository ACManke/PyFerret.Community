Hi Jörg,

This is easy to do if you use the file/format=stream command (version 5.0 required).

For instance, to read a sequence of two byte integers you would use:

file/format=stream/type=i2/var=foo foo.dat

They are all internally converted to float (4 byte floating point) by Ferret, so
for larger integers you might lose some precision.

More details are available in the online Ferret documentation at:

 http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/v50_2000/Chapter2.htm#_VPID_45
