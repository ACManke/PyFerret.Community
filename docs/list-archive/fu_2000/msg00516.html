Hello, 

	whenever you don't require the lables first thing to do is
ppl/reset command to be issued, then the variable which you want to plot
should be issued as "plot or vector or contour or shade"(any one of
them)/set_up variable name (for ex shade/set_up temp) command to be issued
then go unlabel after that you should plot using ppl command.  this will
make you to remove unwanted labels in the final plot.  the commands from
ferret follows like this
ppl/reset 
plot/set_up variable 
go unlabel 1 
go unlable 2 
ppl plot         etc.  


Hope this will help you in getting the plot correctly.
