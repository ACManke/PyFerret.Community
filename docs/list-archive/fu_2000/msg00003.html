Hello,

I have the following ferret script:

define axis /t=1-DEC-1999:1-MAR-2000/npoints=11 bt
define grid /t=bt bg
file /var=temp/grid=bg temp.dat
plot temp

where temp.dat is just a column of 11 values going from 0.0 to 10.0.
When I run it with ferret 4.91 (on a SGI R4000 Indigo running IRIX 6.2),
the time axis does not show up on the plot.  If I change the time axis
to go from 1-JAN-2000 to 1-APR-2000, the problem disappears.  So I
thought that this was a Y2K bug, but if I change the time axis to go
from 1-DEC-1999 to 1-APR-2000, everything is correct.  What is the
problem ?  It shows up also if I go from 1-DEC-1899 to 1-MAR-1900.

This is the version compiled for IRIX 5.x.  Unfortunately, ferret 5.0
compiled for IRIX 6.x uses the MIPS4 instruction sets, which is not
accepted by the machine I use.
