Hi Lev,

The formatted strings "60W" and "81.66N" will not be understood by the
LABEL command.  Use the equivalent numerical values. (Ferret can
translate the formatted strings using grave accent notation, but it takes
a couple of messy steps. Probably not worth it unless you need to
automate this.)

The lack of error message is because "LABEL" is an alias for "PPL
%LABEL". i.e. the command is sent directly to PLOT+ without syntax
checking. PLOT+ is pretty stingy with error messages.

    - steve
