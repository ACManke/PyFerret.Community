Hi Celine -

The problem you are seeing with the MC descriptors you are trying to use has to do
with namelist formatting differences between f77 and f90.  For the linux build of
Ferret v5.22, we used Nag F90, and therefore it expects f90-style namelist formats.
The descriptor file you have falls into the f77 format style for namelist reads.

There is a script in our anonymous ftp which attempts to "linuxize" f77 style
descriptors by converting them to f90 namelist format.   You can pick it up at:

     ftp://ferret.wrc.noaa.gov/pub/bin/bin/linuxize_descriptors

It's possible that this script will not "linuxize" every descriptor out there, but it
at least should give you a good idea of what is required.

If you have other questions, please let me know.

Kevin
