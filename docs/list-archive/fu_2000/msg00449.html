Hello Karen Bice -

Two things are wrong here. First, the depth should be input 
in meters, not cm. That will improve your results right away. 
Second, for most purposes you will want to remove the pressure
effect from densities. This is done by using Theta (potential
temperature) rather than in situ temperature. Do this using the
function THETA_FO, and then calculate your density with Theta,
not temperature, and pressure zero.
