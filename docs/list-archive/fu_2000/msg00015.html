I want to have fill patterns such that values below some threshold have
a pattern as well as a fill color. I can do this with an overlay of a
pattern fill. The code I am using is

fill/set/key/palette=white_centered/pattern=solid/level=(-8,8,1) DATA
pplus fill
fill/overlay/set/key/palette=white/pattern=lite_up_right_to_left/level=(-8,-2,6)
DATA
pplus fill/overlay

However, in the color key the pattern spans the entire color key rather
than just the range I want. Any suggestions?
