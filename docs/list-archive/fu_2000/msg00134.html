Hi Yang,

This can be done far more simply in Ferret V5. Try

yes? let lev = {25.0, 81.25, 156.25, 275.0, 462.5, 712.5,1175.0, 2000.0, 3000.0
, 4000.0, 5500.0}
yes? define axis/from_data/depth/Z/name=my_levels/units=meters lev

    - steve

-------

Note that in V5.1 it is still simpler:

 DEFINE AXIS/DEPTH/UNITS=meters my_levels = {25.0, 81.25, 156.25, 275.0, ....}
