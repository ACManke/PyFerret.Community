Hi Patrick,

Yes - you can control this.

Check out the MARKH ("mark height") command in the PLOT+ Users guide
(http://ferret.wrc.noaa.gov/Ferret/Documentation/PPLUS_Users_Guide/pplus_users_guide.html)

Note that the command "PPL MARKH pen,size" is "persistent", so you use
it as in this example:

yes? ppl markh 1,.2    ! set mark size for future plots
yes? plot/vs/i=1:10 i,i^2    ! make the plot
