I realise this message is slightly off topic, but I just wanted to
highlight the  excellent compression of netcdf files offered by the
relatively new compression tool 'bzip2'. It's a high quality data
compressor freely available on the net.

As an example, on one of my typical 200Mb uncompressed netcdf files, bzip2
manages to get this down to 26Mb whereas gzip can only manage 158Mb (both
on default settings). The speed of the tool is equally impressive.

The web page for bzip2 is "http://sourceware.cygnus.com/bzip2/";

I hope this is of use to the ferret community.
