Hi Gabriel,

Easy:

   DEFINE AXIS/units=meters/depth my_zax = {0,20,40,60,100,180,300}
   SHOW AXIS/z my_zax

But the point on the surface via this scheme will cover the range (-10,10). If
you'd like your surface point to represent, say, (0,10) then you should define the
grid points via their boundaries. See more under the /EDGES qualifier for DEFINE
AXIS in the on-line Users Guide.
