	There is no guaranteed *easy* way to do this. You can try playing
around with the shakey command (see the PPLUS+ manual in the doc
subdirectory of FER_DIR).

1) leave the labels off the key and write them yourself using LABEL
2) let ferret put the labels on and then try to "white out" over this,
   there may some way to do this in ferret (see squares.jnl, maybe)
