Yes Ferret can save output as ASCII.  For instance you could use the
following to save two variables in a specified Fortran format: 

 cancel list/head
 list/format=(2f8.4)/file="myfile.dat"/clobber variableA, variableB

I would suggest consulting the Ferret documentation.
