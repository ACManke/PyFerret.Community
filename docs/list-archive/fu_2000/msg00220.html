Thanks to Mark Verschell, here's the solution to my earlier
query. To pass a title string for a go file plotting routine
running on Linux use the following format:

call the go file as:
GO mygofile myvariable2plot "MY TITLE"

then within the go file use for eg shading:
mygofile.jnl:
...
shade /set_up $1
ppl title, "$2"
ppl shade
