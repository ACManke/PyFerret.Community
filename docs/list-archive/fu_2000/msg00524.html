Hi Will,

Two methods:

  1. Redefine the "indicial" axes using (e.g.)
     DEFINE AXIS/X=<lo>:<hi>:<delta> `my_var,RETURN=xaxis`
  2. define the axes and use use "regridding by association" (G=@ASN)

Check the Users Guide for details
(http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/v52/fer_html.htm)

    - steve
