Natalia,

Perhaps you are not aware of the map projection capabilities available
in Ferret v5.x?

The following Ferret commands produce a polar plot with no gap:

use levitus_climatology
set grid temp
go mp_stereographic_north 180
set region/y=20:90/k=1
go mp_aspect
contour/noaxes temp, x_page,y_page
go mp_fland


More information about map projections is available at:

http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/v50_2000/Chapter6.htm#_VPID_177


-- Jonathan Callahan
