Mr Retish Senan wrote:

Hi Karen!

        The correct usage of the rho_un function is
                        rho_un(sal,temp,Z[G=temp]).

So, the strange values you are getting is due to the fact that the
function is interpreting your temperature as salinity and vice versa..

        The confusion must have arised because the user's guide V4.91 has
given the wrong usage in an example on page 40. The correct usage is as
given in the list of functions on page 42.

Regards
Retish
