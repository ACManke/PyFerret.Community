Haijun,

	I have never found much use for the axnsig command, it is used for auto
labelling, and seems not useful. What you might find more useful is the PPL
command XFOR and YFOR. To get 3 decimal places for example you would use
PPL XFOR,(f4.1)

Hope this helps,
