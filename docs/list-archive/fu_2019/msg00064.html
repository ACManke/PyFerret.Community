Hi,

It looks as though the axis needs to be marked as a "depth" axis. In the file, the coordinate variable needs an attribute

   ST_OCEAN :positive = "down" ;

You can correct how Ferret sees the axis within the Ferret session,

yes? use my_data
yes? set axis/depth `var,return=zaxis`

Then the "show grid" would have a minus sign showing the direction of the axis


    GRID GBT1
 name       axis           # pts      start                   end              subset
 XT_OCEAN  LONGITUDE       720mr     0.25E                    0.25W            full
 YT_OCEAN  LATITUDE        330 r     74.75S                   89.75N           full
 ST_OCEAN  Z (meters)       50 i-     5.0335                5395               full
 TIME      TIME              1 r      17-MAR-2004 13:00    17-MAR-2004 13:00   full
