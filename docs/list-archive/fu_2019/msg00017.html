Dear ferreters,

I have zonal averaged variables, say temperature. I want to compare them by latitude, for example, temperature at 30N and 30S. So the figure should be like latitude from 0 to 90 for the x-axis and zonal temperature for the y-axis. What I planned to do was to re-grid the south hemisphere latitude to north hemisphere latitude first then overlay it on top of the north hemisphere zonal averaged temperature. According to my sh grid command, my re-grid process was successful: sh grid south_EB
    GRID SOUTH_EB
 name       axis              # pts   start                end
 XAX       LONGITUDE          257mr   0E                   1.411W(358.59)
 YAX       LATITUDE            65 r   0                    88.928N
 normal    Z
 normal    T
However, when I plot it out, it still "remembers" its original south hemisphere latitude. Is there a way or a better way to compare south hemisphere data to north hemisphere data?
