I would urge you to first run and study the ferret demos before asking folks on the list, eg 
go mp_demo
but in this case, had this handy:


set reg /y=-90:-60
go mp_stereographic_south
set grid hice
GO mp_grid hice
shade/noaxes/nolab hice*mp_mask, x_page, y_page
!CONTOUR/nolab /over /LEVEL= "(-400, 400,400) (600,3200,200)" hd*mp_mask, x_page, y_page
go mp_land 10
!cont/nolab/col=8 /over /lev=(0,0.1,0.1) (H-0.2)*mp_mask, x_page, y_page
go mp_label 180,-46,0,0,0.2 @P7TITLE
go mp_label 165,-46,0,0,0.2 @p7(mm/yr)
