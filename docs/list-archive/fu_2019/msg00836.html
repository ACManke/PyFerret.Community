you can save the figure into the -nodisplay mode of pyferret.

just restart pyferret with this command (-nodisplay supress the graphical window and nojnl will not create the temperory jnl file (~jnl))

so just run this command you should be able to save it into .ps and .eps or any other format. If you encounter the same problem with any other extension format.

pyferret -nodisplay -nojnl
