Hello Pratik,

Thank you for your question. Unfortunately, I don't think you'll be able to make such a plot in PyFerret as it has limited 3D plotting capabilities.

In particular, the WIRE command is a part of the PPLUS package that is incorporated in PyFerret. It looks like there have been very few updates in this section of the codebase since 1988, and apparently WIRE was never enhanced with additional options like axis labels.

Best regards,
Joshua Osborne
