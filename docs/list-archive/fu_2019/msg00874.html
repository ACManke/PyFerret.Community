Ryo,

Time axes always request for a special, dedicated treatment :

SET DATA sample.nc

FILL/TRANSPOSE/HLIMITS=34S:22S/VLIMITS=0:365/NOKEY/SET_UP eta
PPL txlabp, 1 
PPL FILL


However, even if the axis is on the right like requested, the result is pretty bad on my plot :-((

(with PyFerret v7.63)
