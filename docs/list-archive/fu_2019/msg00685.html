G'day

I think you can manage it by 5th and 6th arguments in the PPL Shakey Command.

ppl shakey 1, 1, 0.19, , 6, 6, 6.45, 6.65, 0.5, 3.6

0.19 is the size of the label in the color key

and 5th position (6) is the number of significant digits you want so eventually it will count that and put it with space. you can play with it. I hope it will work.
