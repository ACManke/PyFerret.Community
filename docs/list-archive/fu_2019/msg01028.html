Please, how do I insert or paste a logo on a plot? 

The logo is in gif format. For example:
...................................................
let my_logo = logo.gif
!let symbol = my_logo
 
!LABEL/over/NOUSER 5, 4, 1, 0, 0.60 "symbol"    !my_logo
!plot/over/title="m<sup>my_logo</sup>" my_logo
 
set mysymbol = "my_logo"; annotate/over/user/x=0/y=50/siz=0.2/halign=0/valign=0 "<&mysymbol>"
