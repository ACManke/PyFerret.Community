  * * * Announcing the official release of PyFerret and Ferret version 7.5   * * *

This includes a few new features including more flexibility in defining the time axis aggregating datasets in time DEFINE DATA/T;  a new SAMPLEXYZT function and an option to list date-strings in ISO 8601 date format.  There are also a few bug fixes.  The pre-built executables are linked with the latest NetCDF library release, netCDF 4.6.3.

The release notes are here:
https://ferret.pmel.noaa.gov/Ferret/documentation/version-75-release-notes

The release is available from the GitHub pages, as listed in the Ferret Downloads pages.

Ansley Manke,  Karl Smith
