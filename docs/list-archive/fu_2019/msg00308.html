Hi,

It should be:

yes? use coads_climatology
yes? fill/palette=no_blue_centered/set  sst[l=1]
yes? ppl shakey 1 1 0.13
yes? ppl fill

Issuing ppl after the initial "fill" or "shade" requires a /set.

See one of the latest threads in the ferret mailing list!

Best,

Martin
