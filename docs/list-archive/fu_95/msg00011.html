model. The model has 20 levels everywhere, but at every x-y location the depth
of those levels is a function of the total depth at that position. Currently,
this is handled by transforming the sigma coordinates to a grid with regular
depths, but I would much prefer to be able to work with the data at the correct
depths - not just so the bottom is portrayed correctly.

Has anyone done this before, or does anyone know of a way to do this?

Of course this is also complicated because the x-y locations are irregular.
