you can use list command to extract the values into an ascii format.say for an example

use test.nc
sh grid temp
then use the grid name of the variable temp in the following command ,

list/file=test.dat/x=40e:50e/y=5s:5n/format=(10f8.4)/clobber  x[g=gcb1],y[g=gcb1],temp[g=gcb1]

hope it helped you
