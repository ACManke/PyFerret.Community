Dear Nitin Patil,

this is certainly possible but it can be tricky sometimes.  What you
want to do is roughly

- define a common grid
- put both variables on the same common grid
- create a new variable which is HR if available, otherwise lowres.

Given variables hr and lr:

let hr_on_lr_grid = hr[gx=lr,gy=lr]
let combined = if (hr_on_lr_grid) then hr_on_lr_grid else lr

You can also define a new grid to put the data onto, for example a
global high resolution grid.
