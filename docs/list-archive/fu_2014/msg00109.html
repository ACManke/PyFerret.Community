Hi,
You can redefine a variable, but first you need to give the original varaible a new name.  SET VARIABLE/NAME does this.
yes? set variable/name=h_snow_in h_snow

yes? let h_snow = h_snow_in*0
yes? set attributes/like=h_snow_in h_snow

yes? save/append/file=ice_model_0sicsit.res.nc h_snow
Any time you define a variable, you'll want to give it attributes - units, title, etc.  You can either do that directly with LET/TITLE= /UNITS= or inherit them from the file variable with the SET ATTRIBUTE command.

Ansley
