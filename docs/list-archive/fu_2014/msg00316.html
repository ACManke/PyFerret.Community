Hi Molulaqhooa,

The @AVE operation that you are using here

   let SST1 =SST[X=180E:180W:0.5@ave,Y=90N:90S:0.5@ave]
   SAVE/FILE = 0.5_ROMS.nc SST1

is not a regridding operation but the @AVE transformation which returns the variable averaged over the x and y regions you specify.   Open up the file and look at the variable and its grid:
yes? cancel variable/all
yes? use 0.5_ROMS.nc
yes? show data
There will just be an axis in time, not in X and Y.

In the documentation, what your commands are doing is this:
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/variables-xpressions/XPRESSIONS#_VPINDEXENTRY_439

not this one:
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/Grids-Regions/GRIDS#_VPINDEXENTRY_654

Ansley

Here is the regridding I performed which regridded the file successfully:

SET DATA ROMS.nc
let SST1 =SST[X=180E:180W:0.5@ave,Y=90N:90S:0.5@ave]
SAVE/FILE =0.5_ROMS.nc SST1
