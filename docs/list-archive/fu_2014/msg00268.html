Jean, in what you wrote, you did not actually define the time axis to be mytime.  If you do, then this should work


define axis/t="10-APR-2014:06:15":"01-AUG-2014:11:00":15/units=minutes mytime
let xtms=t[gt=mytime]
plot/over/line/vs {`xtms[t="1-may-2014 12:00"]`, `xtms[t="1-may-2014 12:00"]`}, {-100, 100}
