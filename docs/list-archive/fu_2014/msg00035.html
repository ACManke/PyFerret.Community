Otacilio,

One way around would be to create an empty axis and latter put manually the time dates in there. 
If you are going to produce a set of similar figures you could use the same code afterwards. It can be boring, but shouldn't be too difficult.

You could use something like:

label/nouser `(($ppl$xlen)/12)*1` -0.2 0 0 0.2 "16-JAN"
label/nouser `(($ppl$xlen)/12)*2` -0.2 0 0 0.2 "16-FEV"
label/nouser `(($ppl$xlen)/12)*3` -0.2 0 0 0.2 "16-MAR" 

and so on.

Regards,

André Rodrigues
