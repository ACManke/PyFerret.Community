Another way to do it, is to define an irregular time axis - per this post: http://www.pmel.noaa.gov/maillists/tmap/ferret_users/fu_2012/msg00267.html

You would have to convert your dates to, call it TIME STEPS. There is a ferret function to do this DAYS1900. I refer you to the post above which details the procedure.
