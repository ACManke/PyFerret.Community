Dear Adriano,

> I'm struggling to read a .bot file which is swan bottom depth file. It has
> 438 columns and 164 rows.
> 
> The command I'm using:
> 
> define axis/x=1:438:1 xaxis
> define axis/y=1:164:1 yaxis
> define grid/x=xaxis/y=yaxis grid_bat
> 
> file/grid=grid_bat/var=bat grid_RJ.bot

add /columns=438  to your file statement and it should work.  You'll
also need to specify the missing value with something like

set var /bad=-99 bat

Of course it's even nicer if you can define the axes with proper lon
and lat coordinates immediately.
