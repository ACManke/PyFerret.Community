Hi Sarah,

This signal indicates that there is a memory reference problem. If you type

man 7 signal

on you machine you can see the various signals.

In your case I think you are requesting more memory than your system has readily available and Ferret is running into problems. I guess it keeps filling its memory cache until it hits the limit and then start to flush it.

You are requesting 20K megawords of memory which is equal to 80GB! Quite a bit and I guess is way more than you have available. Try asking for more reasonable amount.

Also if this is a large 3/4D calculation it may be more suitable (and considerably quicker) to break up the calculation into smaller amounts.
Say just a few latitudes at a time and save the results to disk or save sigma at each time step and then read it back in order to calculate the time average.

Russ
