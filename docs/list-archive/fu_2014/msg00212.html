Hi Fisseha,
The ZAXREPLACE functions are designed to handle this issue. You have a function defined on a "levels" axis, and another function that defined pressure as a function of level. ZAXREPLACE will make the conversion for you, so that your temperature will be on a pressure axis that you define. 

Ansley
