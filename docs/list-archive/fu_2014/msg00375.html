Dear PyFerret users,

I apologize for the minimal information about the changes to PyFerret itself, in addition to the changes in the Ferret engine it uses.  Briefly, the default window and other drawing sizes were modified to try to completely mimic Ferret default sizes, and when saving PDF and PS images from displayed windows, the image is saved on to a "page" that exactly fits the image (so not cutoff and not extra whitespace) if not using the RHEL5 version.

If you are want to use the Ubuntu-12.04 versions of PyFerret: 
Due to problems trying to regenerate the netcdf libraries for Ubuntu-12.04, version 1.1.0 of the Ubuntu-12.04 builds require the installation of the "netcdf" package (which currently provides NetCDF version 4.1.1) from the software package manger.  (For instance, "sudo apt-get netcdf") You may also wish to install the "netcdf-bin" and "nco" packages if you want the NetCDF programs (such as ncdump and ncattr) for working directly with netcdf files.

Karl
