Oops, I see I left off the "d=" for the global data set in my definitions. Silly cut and paste...

That's right, you don't need those 2 lines. They are really just there to make the masking clear.

You shouldn't need to specify the time range in your definition of emiss_awb_bc_common

let EMISS_AWB_BC_COMMON = if  EMISS_AWB_BC[d=2] then EMISS_AWB_BC[d=2] else EMISS_AWB_BC[d=1]

It's much cleaner and more general. As a matter of style, I like using the full names of datasets (or a descriptive symbol) rather than numbers since it's often easier to work out what's going on but that's just personal preference.

Russ

Hi,

Just use a mask to select the indian mask as follows

use global_T63.nc
use india_T63.nc

let mask = if EMISS_AWB_BC[d=india_T63] then 1
let EMISS_AWB_BC_COMMON = if mask then EMISS_AWB_BC[d=india_T63] else EMISS_AWB_BC[global_T63]


In fact you even need to create a mask

let EMISS_AWB_BC_COMMON = if  EMISS_AWB_BC[d=india_T63] then EMISS_AWB_BC[d=india_T63] else EMISS_AWB_BC[global_T63]
