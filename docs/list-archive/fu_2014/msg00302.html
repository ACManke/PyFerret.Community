You need to use the THETA_FO function.  See the page on this in the
Ferret Users manual at

http://ferret.pmel.noaa.gov/Ferret/documentation/release-notes/users-guide/variables-xpressio$

In short, to compute the potential temperature at S=35, T=20°C, and
p=5000 db, use

  let Tpot     = theta_fo(35,20,5000,0)
