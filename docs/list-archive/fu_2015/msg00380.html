Dear Simon,

In ferret, this is basically regridding again, this time to a monthly axis for example.

You need to define a monthly axis and then you can regrid onto that monthly axis using one of the operators, e.g. @sum.

There are some examples in the user manual.

You can save the regridded variables in a netcdf file with the “save” command.

For tasks like those nco might be faster and even easier.

‘hope this helps,

Best wishes,
