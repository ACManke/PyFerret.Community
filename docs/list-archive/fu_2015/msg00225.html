Hi,
Ferret versions 6.6 and higher are linked with the NetCDF-4 and HDF5 libraries. NetCDF-4 uses HDF5 features for compression and chunking, and it can read some HDF5 files. Depending on the file itself, Ferret may be able to open and read HDF5, files, but there are HDF5 files with features and structures that Ferret does not handle. 

 Here are a few messages from the archive with possible suggestions
http://www.pmel.noaa.gov/maillists/tmap/ferret_users/fu_2013/msg00073.html
