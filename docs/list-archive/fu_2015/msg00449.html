Hi Ferreters,

Can I position the [sha]de key using [norm]alised coordinates?

[sha]: http://ferret.pmel.noaa.gov/Ferret/faq/modifying-the-shade-key
[norm]: http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/commands-reference/annotate

This is especially useful when putting a colour bar together with a
label.  So I want something like:

    yes? ppl shakey/norm/xpos=1/ypos=0            [other options]
    yes? annotate  /norm/xpos=1/ypos=-0.1 "m^3/s"

or even better by using a new shadekey command akin to annotate, or
maybe the option of putting other things than text as the last argument
in ANNOTATE (does a colour bar count as 'annotation'?).

The ANNOTATE command was introduced in Ferret 6.95.
