Hi Patrick,
Thank you for writing.  There's been some bubbling discussion of the shortcomings of the Rainbow palette in the visualization software world for a while.  We'd welcome more discussion here - I know there are fans of rainbow palettes as well. 

Note that right now you can set a default palette for Ferret sessions, either with a startup file, or within scripts.  Add the command

  PALETTE palette_name

to a .ferret file, or to a script, and subsequent plots will use that palette.
