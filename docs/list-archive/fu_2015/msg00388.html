Hi Nitin,

> When I am zooming the eps (attached) file the font used for labeling
> x,y,scale and title are not filled (only border is highlighted). Any idea, I
> am using @TR but same for DR as well.

I recommend trying increasing the line width of the letterings.  I
forgot if there is a native method to do that, but you can thicken
lines in Ferret-generated EPS files.  Search the Ferret email archive.

Other than that, I think you would have to replace letterings manually
using an application to edit PostScript files, like Inkscape and
Illustrator.

Having said that, I wonder if your problem matters in practice.  If
you REALLY need to zoom in that much, that would indicate that your
plot includes too much detail.
