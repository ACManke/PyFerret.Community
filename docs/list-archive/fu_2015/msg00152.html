Hi Francois,
You could use the FILE/GRID= command to read the data and put it on the same axis as pclim. 

Since you've created the climatology, you can name the time axis you used, or get it from the grid of pclim with:
yes? define grid/t=`pclim,return=taxis` timegrid
yes? file/skip=1/var="mp p1"/grid=timegrid p.dat
