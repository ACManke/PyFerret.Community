Hi,
The BUFF EMPTY is happening because after a SHADE/SET you would finish the plot with a PPL SHADE. 

To draw the label you need to use -50 for the Y coordinate, as the region is in the southern hemisphere.


   yes? shade/i=50:90/j=20:120/k=1/l=1 SALT
   yes? label/user,70,-50,.034,0,satyam
