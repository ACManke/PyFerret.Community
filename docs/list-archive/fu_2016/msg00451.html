Hi Patrick,

I would like to generate images without any margins.

Mine is not a Ferret answer, but you can always use external tools to chop off the margins:

$ mogrify -trim yourbitmapimage.png  # overwrites
$ pdfcrop yourvectorimage.pdf cropped.pdf

To trim EPS, you can use ps2epsi .  (If your system doesn't have these tools, you can find them by searching the Internet.)
