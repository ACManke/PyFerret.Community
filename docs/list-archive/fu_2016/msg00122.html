Hi ferreters,

 

Sorry if this was addressed already and I did not find it in the list archive.

 

Something went wrong with /vlog/vlim (also /hlog/hlim) logic while in the newer ferret version(s), probably.

 

Attempting the following is OK and produces the plot:

 

yes? let a = 1e-12 + x/1e12

yes? pl/vlog/x=0:1000 a

 

whilst with /vlimit given this bails out:

 

yes? pl/vlog/vlim=1e-12:1e-9/x=0:1000 a

**ERROR: value out of legal range: Limits for log axis negative or too small: -11.0 : -7.00

 

which is nonsense. The logic broke somewhere between v. 6.85 and 6.96.

I’ll appreciate a fix.

 

Cheers,

Sergey
