Dear All,

i am working on the 114 years of SST data, i am having slope of SST (spatial trend) from go regresst command for three time periods of 40, 20 and 54 years. i plotted the spatial pattern of slope for these time periods. now i want to plot slope (spatial trend) at 95% level of significance for these three time periods. i not getting any help for this calculation and which significance level test is suitable for slope from regression script. i hope that i will get help from ferret community.

this is the specification of my data.

name     title                                                 I         J         K         L         M         N
 SST      sea surface temperature (degc)   1:360     1:180     ...       1:1368    ...       ...

name       axis                         # pts           start                     end
 LON       LONGITUDE              360mr      179.5W(-179.5)       179.5E
 LAT       LATITUDE                180 r            89.5S                   89.5N
 normal    Z
 TIME      TIME                    1368 i   01-JAN-1901 00:00    01-DEC-2014 00:00
 normal    E
 normal    F

this is the script for slope calculation

use sstnew.nc

let/quiet p=t[gt=sst]
let/quiet q=sst

go regresst
set viewport upper
fill/hlimits=35E:75W:20/x=35E:75W/y=20S:25N/palette=blue_orange slope
