The pyferret.stats module is provided in the pyferret executables installation tar file, so I am not sure why it would not be found.  
You might want to make sure these files did get installed under $FER_DIR/lib/python2.6/site-packages/pyferret/stats/
You might also verify that $FER_DIR/lib/python2.6/site-packages is given in $PYTHONPATH (where $FER_DIR will be translated to whatever the actual path is).
