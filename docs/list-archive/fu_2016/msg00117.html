Dear Ferret's users,

I have a problem with a XY projection.
My data is from Matlab output in (x,y) netcdf file.
It has a format like this:
-----
yes? use seasons.nc 
sh d/v
     currently SET data sets:
    1> ./seasons.nc  (default)
 
 name     title                             I         J         K         L
 DATAVAL                                   1:100     1:160     ...       ...
               on grid GRF1 with -1.E+34 for missing data
             X=0.5:100.5  Y=0.5:160.5  
 LONGITUDE
                                           1:100     1:160     ...       ...
               on grid GRF1 with -1.E+34 for missing data
             X=0.5:100.5  Y=0.5:160.5  
 LATITUDE                                  1:100     1:160     ...       ...
               on grid GRF1 with -1.E+34 for missing data
             X=0.5:100.5  Y=0.5:160.5  
------

Actually, the data should has a dimension from (X= 90E - 130E) and (Y=12S - 5N) with resolution of 0.25x0.25 degree.
My question is how to transform the data into (lon,lat) netcdf file?
