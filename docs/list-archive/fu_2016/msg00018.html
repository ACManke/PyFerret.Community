Hi Lev,
Double -precision Ferret will write double-precision output. So that would be the first thing to test; perhaps write some small test files and read the data using a short Fortran test program, reading into variables declared as real*8.
There is not an option to write single-precision data with double-precision versions of Ferret; binary and stream file IO is in Ferret but not actively supported. 

Ansley
