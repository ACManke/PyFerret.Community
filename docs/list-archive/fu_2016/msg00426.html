Dear Chinnu,

On an /OVERLAY plot, any specification of /VLIMITS or /HLIMITS is ignored.  The ranges from the first plot are used for all overlays

Also, the PPL SHAKEY command that you used, is applied only to a command issued with /SET.  So, first draw and finish the FILL plot, then do the CONTOUR/OVERLAY.

   fill/set/d=1/vlim=35:180:25/lev=(-inf)(-0.7,0.7,0.1)(inf)/nolab/nokey u
   ppl shakey 1,0,0.16,2,,,9,3,0.25,0.5
   ppl fill

   contour/ov ...
