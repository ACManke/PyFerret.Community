The documentation is here:
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/commands-reference/PLOT

   PLOT/VLOG

will automatically select limits, PLOT/VLOG/VLIMITS=lo:hi lets you specify a range. The values are specified in data units, not log(units).  This means that /VLIMITS can't start at 0.
