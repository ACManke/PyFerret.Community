Hi,
Let me just put in a plug here for PyFerret, which has some nice simplifications and enhancements for handling text - and lets you choose from the fonts available on your system.   The next release should include a PyFerret build for the Mac as well.  See the attached image.
> pyferret

yes? cancel mode logo
yes? set text/font=arial

yes? use etopo20
yes? shade/pal=grayscale rose

yes? set text/font=arial/color=lightblue/bold/italic
yes? annotate/user/xpos=180/ypos=0/HALIGN=0 R1009_3Drest_ocs_C.cice.h.nc
The ANNOTATE command is available in Ferret as well as PyFerret.
http://ferret.pmel.noaa.gov/Ferret/documentation/users-guide/commands-reference/annotate
