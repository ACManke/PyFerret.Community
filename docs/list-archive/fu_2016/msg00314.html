You asked this:

I need to calculate mean of datavalues on a slanted line.SAMPLEXY helps to pick the data values along a slanted line.Statistics/ brief command gives the mean value. But dont know how to assign that mean into a variable. So that i can make a loop to calculate mean over a number of slanted lines.

There are two answers to this.  You have defined a variable,
yes? let sampled_data = SAMPLEXY(variable, xpts, ypts)
The result is a list of values in the X direction (it can also depend on other dimensions such as Z, as seen in the example in the Users Guide.)  You can use a transformation in the X direction to get the mean:
yes? let sample_mean = sampled_data[x=@AVE]
The above is the best answer.  It lets you make any computation on the result of the function.

The second answer, is that once you have issued any STAT command, the results of that are saved in symbols.  Here is the output of a variable I computed using the example in the Users Guide
yes? stat slantsalt
             SAMPLEXY(SALT[X=200:300,Y=0:30],XLON,YLAT)
             X: 0.5 to 50.5
             Y:  N/A
             DEPTH (m): 0 to 5000
             T:  N/A
             E:  N/A
             F:  N/A
             DATA SET: /home/users/tmap/ferret/linux/fer_dsets/data/levitus_climatology.cdf
 
 Total # of data points: 1000 (50*1*20*1*1*1)
 # flagged as bad  data: 174
 Minimum value: 32.424
 Maximum value: 35.006
 Mean    value: 34.435 (unweighted average)
 Standard deviation: 0.3948

yes? show symbol stat*
STAT_MIN = "32.424"
STAT_MAX = "35.006"
STAT_MEAN = "34.435"
STAT_STD = "0.3948"
STAT_NGOOD = "826"
STAT_NBAD = "174"

So I can use the symbol STAT_MEAN to define a variable.
yes? let sample_mean = ($stat_mean)
Note that for this example, the mean that was computed is the overall mean in X and Z.  X is the list of sampled points, at each Z in the original data.
