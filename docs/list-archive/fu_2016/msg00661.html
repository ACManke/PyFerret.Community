Hi Estanislao,

Paulo is correct but use can add few more thing in this command like time step size and memory size.

repeat/l=2191:4380:1 ( save/append/file=normal.nc sossheig; set mem/size = 1000 )

l=2191:4380:1 (here one is the time step size) and  1000 is the memory size (adjustable)

it will free your memory at every step to make a faster process. you can adjust the memory size according the configuration of your machine i.e. whether it will go above  1000 or it has lower then 1000.

please mail us back if you have any problem for this.

regards, saurabh
