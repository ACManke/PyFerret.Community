Hi Sangeeth,

You need to use Patrick Brockmann's contributed script def_monthaxis_days.jnl.

go/help def_monthaxis_days

will give you some info on how to use it.


Something like

go def_monthaxis_days STANDARD 2010 2014 mytaxis

or

go def_monthaxis_days NOLEAP 2010 2014 mytaxis


and

let var_monthly_ave = var[gt=mytaxis@ave] ! Regrid to monthly time axis using average

Russ
