I'm trying to make a nice image of sea ice data on a curvilinear grid with a
semi-realistic globe underneath it.

Seems to me the calling order should be something like:

[...]
set reg/x=0:360/y=90S:90N
go mp_orthographic 0 90
go mp_fland 120 land_sea basemap all
go mp_land 1
!
go mp_orthographic 0 90
let mp_lambda = lon[d=seaice.nc] * deg2rad
let mp_phi    = lat[d=seaice.nc] * deg2rad
set grid `seaice,return=grid`
shade/lev=(0.2,8,0.2)(8)/title="" seaice*mp_mask,x_page,y_page
[...]

But the underlying mp_fland image is gone - all I get is the shaded seaice 
data.

What am I doing wrong?
