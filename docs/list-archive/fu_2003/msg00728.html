Hello Hein,
The erorr message may be reporting only part of the actual
memory that was reqired for the command to run.  There
are a couple of discussions of memory management and
requirements in the Users Guide.  Please look under these
two index entries

   "memory, insufficient memory"
and
  "memory, large calculations"

In the discussion of large calculations, you will see that when
you have made a number of variable definitions, the size of
the data that must be loaded to evaluate them may be far
larger than the size of the final result.  You may be able to
see what is happening by setting SET MODE DIAGNOSTIC
before your SAVE command.  There is an FAQ on how to
interpret the output of the diagnostic mode, at

http://ferret.pmel.noaa.gov/Ferret/FAQ/other/mode_diagnostic.html

Please let us know if you still have questions about the use
of memory in Ferret.

Ansley Manke
