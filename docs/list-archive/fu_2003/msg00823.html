Hi ,
          You can make use of the inbuilt journal file for this purpose
set your pressure temperature ans salinity as follows for this purpose
set reg/z=0:1000
let dyn_S=mavsal
let dyn_T=mavtmp
let dyn_P=P
go dynamic_height

results will be available in GEO_U,GEO_V  , GEO_Uz,GEO_Vz
good luck
