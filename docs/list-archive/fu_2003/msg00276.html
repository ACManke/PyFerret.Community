Hi Rong,
The code for reading descriptor files is older code, which is very
particular about the formatting in these attributes.  You need two
blank spaces in front of the letters

MC and 1A in the lines
  D_TYPE         = ' MC',
  D_FORMAT       = ' 1A',

We will note this in the documentation, and also will look into changing
Ferret so that the next version could recognize the text with one space,
or no spaces.

Ansley Manke
