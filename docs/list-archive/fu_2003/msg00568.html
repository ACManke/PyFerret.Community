Hi Marc,
Here's a message about creating a custom colorbar in
Ferret using a viewport.  The thrust of the question was
making a continuous colorbar, without lines between the
colors (which has now been made even easier with the
/FILL/KEY=CONTINUOUS option).  But the example
will give you some ideas if you want to make up your own
types of color keys.

http://ferret.pmel.noaa.gov/Ferret/FAQ/custom_plots/continuous_colorbar.html

If you do develop a script you think would be of general
interest, please consider contributing it back to the Users
community.

Ansley Manke
