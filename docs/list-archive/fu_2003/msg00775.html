Hi Lorenzo
You can do this with the SAMPLEXY function.  For an example
look up "cross-section, along an x-y curve" in the Users Guide
index.  What you'll do is generate a list of x-y values along the
curve you want to use, and the call to SAMPLEXY returns
the data in two dimensions; along the curve versus depth.
Then you'll do a shade or fill plot of that.
