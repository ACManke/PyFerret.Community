Hi Katsumi,
	If its just one zero that needs to be added at the start of
each series you could use
	let b=if(i eq 1)then 0 else a[i=@shf:-1]
For example
	let a = {10,2,35}
	let b=if(i eq 1)then 0 else a[i=@shf:-1]
	list/i=1:4 b
 1   / 1:   0.00
 2   / 2:  10.00
 3   / 3:   2.00
 4   / 4:  35.00
