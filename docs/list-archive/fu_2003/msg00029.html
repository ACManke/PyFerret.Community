Hi Michael,

	There are two PlotPlus commands that go with the vector
command that give some flexibility to the scale arrow.  They are
	ppl veckey xx,yy,ipos,format
where xx,yy give a location for the scale arrow, ipos is an option
I think is not implemented, and FORMAT IS WHAT YOU NEED.

For example to add "m/s" to the scale vectors value but leave its
default value and position you could
	ppl veckey,,,,(f5.1,'' m/s'')
	vector/i=1:10/j=1:8 i+j,i-j
(Notice the ppl veckey,,,, allows default values for the first 3
parameters xx,yy,ipos.)  If you use "ppl veckey/nouser" xx and yy
are in inches, otherwise they are in user units.

	The second command that may be of interest allows the
length and scale of the scale arrow to be chosen. (By default it is
0.5 inches representing twice the mean vector length.)
	ppl vecset,len,sc
makes the scale arrow "len" inches long representing "sc" user units.
