Hi Rolando,
There are a couple of examples in the Ferret distribution.
There is a general demo script of all kinds of vector plotting:

   yes? go vector_demo

And here is another demonstration:

   yes? go wind_barbs

You can copy this script, and use the ideas for your own data.

In the vector_demo script you'll see that there is also a script
"stick_vectors.jnl"  which makes a stick vector plot of a line of
vector values.  Type:

   yes? go/help stick_vectors

and you'll see a way to demonstrate the plot.
