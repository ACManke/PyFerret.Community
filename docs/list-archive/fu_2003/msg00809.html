Hi Vincent -

PPL CONSET is a funny command that does not operate linearly.
Sometimes you need to change it by 1/10ths of inches down to
small values like 0.5 inch to see any result. Then it may suddenly
give you lots of contours (too many). It is a strange algorithm
that I don't understand (very old code that is pre-pplus, ~1968).

Usually the behavior of getting no contours occurs when the
data grid is exceptionally dense. Again, I'm not sure why.
If it's not too much trouble, try decimating the data (if
pplus is having this problem then the data must be way denser
than is needed for a plot).
