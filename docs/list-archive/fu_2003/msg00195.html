Hi -
You might be able to use the SET DATA/FORMAT=DELIMITED
command where you can tell Ferret that one of the fields is a DATE,
if the date is in one of these forms:

     DATE - mm/dd/yy or mm/dd/yyyy or yyyy-mm-dd or yyyymmdd - value returned is days
     from 1-jan-1900 (consistent with the DAYS1900 function)

     EURODATE - dd/mm/yy or dd/mm/yyyy or yyyy-mm-dd TIME - e.g. hh:mm or
     hh:mm:ss.s/

If it is not in one of these forms, you could perhaps read
year, month, day, individually with a formatted read statement,
and then use the days1900 function to make a variable which
then could be used to define a time axis.
