Hi Jennifer,
To answer the first question, at present there is a string variable
in Ferret for the dataset names - that's 255 characters long.  Your
first file is over 270 characters long, the second just under 255.
This is something we will increase in the next release of Ferret.
As more of these kinds of datasets are used, I'm sure there will
be the need for longer names.

Ansley Manke
