Hi Michael,
	The trick is in knowing what range of x and y values are being
plotted.  One way to get this is to type
	ppl listsym
which lists a bunch of characteristics of the current plot.
Is you do this following
	go basemap
you will notice that
   21  XAXIS_MIN                        20.000000
   22  XAXIS_MAX                        380.0000
so it is clear that to appear correctly the longitude values of your
cruisetrack must fall in the range 20:380 degrees.
For example
	plot/o/i=350:370/line=2/sym=20/vs i,0*i+45
will show up straddling the dateline at 45N.
