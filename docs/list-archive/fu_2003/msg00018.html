Hi Gary,
There is a section in the Users Guide on making climatologies from
time series data. There is a paragraph there about how to restrict
the data to a particular time period. See "climatology, creating" in
the index to the on-line Users Guide
http://ferret.pmel.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm

To make monthly data you would define a monthly time axis for the
period in question, and regrid your data to it:

DEFINE AXIS/T="1-Jan-1961":"1-jan-1991":1/unit=month t_month
LET precip_month = precip[GT=t_month]

Hope this will get you started!

Ansley Manke
