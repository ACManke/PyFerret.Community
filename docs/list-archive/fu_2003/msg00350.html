Hi Mark,
There's a limit on the length of a dataset name in Ferret,  which is 256
characters, including the constraints.  If you use fewer constraint expressions,
the DODS access works fine.  Probably the simplest way around this is
to specify the DODS file  name and then use Ferret commands to winnow
down the data you actually request.

Ansley
