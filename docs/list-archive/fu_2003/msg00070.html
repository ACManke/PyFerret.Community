The following command should do it
vector/set strx,stry
go remove_logo
go unlabel 4
go unlabel 5
go unlabel 6
ppl vector

As for the label numbers which you want to hide, do
ppl listsym
will show you the corresponding label number.
Good luck!
Muyin
