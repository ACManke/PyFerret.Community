Hi MMA,

I think the problem (or at least one of them) is with the format
qualifier.  You have it as unformatted, which implies binary not
ascii.  Try something like:

set data/format="(f9.0,f9.6)"/var=var1,var2/col=2 test.dat

I hope this helps.

Jeremy
