Hi Gnzeng,
We are looking into methods for regridding and interpolating
on rotated or curvilinear grids.  Ferret can make plots of
such variables (see references to curvlilinear coordinates in
the Users Guide) but does not now have the capability to do
analysis with this kind of data.  We plan to provide this in the
future, however.

In the meantime, if you have code in Fortran or C which makes
the calculations, it should be possible to implement that code
as a Ferret External Function.

Ansley Manke
