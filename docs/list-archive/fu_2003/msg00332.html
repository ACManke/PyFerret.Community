Hi Ron,
Yes, you can do this using the SAMPLEXY function.  You
sample your field at the set of (xpts, ypts) which define the track,
and then you have a cross section of depth vs points along the
track.  There's an example of this in the Users Guide,  see
"SAMPLEXY, defined" in the index
http://ferret.pmel.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm
