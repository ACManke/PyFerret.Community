Hi Maria -

I believe what is going on in your case is that you are attempting to create GIF files when your X-server is in TrueColor mode (24 bit color). You don't mention what version of Ferret you are running, but the most recent version of Ferret (v5.50) which is available no longer has a problem creating gif files under 24 bit color.
For your solution, therefore, you either need to upgrade to the newest Ferret version, or else modify your X-server so that it is running in PseudoColor Mode (8 bit color).

Let me know if you have any other questions.

Kevin
