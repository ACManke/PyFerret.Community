Hi Vincent,
To use the PPL syntax, it would be (including the first VECTOR/SET
command that would come before the PPL command)

   yes? VECTOR/OVER/SET uwnd,vwnd
  yes? PPL VECTOR/OVER,3,4

with a comma between the OVER and the xskip,yskip values.

Note that the Ferret qualifiers /XSKIP and /YSKIP do the
same in a single commandand is probably clearer

  yes? VECTOR/OVER/XSKIP=3/YSKIP=4 uwnd,vwnd
