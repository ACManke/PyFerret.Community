Enils,

The following FAQ describes how to use code distributed with LAS to access data stored in a DBMS. This works by using perl code to access the data and then write it out as a NetCDF file which Ferret then ingests to create the graphical output for LAS. You will also find a standalone script which you can use to extract data and spit it out in a COARDS compliant, Ferret friendly NetCDF file.

http://ferret.pmel.noaa.gov/Ferret/LAS/FAQ/relational_database_access.htm
