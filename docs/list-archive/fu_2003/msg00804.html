Hi Vincent,
The Ferret vector command has a qualifier VECTOR/LENGTH
which lets you set what the vector length represents:

    yes? VECTOR/LENGTH=100 U,V

Creates a vector arrow plot of velocities with ½ inch vectors for
speeds of 100.
