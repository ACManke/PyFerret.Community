Hi Sukanya,
	In addition to the built in smoothers in ferret (@sbx,...)
you can apply your own using one of the "convolve" functions described
in the user guide.  All you need to do is to provide the filter weights
appropriate to the filter (some sort of cosine-lanczos I recall was
commonly used to create a 40-hour lowpass). Pick the one that suits
from the literature or a time series analysis book, then use ferret
first to compute the weights, then to apply them in a "convolution"
calculation. See example under CONVOLVEI.

Good luck,
Mick
PS I'm assuming your input time series is evenly spaced.
