hi Vincent,
	you can switch off longitude formating by
can mode longitudeformat

and then the ^oE/W does not appear in the axes labeling. To
switch back to the default use
set mode longitudeformat

Similarly
can/set mode latitudeformat
will work for latitude labeling.
