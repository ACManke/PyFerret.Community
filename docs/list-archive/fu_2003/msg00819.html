Hi everyone,
Daryl has a file of data described by curvlinear coordinates:

yes? show data
     currently SET data sets:
    1> ./03111012_output.nc  (default)
 name     title                             I         J         K      L
 LAT      latitude                         1:60      1:45      ...     ...
 LON      longitude                        1:60      1:45      ...     ...
 BDECKT                                    1:60      1:45      ...     1:48
So he can plot this with the curlvilinear forms of the plot commands. 
See the several  topics under "curvlinear coordinates" in the Ferret Users Guide.

For instance,

yes? shade/L=1 bdeckt, lon, lat
