Hi Sim,

The developers are working on a version that supports >2GB files (and DODS
access too).  A Linux beta is available from

ftp://www.ferret.noaa.gov/special_request/aw/ferret_v553beta3.gz

BEWARE, this beta doesn't properly "FILL" data on irregular axes due to a
graphics problem (unrelated to the >2GB access).  But you can give it a
try for processing files.  Let them know if you find any other problems.  
I think the official v5.53 release -- with bug fixes -- may be coming out
soon.
