Hi Mulyono,

I think you need the "event" transform (@EVN -- see SHOW TRANSFORMS).

let a = sin(I/13)
plot/vlim=-1:6 a[i=1:200]
plot/over a[i=1:200@evn:0.4]
You'll note that the @EVN transform increments each time that an "event" of crossing the value 0.4 occurs.
