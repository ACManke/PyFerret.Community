Hi Antonio -

Chances are the problem you are seeing is due to the fact that your X window server is running in TrueColor mode. In previous versions of Ferret, attempting to create a gif file in such an environment caused a crash. However, with the newest version of Ferret (v5.50), this is no longer an issue. GIF files can be created using either 8-bit PsuedoColor, or 24-bit TrueColor.

Which version of Ferret are you running, and on which platform?
