Hi Yogesh,
You can do this with the files as they are.  Use the [D=]
specifier for the variables to specify which dataset they are
in.  See the first part of Chapter 2 of the Users Guide for more
about datasets:  Chapter 2, section 1 Overview in
http://ferret.pmel.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm
USe mix.nc
USE air.2002.nc

set view ul
plot m001[d=1]

set view lr
plot air[d=2]

! and so forth.
 

Ansley Manke
