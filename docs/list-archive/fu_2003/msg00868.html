 Kurian wrote:
dear yogesh,
you have to give the commands in the following sequence....

yes? use mmix.nc
yes? PPL AXLSZE, 0.0, 0.0 ! to remove the numeric
labels

yes? plot CO2_mix[x=130,y=40,l=263] ! to make the plot

now..it is a good habit to set all ferret settings back to the default
values if you have changed it during a ferret section...so after finishing
all your plotting & labelling commands for your plot without the
numeric labels, set back the default values. if you are ending your
ferret section with this plot, there is no need to set it back to
default. Suppose if you want to make one more plot, with the same jnl file
(or in the same ferret section), 'with numeric labelling' you have set the
AXLSZE to default values..OK. for this you have to give,

yes? PPL AXLSZE, 0.1, 0.1 ! after the plot without numeric labels
(if you are working with time series, to remove the time axis label you
need to give PPL TXLSZE 0, note TXLSZE have one argument only)

hope this will clear your doubt...
