On Thu, 21 Aug 2003, Marc Hansen wrote:
> Does anyone have experience creating a log-scaled colorbar in ferret?
> For example with levels at
> 0.1,0.2,...,0.9,1,2,3...8,9,10,20,30,...,80,90,100 spaced the way log
> graph paper would be?  I have an example of what I'm talking about at:
> http://www.soe.ucsc.edu/~mhansen/chlor_colorbar.gif

Hi Marc,

No doubt it could be done by defining a special viewport and using FILL
with a bunch of LABEL commands.  But if it were me, I'd just shade the log
of the variable and label the colorbar with units of log(mg/m^3).  You
could always overlay contours at useful reference values like .1, 1, 10,
etc.
