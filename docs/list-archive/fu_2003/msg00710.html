Hi Boyin,
Here is a message from the Ferret Users' list archives about
one solution, using the FFTA function of Ferret to compute
a running power spectrum:

http://ferret.wrc.noaa.gov/Ferret/Mail_Archives/fu_2000/msg00248.html

The script fft2drun.jnl is among the go files in the Ferret
distribution.  Note that you need to copy this script to your
own local directory and edit it; there are comment lines which
point to places where you need to modify it for your own data.
