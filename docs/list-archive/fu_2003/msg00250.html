Hi Stéphanie -

The problem you are seeing in creating GIF's is most likely due to your X server running in TrueColor mode.  Ferret versions v5.40 and earlier were unable to create gif files in that situation.  

However, the newest version of Ferret (v5.51) does not have that problem and is able to create GIF files with an X server in either PseudoColor or TrueColor mode.

To solve your problem you can either download the most recent version of Ferret (at http://ferret.wrc.noaa.gov/Ferret/Downloads/win32_downloads.html) or reconfigure your server to run in PseudoColor mode.

If you have any questions regarding this, please let me know.  

Kevin
