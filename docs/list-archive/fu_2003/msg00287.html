Hi,
You can control the minimum and maximum values that are plotted
in several ways:
1) with the /HLIMITS and /VLIMITS qualifiers on the plot command,

yes? USE levitus_climatology
yes? PLOT temp[K=1,Y=0]
yes? PLOT/HLIMITS=50:200/VLIMITS=20:30 temp[K=1,Y=0]

2) Or, you can select a sub-region of the data to work with:

yes? SHADE temp[K=1]
yes? SHADE temp[K=1,X=150E:110W,Y=-50:50]

You can find information about this topic by looking under "region"
in the Users Guide, and also Chapter 6: Customizing Plots.
http://ferret.pmel.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm
