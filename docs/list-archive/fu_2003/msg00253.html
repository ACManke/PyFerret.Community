On Wed, 26 Mar 2003, Chi wrote:
> what does 
>   "let outvar = 1/0 * x[g=gall] * z[g=gall]"
> mean?

Hi Chi,

1) x[g=gal1] returns the x-axis values of the grid "gall".

2) z[g=gal1] returns the z-axis values of the grid "gall".

3) Multiplying these gives x*z on a 2-dimensional XZ mesh.

4) Multiplying by 1/0 converts all values to missing values.

So OUTVAR is a 2-dimensional XZ grid containing all missing values -- it's
a blank template for organizing the profiles.
