Hi Yogesh,

Yes, "set viewport" is the command you want.  See the user's guide at
 http://ferret.wrc.noaa.gov/Ferret/Documentation/Users_Guide/current/fer_html.htm

click on SET VIEWPORT in the left hand frame for more info.

example using predifined viewports
(ul = upper left):

yes? use levitus_climatology
yes? set viewport ul
yes? shade temp[k=1]
yes? set viewport ur
yes? shade temp[k=2]
yes? set viewport ll
yes? shade temp[k=3]
yes? set viewport lr
yes? shade temp[k=4]
