Hi Mike,
	Ansley beat me to the punch.  The only addition I would make
is that although "go unlabel 10" removes this label, its symbol LAB10
is still available so you could use
label/nouser `($PPL$XLEN)/2` `1.25*($PPL$YLEN)` 0 0 0.15 `"($LAB10)"`
to insert it. (The double quotes are because of the colon in URL.)

	Another suggestion is to use "fill/line/size=0 temperature_hr"
in the lower panel to add the contour lines to the fill, with almost
invisible contour labels.
